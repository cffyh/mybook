# 格罗滕迪克与《收获与播种》：一份关于数学、思维与人的案例研究

> "发现的努力，本质上是一种认识的努力；而认识，归根到底是爱。"
> ——这是贯穿《收获与播种》的精神，也是本系列想要梳理清楚的东西。

亚历山大·格罗滕迪克（Alexander Grothendieck, 1928—2014）是二十世纪最深刻、也最难被归类的数学家之一。他在 1958—1970 年间几乎以一己之力重建了代数几何的地基，又在声誉最盛时退出数学界，晚年隐居。1983—1986 年，他写下一份篇幅超过一千页的手稿《收获与播种》（*Récoltes et Semailles*），既是数学回忆录，又是思想自传，还是一份对数学共同体的尖锐控诉与对"发现"本身的冥想。

这份手稿之所以"非常不容易读"，原因有三：

1. **体量庞大、结构缠绕**——它不是线性论述，而是层层套叠的反思、脚注、再反思，作者会在一个念头上反复回旋；
2. **主题混杂**——数学、心理、伦理、灵性、个人恩怨交织在一起，常常在同一页里从拓扑斯跳到童年，再跳到对学生的失望；
3. **情绪浓度极高**——后半部"葬礼"几乎是一场公开的精神审判，需要剥离情绪才能看清其下的洞见。

本系列的目的，不是复述这本书，而是**把其中真正有长久价值的经验与观点抽取出来**，按主题重新组织，使它们可以脱离原书的私人语境，成为关于"如何思考、如何发现、如何做学问"的可用资源。

## 一句话提要

格罗滕迪克留给我们最核心的一条经验是：**真正深刻的进展，往往不来自更用力地解决旧问题，而来自换一个层次重新提出问题，让问题在新的、更自然的框架里"自己消解"。** 其余各篇都是这条主线的展开。
