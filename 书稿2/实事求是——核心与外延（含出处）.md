# 实事求是——核心与外延（含出处）

> 配套《实事求是——被说滥，却极少被说中》的结构化底稿。把"核心（这个词内部是什么）"与"外延（它站在什么之上、靠什么驱动、覆盖多大范围）"一并收齐，每条标注素材出处，便于后续写作时溯源、取用。

**一句话真义**：实事求是，是在"世界存在可知的规律"这一信念之上，敬畏客观、克制自我、并与逻辑携手，持续地从事实出发、向规律深挖、再用新事实改正自己的一整套**校准纪律**。

**字面三字**：**实事**＝客观存在、可验证的事实（起点）；**求**＝主动探求（不是"看见"）；**是**＝规律、本质、真正对的判断（汉书颜师古注："务得事实，每求真是也"——"是"是"真是/真理"，不是"事实"的同义反复）。

---

## 一、核心：这个词内部是什么

> 三层由表入里的递进，外加一组四侧面。

### 1. 双重身份：态度与方法的合一
- **作为态度**：对"客观世界优先性与复杂性"的敬畏，承认主观与客观永远有距离，拒绝唯理论、唯经验、唯权威。
- **作为方法**：让认知扎根于事实的路径。
- 脱离态度的方法是空壳，脱离方法的态度是空话。
- *出处*：《逻辑和实事求是——态度和方法的合一》。

### 2. 动态闭环：一台校准器
- ①起点：从实际出发，主动获取事实（调研／观察／实验）；
- ②过程：不停在表面，提炼出本质规律；
- ③终点：在实践中用新事实检验、修正，不符就改。
- 本质＝主观认知一圈圈向客观逼近；不是一次性的姿态。
- *出处*：《逻辑和实事求是——态度和方法的合一》；控制论闭环对应《互动—反馈—自组织—稳态》《控制论的奠基》。

### 3. 内核：对抗自欺的纪律（最难、最少人懂）
- 克制自我的幻想、情绪与偏见——"从实际出发，而不是从愿望、情绪或教条出发"。
- 它是"独立面对现实世界"的第一原则：现实里每个问题都是新问题，没有现成答案可抄——与"求学阶段"接受现成知识根本不同。
- *出处*：《勇敢：open+接地气》（"克制自我的幻想和偏见"）；《求学阶段-真正的学习》；通俗化表达见《求真求实》。

### 4. 四侧面（一词熔铸、缺一不可）
| 侧面 | 含义 | 只取此面的替代词 → 丢了什么 |
|---|---|---|
| 事实之基 | 从客观、可验证的事实出发 | "务实／接地气" → 只剩起点 |
| 规律之的 | "求是"＝穿透表面抵达规律本质 | "求真／即事求理" → 不必然钉在事实、不含检验 |
| 校准之环 | 用新事实持续检验、修正 | "校准／迭代修正" → 丢了态度温度 |
| 克己之诚 | 敬畏客观、克制自我、对抗自欺 | "诚实／不自欺" → 丢了方法与目标 |

## 二、外延：它站在什么之上、靠什么驱动、覆盖多大范围

### ① 地基——规律信仰（实在论＋可知论）
"求是"之所以成立，预设了"客观世界存在不以人意志为转移、且终可被认知的规律"。没有这个信念，"求"就没有对象。
- *出处*：《系统—结构—作用力》："相信世界上存在不以人的意志转移的规律。"

### ② 搭档——逻辑（不可分的另一半）
循环第②步"从事实提炼规律"靠逻辑与理性驱动。实事求是与逻辑是**一对**全局规则：前者校准"主观↔客观"的距离，后者守住"主观思维内部"的一致与有效。单讲实事求是，会漏掉它必须与逻辑配对才完整。
- *出处*：《逻辑和实事求是——态度和方法的合一》；《两个关注点：实力+问题空间》（"逻辑和实事求是"应对问题空间）；作为科学方法的应用见《逻辑是什么》《高德拉特》。

### ③ 尺度——科学与日常的共同底层
它不是"处世"专用，而是**实证精神／科学方法在日常里的同一条原则**：实验室里"从事实出发、求规律、受检验"，与独立处理现实问题，本是同一件事。一头连"科学的理想世界"，一头连"日常的现实世界"。
- *出处*：《实事求是：科学理想世界与日常现实世界的共同底层原则》（标题即主旨）。

### ④ 循环为何必需——事实的复杂与无限
它必须循环，不只是为求严谨：事实本身复杂而无限，人永远拿不全，认知只能是阶段性的——是客观逼着人一轮轮校准。
- *出处*：《知识是阶段性的成果》《以有涯随无涯，以有限应对无限》；难度对应《实事求是…共同底层原则》中"事实的复杂性和无限性"。

### ⑤ 检验依赖输出与互动
第③步"新的事实"从哪来？从实践、从与世界的互动中来。不互动就没有反馈，校准循环便会空转。
- *出处*：《互动与表达——人与世界关系的内核》（成文稿）；素材《互动和表达》《输出调动输入》。

## 三、常见误解（都把"是"读成了"实事"）

| 误读 | 丢掉了什么 |
|---|---|
| ＝诚实、说实话 | 道德义，丢了认识论义（诚实是不骗人，实事求是首先是不骗自己） |
| ＝务实、接地气 | 只剩"实事"，丢了"求是"（不再求规律本质） |
| ＝静态地尊重事实 | 丢了校准闭环（它会自我推翻） |
| ＝客观主义、不要理论 | 它反的是教条，不是理论；求规律本就靠逻辑理性 |
| ＝一句口号 | 磨成空话，抽走"克制自我"的硬内核 |

## 四、若要换个说法（按想强调的侧面选）

- **最贴奥义（校准＋对抗自欺）**：用客观校准主观；"承认自己可能错，然后让现实来改你"。
- **纠正"只看事实"（求是＝挖规律）**：即事求理／由实求理；"事实打底，规律为的，检验收口"。
- **强调态度内核**：按世界本来的样子去认识它、对待它；"跟客观死磕，不跟自己的幻想合谋"。
- **强调反教条**：不唯书、不唯上、只唯实（注意"只唯实"易滑回"只看事实"，最好与"求规律"并提）。
