# 稿一 · "相互契合的特征向量"的底层拆解

从底层实现看，"把词转成相互契合的特征向量"这句话大致方向对，但用词很模糊。拆开讲：

## 1. Embedding：从符号到向量

输入是 token id（整数）。第一步是查表：

```
E ∈ R^{V×d}      # V=词表大小(~10万), d=模型维度(如4096)
x_i = E[token_i] # 一次行索引，不是矩阵乘法
```

这一步得到的向量是**上下文无关的**。"bank" 在这里只有一个向量，无论后面跟 river 还是 account。所以严格说，embedding 层本身谈不上"契合"。

## 2. 真正的"契合"发生在 Attention 里

单头 attention：

```
Q = XW_q,  K = XW_k,  V = XW_v      # W ∈ R^{d×d_k}
A = softmax(QKᵀ / √d_k + M)         # M 是 causal mask
O = AV
```

关键在 `QKᵀ`。这是位置 i 的 query 向量和位置 j 的 key 向量做内积——**这就是"契合"的字面含义**：内积大 = 方向对齐 = 位置 i 认为位置 j 与自己相关，于是 j 的 value 被大权重地加进 i 的表示里。

`√d_k` 是因为若 q、k 各分量独立、方差为 1，内积方差为 d_k，不缩放会把 softmax 推进饱和区，梯度消失。

注意 W_q ≠ W_k。这是有意的：相关性是**非对称的**。形容词要找它修饰的名词，名词不一定要回头找形容词。如果强行 W_q = W_k，注意力矩阵会被迫对称，表达能力下降。

## 3. 残差流：向量是被逐层"改写"的

```
x ← x + Attn(LN(x))
x ← x + FFN(LN(x))
```

残差连接的意义常被低估。它让 x 成为一条贯穿全部 L 层的**残差流（residual stream）**，每个子层都是往这条流上"加一个增量"，而不是替换。所以第 12 层的 "bank" 向量 = 初始 embedding + 前 12 层每个 attention head 和 FFN 写入的增量之和。

这也是"multi-head"的实际动机：h 个头并行，每个头用低维投影（d_k = d/h）在残差流上读一个子空间、写一个子空间。不同头学到不同的关系类型（有的做指代消解，有的追踪句法依存，有的做归纳复制），互不干扰地叠加。

## 4. FFN 才是存事实的地方

```
FFN(x) = W_down · σ(W_up · x)       # W_up: d→4d, W_down: 4d→d
```

现在比较主流的一个解释是：把 W_up 的每一行看成一个"检测器" k_i，W_down 的每一列看成一个"值" v_i，那么

```
FFN(x) = Σ_i σ(k_i · x) · v_i
```

这就是一个 **key-value memory**：x 与某个 k_i 对齐时，对应的 v_i 被写入残差流。参数量上 FFN 占 Transformer 的三分之二左右，事实性知识主要压在这里，而 attention 主要负责"把该聚到一起的信息聚起来"。

## 5. 关于"和人脑机制一模一样"

这个论断需要收紧。

**可辩护的部分**：分布式表示（一个概念由大量神经元/维度的激活模式共同编码，而非单个单元）、上下文调制表示——这些确实和神经科学的一些观察对得上。可解释性研究（sparse autoencoder 那条线）发现模型内部存在可分离、可干预的特征方向，比如把某个方向加到残差流上能稳定改变输出行为。这比"复读机"要强得多。

**不可辩护的部分**：

- 目标函数完全不同。LLM 的训练信号只有 next-token cross-entropy；人脑不做这件事。
- 学习机制不同。反向传播需要权重对称传递误差，生物神经元没有这个机制（weight transport problem 至今无解）。
- 架构不同。全局 attention 需要 O(n²) 的全连接比较，皮层没有这种连接模式。
- 没有在线学习、没有具身、没有感觉运动闭环。

更重要的是逻辑问题。你的论证是"必须先真正理解，才可能给出正确回答"——这是把结论当前提。行为正确是否**蕴含**理解，恰恰是争议的核心，不能拿来当论据。反例是现成的：模型能在 MMLU 上拿高分，却在 reversal curse 上翻车（学过 "A is B" 无法泛化到 "B is A"）；能做复杂推理，却对提示词的表面扰动敏感。这些现象很难用"它理解了"来自然解释。

## 6. 更站得住的说法

放弃"一模一样"，改成：

> Transformer 通过 attention 在向量空间中构建 token 之间的动态、非对称、多子空间的关系图，并在残差流上逐层累积成上下文相关的表示。这个过程产生了结构化的内部表示——可干预、可解释、支持组合泛化。它显然不是查找表，但它是否构成"理解"取决于你如何定义理解，而这个定义目前没有共识。

"高级复读机"是错的。"和人脑一模一样"也是错的。前者低估了机制，后者混淆了功能相似与机制同构。

真正有意思的问题不是"它理不理解"，而是：这套机制在什么任务分布上泛化、在什么地方系统性失效——这是可以实证回答的。

# 稿二 · 功能/机制/结构三层判据

| 层 | 判据 | 反例长什么样 |
|---|---|---|
| **功能相似** | 输入输出行为在某个任务分布上重合 | 计算器和心算的人都能算 17×23 |
| **机制相似** | 达成功能的**计算过程**同构（同样的更新规则、同样的目标） | 计算器用二进制加法器，人用九九表 + 进位 |
| **结构相似** | 承载机制的**物理组织**同构（连接拓扑、时间常数、模块划分） | 硅晶体管 vs 皮层柱 |

结构相似 ⟹ 机制相似 ⟹ 功能相似，反向都不成立。Hinton 的滑动主要发生在第一个箭头的反向。

---

## 一、功能层：相似度最高

### 1.1 表征几何对齐（这是最硬的证据）

Schrimpf 等人的做法：给人被试和 LLM 同一批句子，取人的 fMRI/ECoG 响应向量 $y \in \mathbb{R}^{n_{\text{voxel}}}$ 和模型某层的激活 $h^{(\ell)} \in \mathbb{R}^{d}$，拟合一个线性 encoding model：

$$\hat{y} = W h^{(\ell)}, \quad W = \arg\min_W \|Y - HW\|_F^2 + \lambda\|W\|_F^2$$

用留出集上的 Pearson $r$ 作为 brain score。结果：在语言网络（LH language network）上，GPT-2 类模型的中间层能解释接近 noise ceiling 的方差。

**但这个证据比它看起来弱。** 线性可解码只说明信息**存在于**表征中，不说明它被**以那种方式使用**。一个随机初始化的宽网络也能达到不低的 brain score——因为随机高维投影保留了输入的大部分信息。真正的对照实验（untrained baseline、词频 baseline）显示训练带来的增益是实在的但没有宣传得那么大。

### 1.2 惊异度与阅读时间

心理语言学里的 surprisal theory：

$$\text{RT}(w_t) \propto -\log P(w_t \mid w_{<t})$$

LLM 给出的 $-\log P$ 与人类眼动/自定步速阅读时间的相关性，比 n-gram 模型好得多。这条线很干净。

**但有个反直觉的发现**：模型越大、困惑度越低，与人类阅读时间的拟合反而**变差**（Oh & Schuler 2023）。因为大模型对罕见词的预测太准了，人类没那么准。这说明二者的预测分布形状不同——功能相似在这里出现了可测量的裂缝。

### 1.3 涌现的功能对应

一些确实成立的对应：

- **Induction head** ≈ 序列类比推理。在小模型上可以精确定位到 2 层电路：前一层的 previous-token head 把 $t-1$ 的信息写到 $t$，后一层的 induction head 用 $q_t \cdot k_{t'}$ 匹配"上次出现 A 之后是 B"，输出 B。这是 in-context learning 的机械基础。人脑有没有同构电路？未知。
- **多义神经元 / superposition**。特征数 $m \gg$ 维度 $d$ 时，模型把特征以近正交方向塞进 $\mathbb{R}^d$：$h \approx \sum_i a_i f_i$，$f_i \cdot f_j \approx 0$ 但不严格为 0。这与皮层的分布式稀疏编码在**统计性质上**相似。

## 二、机制层：这里几乎全是差异

### 2.1 目标函数

**LLM：**

$$\mathcal{L} = -\mathbb{E}_{x \sim \mathcal{D}} \sum_{t=1}^{T} \log p_\theta(x_t \mid x_{<t})$$

单一标量，全局，密集监督（每个 token 都有梯度）。

**人脑：** 没有单一 $\mathcal{L}$。目前最好的理解是一个异质集合：

- 多巴胺 VTA/SNc 编码 TD error：$\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t)$，这是**稀疏**的、**有延迟**的、**标量**信号
- 小脑用 climbing fiber 送 error 做监督学习，但只在运动域
- 皮层大量突触更新是无监督的（STDP、homeostatic plasticity）
- 多重时间尺度的内稳态约束

**关键差异不在"有没有目标"，而在监督密度。** LLM 每个 token 位置都有一个正确答案。人脑的绝大部分学习没有 teacher signal。

有人辩护说人脑也在做预测编码（predictive coding），目标是最小化 surprise：

$$\mathcal{F} = \underbrace{D_{KL}[q(z) \| p(z)]}_{\text{complexity}} - \underbrace{\mathbb{E}_q[\log p(x|z)]}_{\text{accuracy}}$$

（free energy）。方向上和 next-token 有亲缘关系。**但**：predictive coding 预测的是**感觉输入**（下一帧视网膜像素、下一段听觉波形），不是**离散符号**。人脑没有 32000 类的 softmax 头。

$$\text{LLM: } p(x_t | x_{<t}) = \text{softmax}(W_U h_t^{(L)}) \in \Delta^{|V|-1}$$

这个 $W_U \in \mathbb{R}^{|V| \times d}$ 的 unembedding 矩阵，在皮层里找不到对应物。

### 2.2 学习规则：weight transport problem

反向传播的更新：对于 $h^{(\ell)} = \sigma(W^{(\ell)} h^{(\ell-1)})$，

$$\frac{\partial \mathcal{L}}{\partial W^{(\ell)}} = \delta^{(\ell)} (h^{(\ell-1)})^\top, \quad \delta^{(\ell)} = \left( (W^{(\ell+1)})^\top \delta^{(\ell+1)} \right) \odot \sigma'(a^{(\ell)})$$

**注意 $(W^{(\ell+1)})^\top$。** 误差反传时用的是前向权重的**转置**。这要求：

1. 存在一条反向通路，其权重**精确等于**前向权重的转置
2. 突触必须知道自己在反向通路里的对应者是谁

生物神经元做不到。轴突单向传导，突触前不知道突触后的权重值。这就是 **weight transport problem**（Grossberg 1987）。

已提出的绕开方案，以及为什么都不够：

| 方案 | 机制 | 问题 |
|---|---|---|
| **Feedback Alignment** (Lillicrap 2016) | 用固定随机矩阵 $B$ 代替 $W^\top$：$\delta^{(\ell)} = (B^{(\ell+1)} \delta^{(\ell+1)}) \odot \sigma'$ | 前向权重会"对齐"到 $B$，但在深网络和卷积上崩溃；scale 不上去 |
| **Sign Symmetry** | 只要 $\text{sign}(B) = \text{sign}(W^\top)$ | 仍需知道符号，仍是 transport |
| **Target Propagation** | 每层学一个近似逆 $g^{(\ell)} \approx (f^{(\ell)})^{-1}$，传播 target 而非 error | 逆函数难学；深层误差累积 |
| **Predictive Coding Networks** | 局部误差单元 $\epsilon^{(\ell)} = h^{(\ell)} - \hat{h}^{(\ell)}$，迭代到平衡后更新 | 在某些条件下**收敛到 BP 的梯度**（Whittington & Bogacz），但需要多轮 relaxation，且仍需对称的前馈/反馈连接 |
| **Equilibrium Propagation** (Scellier & Bengio) | 能量模型，两个相位的稳态差给出梯度 | 要求网络对称权重 $W = W^\top$，等于把问题换了个位置 |

**现状：没有一个方案同时满足（a）纯局部（b）scale 到大模型（c）不偷偷引入某种对称性。**

生物上真实存在的是 **STDP**：

$$\Delta w_{ij} \propto \begin{cases} A_+ e^{-\Delta t / \tau_+} & \Delta t = t_{\text{post}} - t_{\text{pre}} > 0 \\ -A_- e^{\Delta t / \tau_-} & \Delta t < 0\end{cases}$$

纯局部，只依赖前后神经元的 spike 时序。它能做无监督的时序关联学习。它**不能**做 credit assignment across 96 layers。

**这是最深的机制鸿沟。** 不是"实现细节不同"，是"没人知道大脑怎么解决深度信用分配问题"。

### 2.3 时间与激活

**LLM：** 激活是实值向量 $h \in \mathbb{R}^d$，一次前向传播离散地过完 $L$ 层，无内部时钟。

**神经元：** 输出是 spike train，$s(t) = \sum_k \delta(t - t_k)$。信息编码在 rate、timing、synchrony 里。LIF 模型：

$$\tau_m \frac{dV}{dt} = -(V - V_{\text{rest}}) + R I(t), \quad \text{if } V > V_{\text{th}}: \text{spike, } V \leftarrow V_{\text{reset}}$$

这不是"$\sigma(Wx)$ 的生物版本"。它是一个带阈值的动力系统，有不应期、有膜时间常数 $\tau_m \sim 10\text{--}20$ms、有自发放电。

**"激活值 = 发放率"这个映射**（rate coding）是一个**建模假设**，在某些条件下成立，但忽略了 spike timing 携带的信息。而 attention 的 $\text{softmax}(QK^\top/\sqrt{d_k})$ 需要在**同一时刻**比较所有位置——spike 网络没有这种"同时性"。

### 2.4 归一化

LayerNorm：

$$\text{LN}(x) = \gamma \odot \frac{x - \mu}{\sqrt{\sigma^2 + \epsilon}} + \beta, \quad \mu = \frac{1}{d}\sum_i x_i$$

这需要**计算整层激活的均值和方差**——一个全局统计量。皮层有 divisive normalization（Carandini & Heeger），由抑制性中间神经元实现：

$$r_i = \frac{x_i^n}{\sigma^n + \sum_{j \in \text{pool}} x_j^n}$$

**这个是局部的**（pool 是空间邻域），且没有减均值项。功能上都做增益控制，机制上一个是全局矩统计，一个是局部抑制池。这是**趋同演化**，不是同构。

## 三、结构层：差异最大

### 3.1 Attention 的 $O(n^2)$ 全连接

$$A_{ij} = \frac{\exp(q_i \cdot k_j / \sqrt{d_k})}{\sum_{j'} \exp(q_i \cdot k_{j'} / \sqrt{d_k})}, \quad i, j \in [1, n]$$

每个位置和**所有**其他位置比较。$n = 128\text{k}$ 时，$A$ 有 $1.6 \times 10^{10}$ 个元素。

皮层连接是什么样？

- 局部连接为主：一个锥体神经元的树突覆盖 ~$300\mu m$，绝大部分输入来自邻近柱
- 连接概率随距离衰减：$p(d) \approx p_0 e^{-d/\lambda}$，$\lambda \sim 100\text{--}200\mu m$
- 长程连接稀疏且**有拓扑组织**（retinotopy、tonotopy），不是任意的 all-to-all
- 皮层总连接密度 $\sim 10^{-6}$（$10^{11}$ 神经元，每个 $10^4$ 突触）

**"每个 token 可以直接看到任意远的 token"在皮层里没有对应结构。** 长程依赖在生物系统里靠的是循环动力学 + 工作记忆的持续放电，是**时间上的**保持，不是空间上的直连。

有人说 attention 类似 hippocampus 的 pattern completion，或者类似 thalamo-cortical loop 的 routing。这些是**功能类比**，不是结构对应。

### 3.2 深度与循环

- Transformer：$L$ 层前馈，每层不同权重 $\theta^{(\ell)}$，无循环。
- 皮层：视觉腹侧流从 V1 到 IT 大约 8-10 个突触层级，**但反馈连接的数量超过前馈连接**。V1 收到的来自 V2 的反馈投射，比它送出去的前馈投射还多。

$$\text{cortex} = \text{shallow} + \text{massively recurrent}$$
$$\text{transformer} = \text{deep} + \text{feedforward}$$

Hinton 在演讲里说"迭代式的、层间共享权重的 transformer 更贴切"——他知道这个问题。Universal Transformer / ALBERT 这类共享权重的变体确实更像循环，但主流大模型不是这么做的。

**注意这是一个真实的等价关系**：weight-tied recurrence 展开 $T$ 步 ≈ 深度 $T$ 的独立层，但后者参数量是前者的 $T$ 倍。生物受限于基因组能编码的参数量，必然走 recurrence；工程无此约束，走了 depth。这是**约束不同导致的结构分化**，不是随机差异。

### 3.3 参数量与突触

| | LLM (~400B) | 人脑 |
|---|---|---|
| 参数 | $4 \times 10^{11}$ | $\sim 10^{14}$ 突触 |
| 精度 | fp16/int8 | 突触强度可能只有 ~4.7 bit（Bartol 2015 海马测量） |
| 更新 | 全局同步，batch | 异步，持续，局部 |
| 能耗 | $\sim 10^6$ W (训练) | 20 W |

有效参数比较其实不公平：突触不是自由参数，大量结构由基因组（$\sim 10^9$ bit，实际编码脑结构的远少于此）先验决定。人脑的"参数"很大程度是 hard-coded prior + 少量学习。

## 四、四条论断逐条判定

| 论断 | 判定 | 修正 |
|---|---|---|
| **目标函数完全不同** | ✅ 成立 | 但"人脑不做 next-token"要小心：predictive processing 假说认为人脑做感觉预测，与 next-token 是同一族。差别是**预测对象**（连续感觉流 vs 离散符号）和**监督密度**。说"完全不同"略强，说"同族但不同类"更准 |
| **weight transport problem 至今无解** | ✅ 成立，且这是最强的一条 | 补充：predictive coding networks 在特定条件下可证收敛到 BP 梯度，但代价是引入了另一种对称性要求。所以是"换了个问题"而非"解决" |
| **架构不同，$O(n^2)$ 全连接** | ✅ 成立 | 补充：这不只是"皮层没有"，而是皮层**不可能有**——轴突的物理布线成本随连接数超线性增长，脑容积是硬约束（wiring cost minimization 是皮层布局的主要解释原则之一） |
| **无在线学习、无具身、无感觉运动闭环** | ⚠️ 部分成立，需拆开 | 见下 |

**第四条要拆：**

- **在线学习**：确实没有。推理时 $\theta$ 冻结。但 in-context learning 是一个真实的现象：模型在 forward pass 里实现了某种隐式优化。有工作证明单层线性 attention 在特定构造下**精确等价于**对 in-context 数据做一步梯度下降：

  $$\Delta W_{\text{implicit}} = -\eta \sum_i (W x_i - y_i) x_i^\top$$

  所以"没有在线学习"要限定为"没有**权重**的在线更新"。激活层面存在快速适应。这与人脑的短时程可塑性（STP）和工作记忆的区分有对应。

- **具身 / 感觉运动闭环**：成立，且这是**功能层**的差异不只是结构层的。没有闭环意味着没有 intervention，没有 intervention 意味着只能学到相关而非因果。这解释了一批失败模式：reversal curse、对反事实的脆弱、物理常识错误。

  形式化一点：观察数据只给你 $P(Y|X)$，干预给你 $P(Y|do(X))$。二者在有 confounder $Z$ 时不等：

  $$P(Y|do(X)) = \sum_z P(Y|X,z)P(z) \ne \sum_z P(Y|X,z)P(z|X) = P(Y|X)$$

  纯语料训练只能逼近后者。这是一个**原理性**限制，不是数据量能解决的。

---

## 五、结论

**功能层：中到高相似。** 表征几何可线性对齐，surprisal 预测阅读时间。但对齐可能被夸大（随机基线不弱），且随规模增大出现系统性偏离。

**机制层：低相似。** 目标函数不同族（监督密度差几个数量级）；学习规则的核心问题（深度信用分配）在生物系统里**完全未解**；激活的时间结构不同（连续 rate vs spike timing）；归一化是趋同而非同源。

**结构层：几乎不相似。** attention 的全连接在皮层物理上不可实现；深度 vs 循环的取舍方向相反；参数精度、更新方式、能耗都差数量级。

所以那句"和人类大脑理解语言的机制一模一样"——**在功能层是一个可辩护的弱主张。在机制层是一个未经证实的强主张。在结构层是错的。**

Hinton 自己写"very similar"、"same sort of thing"、"we don't fully know how we do it"，他是在功能层说话。把它拔高到机制层再拔高到结构层，是读者做的，不是他做的。

---

有一个更值得追的问题：**上面这些差异里，哪些是"实现细节"，哪些是"必要条件"？**

如果理解只需要"构造一个满足局部约束的高维表征结构"，那 BP vs STDP 就是实现细节，attention vs recurrence 也是。如果理解需要"通过干预验证因果结构"，那具身就是必要条件，而 LLM 的天花板是可以先验断言的。

这个问题目前无法回答，因为我们对"理解"仍然没有一个可操作的定义。这也正是那个挑战之所以有力的原因——**论证是循环的，但没人能给出打破循环的定义。**
