# 毛泽东两段论述——"实事求是·控制闭环·转化框架"三股合流的经典文本案例

**素材来源**：毛泽东《中国革命战争的战略问题》两段。
> 指挥员的正确的部署来源于正确的决心，正确的决心来源于正确的判断，正确的判断来源于周到的和必要的侦察，和对于各种侦察材料联贯起来的思索。
> 事物是在矛盾运动中的，唯有变才是唯一的不变。

**用途**：与《五点四环节教学法》札记并排，作为一个文本里同时拧着三股理论的样例——实事求是（认识论）＋控制闭环（动力学引擎）＋转化框架（本体动力学）。一个不用这些术语的实践者，把它们全说全了。

---

## 一、两句的关系（先点破）

前一句讲**认识如何通向行动**，后一句讲**认识为什么永不能停**。第二句是第一句必须循环的理由。分拆后必须合读。

---

## 二、第一句：一条从行动倒溯到事实的链

部署 ← 决心 ← 判断 ← 侦察 + 联贯思索。毛从结果往回追到根。三层意思：

**(1) 它就是"感知—建模—决断—行动"的认识闭环。**
- 侦察 = 采集事实 = 实事求是的"求实事"
- 联贯思索 = 加工 = "求是"，从散材料里求出规律/骨架
- 判断 = 对局势的理解（模型成形）
- 决心 = 选定方向
- 部署 = 落地

整条链的脊梁是**实事求是**：行动必须扎在事实上，不扎在愿望、面子、情绪上。对应《实事求是——被说滥，却极少被说中》里的"无我的实事求是"——把指挥员的主观愿望从判断里拿掉，否则部署再漂亮也是空中楼阁。

**(2) 最要紧的词是"联贯起来的思索"。**
侦察材料本身只是散点、孤立的"实体"；"联贯"才把它们接成结构——这是**系统—结构**那一刀：从实体到关系。它也是"把书读厚再读薄"里**读薄**的那一下，是《洞察看见骨架，演绎让它落地》里的**洞察**。
所以毛这条链是**洞察→演绎两冲程**：侦察+联贯思索（归纳，见骨架）→ 判断（骨架成形）→ 决心+部署（演绎，落地）。判断是铰链——归纳翻转成演绎的转折点。

**(3) 它还是一套根因思维。**
"正确"重复四遍：正确性沿链传递，错误也沿链传递。上游错了（侦察马虎、思索不联贯），下游决心再果断也救不回——garbage in garbage out。部署失败时不在部署上修补，要回溯哪一环断了。瓶颈永远在上游的感知与加工，不在下游的执行。同于**中间环节/杠杆点**、同于控制论"控制作用于偏差的根源"。

---

## 三、第二句：变是底色，但变有章法

"事物在矛盾运动中，唯有变是唯一的不变"——直接落在《世界的本质是转化》上。矛盾＝作用力，矛盾运动＝动力学主线：**相互作用→结构变化→状态转化**。

但框架要补一句，否则"唯有变"会滑向虚无：变之中有**守恒**、有**相似相异**。穿过转化而留存的不是事物本身，而是结构、同构、不同转化间反复出现的"相似"。更准的说法：万物皆变，但变有章法——守恒量不变，同构模式反复重演。这一补，把"唯有变"从漂亮虚话拽回成可操作的认识论。

---

## 四、合流：为什么两句必须一起读

**正因为世界不停地变（第二句），第一句那条链就不能是一次性的直线，必须是循环。**

侦察→思索→判断→决心→部署走完一轮，局势已又变。昨天正确的判断，今天就是过期模型，因为"事物在矛盾运动中"。决心一旦下定，仍须随新侦察揭示的新事实不断修正。所以实事求是不是一个动作，而是一条**纪律**——因为现实在移动，必须不停重新校准。一切教条主义、本本主义必败，就是拿旧地图走在动的疆域上。

换成框架语言：第二句是第一句**必须迭代、必须反馈**的理由（《迭代：复杂世界的基本形态》《互动—反馈—自组织—稳态》）。复杂多变的世界里，认识只能以闭环的、永不终结的形态存在。

再接回**战略**（全局视野+主动权+根据地）：在变的世界里，**主动权的本质是让认识闭环转得比环境变化更快**——即《五点四环节》札记里那个"缩短死时间"。侦察—判断—部署回路若比战局漂移得慢，主动权就丢了。毛把侦察放在链条的根、把"联贯思索"紧咬其后，正是在拼命压缩回路滞后。

**收束**：第一句给出认识通向行动的结构，第二句给出这个结构必须永远循环的理由；合起来，就是实事求是为什么是一条终身纪律，而非一次顿悟。

## 五、三股理论在这一个文本里的对应表

| 理论股 | 对应文本要素 |
|---|---|
| **实事求是（认识论）** | 侦察求实事 → 联贯思索求是 → 拿掉主观愿望的"无我"判断 |
| **控制闭环（动力学引擎）** | 侦察→判断→决心→部署的感知-决断-行动回路；根因回溯；缩短死时间=争主动权 |
| **转化框架（本体动力学）** | 矛盾运动＝相互作用→结构变化→状态转化；唯有变不变，但守恒与相似相异给变立章法 |
| **洞察—演绎（直观）** | 侦察+联贯思索（归纳见骨架）→ 判断（铰链）→ 决心部署（演绎落地） |
| **迭代/反馈** | 世界在变 → 认识链必须闭环重复，决心须随新事实修正 |
| **战略论** | 主动权＝认识闭环快过环境漂移 |

## 六、可迁移原理（与五点四环节札记互参）

1. **行动的质量由其最上游的环节封顶。** 决心、部署都救不回马虎的侦察与不联贯的思索；要改结果，回溯到链的根。
2. **散材料必须"联贯"成结构才有用。** 信息的价值不在采集量，在能否接成骨架——读厚之后的读薄。
3. **变是常态，所以认识必须是闭环纪律而非一次性顿悟。** 实事求是的难，不难在某一次求真，难在持续地、对抗自欺地重新校准。
4. **"唯有变"需要"守恒+相似相异"来补全**，否则沦为虚无；真正穿越变化留存的是同构的结构。
5. **主动权＝你的认识闭环快过世界的漂移速度。** 与五点四环节"缩短死时间"是同一条原理在战场与课堂的两次现身。
