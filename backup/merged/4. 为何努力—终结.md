# 4. 为何努力—终结

关于堕落，我是有价值的。也许现在，你可以理解这个价值了。

为了理想，我愿意过着无耻下流的生活。

追求世俗生活，在理想主义在看来，就是追求无耻下流的生活。所有有关名利权的游戏，都是无耻下流的。精神分析法是用来分析病人的，如果你学会一招半式，用来分析自己可以理解，没事拿来分析同学同事，你就会让人特别恶心。同样，关于理想，我们努力是用来解决真理的，但用获得的真理来追求世俗，自然就会视对手、他人和社会为分析对象，利用其弱点，为自己的私欲服务，这同样特别恶心。

洛斯阿拉莫斯的科学家们竭尽全力研发了原子弹，尤其默默奉献的总工程师费米，他们最终为结束二战做出巨大贡献。但是战后大多数参与其中的人，都感觉到恶心，罪恶。领导者奥本海默引用印度古诗说：“如果一千个太阳在天空一起放光，人类就会灭亡，我似乎成为死神，成为世界万物的毁灭者！”奥本海默作为美国代表团成员在联合国大会上脱口而出：“总统先生，我的双手沾满了鲜血”。面对所有的媒体，奥本海默坦言：“无论是指责、讽刺或赞扬，都不能使物理学家摆脱本能的内疚，因为他们知道，他们的这种知识本来不应当拿出来使用。”

所以，海森堡才是伟大的。他才是玻尔的真传，才是原子核的大师。但在德国，他不仅没有努力争取，而且一直消极对抗，让德国当局相信，原子弹不能短期之内造出来，就算造出来，也并不好控制。

如果换成中国，有科学家能够在那时制造一颗原子弹投向日本，我们的民族感情会怎么样？这位科学家的自豪感又会怎么样，我猜，他会被写成历史，成为英雄。

这就是实用主义的遗憾。

**没有极致理想的人，是无法理解这种罪恶感的**，因为他们没有信仰的感觉。

我曾经就是这样的人，我为了理想，采取了不择手段的方式，愿意过无耻下流的生活。

如果不得不和烈士和小丑走在同一条道路上，能够离理想更近，我愿意。

如果一箪食一瓢饮在陋巷，或累累若丧家之狗，能够离理想更近，我愿意。

如果在名利权的游戏中，攀高附贵、曲线救国，能够离理想更近，我愿意。

我原来就是这样一个人啊。

等我真正过上无耻下流的生活后**，我感觉到我思想深处是堕落的。**

**幸好我堕落了，否则我这个**狂徒，一定会做出很多违法乱纪的事，甚至干尽坏事。

所以，哲学系会有张国焘、陈公博、石平太郎这样的人，一点不怪。

同样的理想主义者，**也是有善恶的。更多的是善恶不分，骑墙看**，随风倒。也就是说，如果中国能够用上石平这样的人，**如果他留学时去了欧美**，他一定不会变得这么让人恶心吧。

哪怕是堕落于理想而自杀的人，海子和顾城，死的太不一样了。

理想主义的外衣下，往往是一个自私自利的狂徒。

这其中，当然包括很多社会学家和革命家，明晃晃地举着为人类为民族为国家为正义的旗号，但依然把自己的利益置于最高处。神圣不可侵犯的，不是真理，而是他们的理想或主义。

再说就扯远了。自我批判并不能借此来批判别人或历史，那是“我是流氓我怕谁”的玩法。

拉回来说努力的事。一个努力到撞理想天花板的人，也可能是一个自私的家伙。

为本我努力的人，究竟是不明白“为何努力”的价值啊。

因为这个世上存在超我的人格。

这是我堕落到世俗之中，和很多世俗的人打交道、甚至残酷竞争之后，才明白的。

当我第一次写长篇小说有机会和作家出版社谈判，我听完编辑的一大堆话，要把我归为某系列作家，让我重新修改，我表示拒绝时；当我目睹我放弃之职位，有些人不择手段用尽无耻之力去争取时；当我在数千人的考场中脱颖而出，第一个获得某报职位，但因不能发挥自己专长而不得不拒聘时；我就会发现，我与这个世俗社会是格格不入的。

我写完第一部长篇时才20多岁，回头一看，那故事不要太傻，这样的故事，我一年就能写几部，但当年和《钥匙》、《谁比谁傻》放在一个系列，我确实有点无法接受。我不愿意成为那样的作家，这基本上表示，我今后根本就成不了作家了。同样，在职场有太多的潜规则，公司一大，必须要站队，要斗争，要你死我活。这些东西，对于一个哲学系的人而言，确实是小儿科。但我为了十几万的年薪，干这种勾当，有五斗米存食的我也折不起这个腰。这基本上意味着，我今后根本无法在职场混了。同样，尽管我刚写几篇新闻就名震江湖，但也很难会有一家媒体能够收留我，因为我只想做自己认为对的事，哪怕这事和报社的潜规则有冲突。

我根本**就无法在这个世俗的社会生存，尽管**我总给人一种“北大高材生”的才华横溢感。

这种感觉其实在我第一份工作——政治老师时，就有强烈的感觉。我总是喜欢按照自己的方式来上课，传授知识，以及对付考试，也总是喜欢按照自己的理解来管理班级。

我始终有一个这样的强逻辑，我有办法胜任这些工作。不仅自己玩得嗨，也完全经得起世俗标准的检验。我可以让学生同样嗨起来，然后考得好。但是，我忽略了其他老师的反应以及学校领导的意见，何况还有百年名校的历史责任。同样，我能够写出好看的故事，成为优秀作家，同样也能让出版商挣到钱，为什么要接受那么多条件？却忽略了大出版社也是有自己计划以及历史文化，大公司和大报社更是这样。这样的地方，我听到最多的话是“地球缺了谁都一样转“。

所以，我只能自己干。

但当自己干时，不得不面对员工管理，甲方关系，甚至工商、税务、社区服务各种层出不穷的麻烦，也是我不能胜任的。我不得不组建团队。我成为老板之后，开始拿出自己的规则时，才发现，原来学校、公司、出版社或报社的原则是有用的。我和他们一样，并不喜欢规则的例外者或创新者，哪怕他的能力有孙悟空那样强大。这就是世俗的力量，以及世俗的玩法。你要深入到各个层面，玩转所有环节后，才会明白其中的道理。

这个道理是，世俗之所以这么强大，不是因为历史久远，人数众多，而是自成一个系统，谁想打破或重新规划，都是万难的。

这个世俗系统是一个强力场，是一个黑洞，一旦接近，就会被彻底吸引进去。

难怪理想主义者都轻世俗。难怪理想主义者没几个是干净的。难怪有些干净的理想主义者在世俗生活会穷困潦倒。

从此，我对世俗的理解完全变了。我甚至接受了曾经嗤之以鼻的HB同学的理念。

挣钱，是一生的事业，也是我的理想啊。

哈哈哈哈哈哈……

我曾经笑过人家，那是大四，尽管他后来挣钱的速度不要太快。后来，我开始笑自己，笑完之后，我也发现，我挣钱的速度不要太快。所以，你也可以起劲笑笑。

笑完之后，我发现，我服务过的中学还在原地（我一直以为下滑，但2019年700分以上的名单中，该中学居然出现了两次，我猜换校长了，一查果然。）当然，我服务过的公司早倒闭了，尽管有一家曾经那么大。我服务过的报社好几家，也不灵了。至于出版社，也早没当年的地位了。行业如此，那些曾经自以为是的老板们，以及激烈竞争的员工们，会是一种什么生活和心态呢？

这就是我所谓的”名利权是简单模式，世界是善良模式“的说法。

你把挣钱切换成理想来努力时，是完全不一样的效果。我有一位同学在切换时，经常遇到困境，他就想：”我就不信，那些大学都没念的人怎么可能比我干得好“。这就是典型没有切换过来的人。正确的问法是：”那些高中生居然和我是同事，而且还做得更好，他们究竟是怎么做到的？“我就是这样问的，在第一份下海的工作中，从一个高中生那里，学会了太多无聊的技巧，这些技巧能保证，下回我遇到同样的对手，能轻轻松松地把他们收在手下。

那么，问题就来了。

在无耻下流的社会生活太久，你对曾经的理想还有印象吗？

你的成功有太多无耻下流的东西，你的内心是如何感受的？

面对那些显而易见的弱者，你要不要继续挣他们的钱呢？

这基本就是问草原上的狮子，吃羊的几个逻辑。而这头狮子，曾经在动物园长大，原本是一个素食主义者。

好在这是一个比喻，我并不是那头狮子，也并不是任何猛兽。而且，我是在自问，在反思，在**自责。我为什么要把挣钱当成理想来呢？如果挣钱真的是我最后的理想，那么，这个理想世界里，怎么可以只有我一个人呢？**

当一头狮子吃多了羊，是不可能反思出罪恶的，更不能通过对罪恶的反思找到善意。但人是可以的，至少我找到了。

在我现在的理想世界中，有我的家人，亲人，以及亲密的朋友，他们应该得到我的帮助。以此为圆心，扩散出去，我曾经的同学、老师、朋友，那些曾经帮助过我的人，哪怕是通过他们自以为是的方式，包括主观上伤害我客观上帮助我的都算，也应该得到我的帮助。还有校友、老乡、所有经历和我类似的人，我其实都是想帮助的。当然，那些本身生活环境就差的人，更值得帮助。

这种帮助，是需要钱的，至少是需要钱来做支撑的。

但不仅仅是钱，更需要出力。我捐过钱，也去支过教，这也算是捐人吧。

这种念头，并不是挣到钱时，我才这样想的，而是把挣钱当成理想来实践时，我就这样想的。我的公司，一开始就是找会计公司来做账，他们绝不会因为那点服务费给你做假账。同样，我从未拖欠过员工、乙方以及合作方的一分钱，一直都是传说中的模范。很多年中，我只开除过一个员工，我甚至从未修改过公司的提成规定，导致公司很小时的大比例提成，在业务扩大后老员工挣得不要太嗨，我对公司老员工的尊重，是通过制度上的现金来体现的。但他们真正体会到这种善意时，是在公司转手之后，或者他们也当上老板之后的事了。

我也希望，所有我曾帮助过或影响过的人，能找到自己的本善，并力所能及的表现出来。

成为风之后，自然就会有余力，去帮助这个世界。

然后，在帮助世界的行动中，找到继续努力的动力。

为他人而努力时，才算真正找到努力的价值。
