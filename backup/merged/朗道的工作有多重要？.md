# 朗道的工作有多重要？

朗道把物理学发展成了美学，对称性与对称性破缺的思想影响太深远了。

看到一个物理现象，定义几种基本运动，考察运动过程中的不变量，理论就已经建立起来了。

非常神奇的一点就是，朗道可以自由的使用物理与数学知识，通过实验获得物理方程后，就开始了对称性分析，叠加性质，时空变换不变性，相差一个时间微分，允许一个任意常数，偶函数所以函数形式为偶次幂级数，系数即为物理量。

数值叠加体现为概率乘法，所以态必然为指数形式，三角函数具有周期性，所以运动有边界，有限区域无穷运动会导致稠密性与混沌。

这些物理性质与数学形式的关联优美，完美甚至于不可思议，我觉得在朗道看来，

- 物理 = 数学
- 物理系统结构 = 对称性
- 物理系统运动 = 群/半群【群代表了周期，比如旋转，半群代表了有序，比如时间】

所以，在朗道之后，对称性理论成为理论物理的基本手段，诺特发现了数学对称性的物理意义，朗道将其推广到所有领域。

其实，还有一种思想，那就是序，最小作用量原理，半群，时间箭头，相变，最优化。这就是对称性破缺。

- 序与对称可以描述几乎所有的物理学理论。一个是时间发展，一个是相空间各态历经。

不过，还有一个东西，那就是奇点，混沌与非线性，这个就需要后人来解决了。或许他意味着数学中的环。环的物理意义可能就是奇异性与混沌。因为非线性微分方程的解是函数环中的子结构。
