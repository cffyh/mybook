# 书面表达是一种思考逻辑

书面表达是一种思考逻辑，关键在于把事物描述清楚。

书面表达是把一件事情表达完整。口头交流的关键是双方一起来拼图，各取所需;

书面表达在于描述清楚，口头在于故事交流，实时的交互。

# 口头以书面思考为基础

口头以书面思考为基础，按照对方诉求，快速清晰吐出关键信息，互动拼图，逻辑是必要手段。

- 最关键的是：了解对方的诉求；
- 对方会思考，所以核心点在于，简短吐出关键信息。不求全面，这和书面表达不同。
- 实时要求高、语气和表情关键、传输速率有限；
    - 语音 < 文字 < 表格 < 制图 < PPT；

我们会发现，那些口头演讲厉害的人，没啥能流传的大思想，都是一些零碎，为办事而服务的细致小想法。

锻炼口头表达的一个办法是，相同故事在不同的对象和场合重复讲。

关键词表达法，重复多次的讲演

- 第一次讲演时，可以只记住关键词， 然后再临场造句。
- 第n次，就可以根据第n-1次讲演的，再回顾。
- 回顾和重复是提升的关键。

# 表达是信息的描述，需要对信息进行编码.

如何将一个多维系统编码为一维符号，要保证正确性，就得借助于逻辑。

**逻辑是正确性的学问，每一步都正确，一步步连在一起就能达到目的地。**

数学符号，表达定量关系，代表了逻辑的巅峰。

逻辑：

- 逻辑就是目的明确，有理有据 => 串点成线。
- 逻辑，批判，辩证，一脉相承，层层递进。
- **逻辑和表达的高度一致：围绕目的，串点成线。只抽取和目标相关的部分，而不是试图涵盖全部的信息。**

逻辑思维因为动作很明确，所以是容易训练的。

目标明确—>注意力和记忆不发散—>精确记忆强—>逻辑思维能力强。

语气，语速，语调能决定沟通效果，谈关键事情，远程比当面要失去决定性的信息。
