# 63 记忆



“推背图，只是一种记忆方法，**一个背字可以概述，那就是把数学当物理背，物理当化学背，化学当生物背，生物当语文背，语文当政治背，政治当英语背，英语当天书背**……”

于是，大家展开讨论，一项项列举出来，果然十分可观。人的五官，眼、耳、口、鼻、身都有记忆功能。眼睛的图像记忆又可细为动态与静态、彩色和黑白、规律与混乱等；耳朵的声音记忆，包含音调、音色、强弱、节奏的变化；嘴上记忆与声音不可分离，但也可无声朗读；鼻子的嗅觉记忆持续时间很久；**全身的感觉记忆就是传说中的通感，会与环境密切相联系……**

章若兰分享的是艾**宾浩斯的遗忘曲线，刘天如同听天书一样，记忆有周期，遗忘很迅速，复习是根本，排列很重要**。背英语单词，不考虑遗忘曲线，基本上都是事倍功半。

孙修吾更牛，他介绍了一本分析“人的大脑”的书，说大脑皮层记忆功能，都是神经元的生化作用，本质是**记忆单个的字词，无法记住长句，除了律诗、词赋和歌词这些有字数和韵律可依的，一般的句子，遗忘率那是相当高的，这是神经构造的本质**。所以，要记住长篇大论是不太可能的，**只需要记住关键词，然后一点点重新凑成句子。**

谢少武说，要背诵**的东西都可以联想，联想到一些好玩的、有趣的，记起来就**快了。江睿说，**直接死记简单明了，效果很好，记不下来的再想其它办法**，总之，**记忆是越练越好。**

这一晚，刘天没休息好，他**反复想着体育锻炼和记忆训练的联系**，一点点琢磨所谓的遗忘力，越想越觉得兴奋，反反复复地想着，最终昏昏睡去时，也不得要领。等想清楚后，刘天写了一篇感悟，题目就叫《**肌肉训练：为了遗忘的记忆》。**



曹烽说，形象记忆很好，很容易记着一些东西，其它记忆方法也一样，不管什么方法都能记住一些东西，想忘都难。谁不会背书呢？记住了，但是记不牢，特别是记不全，这才是大问题。他说提供这个例子供大家参考，三十六计可以这样记全了。

写完，曹烽在第四句“辈”字上画一个圈，最后一句画了一条线，最末一字“杀”也画一个圈。说，除了画线的画圈的字，一共34个字，假字后面的“反借假”按重算，36个字就全了，全是三十六计的第一个字。而最后一句，只是为了加深对“反借假”的印象，客间、尸刀、痴道分别对应反、假、借共计六条计谋，对于熟悉的人而言，可以不记。整个打油诗的效果就是“辈杀”，即“背杀”。
