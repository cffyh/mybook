# 工作主线

核心目标、主要矛盾、主线、关键事情，他们核心思想是一样，但他们是不同的涵义，处于不同的抽象层次。

主线是[战略](./战略.md)的表现 ，[关键事情](./洞察身边事物和业务的特性本质；找到关键控制点并由此搭建资源梯队.md#关键事情)   +    [环环相扣](./洞察身边事物和业务的特性本质；找到关键控制点并由此搭建资源梯队.md#环环相扣)

- 关键事情和主要矛盾的区别
    - 矛盾是本质，但关键事情是具象，更关注结果。
- 关键事情和核心目标的区别
    - 目标是想要，但关键事情是路径的载体。

如果有了自己的主线，当前的工作任务，是积累的一部分，不依赖这一个任务做到深、广。如果这个任务具备一定的挑战性，完成之后的总结那就更有价值了。

要学习的东西太多了：是主线不够精准，需要主线具备真正的核心竞争力。

工作没有遇到巨大的挑战导致无法快速晋升的问题：

- 如果有了自己的主线，不需要“巨大”的挑战，不依赖当前任务的任务，多个平凡的任务，在主线的深度和体系化加持下，也能出类拔萃。然后又因为有了主线，就有可能可以发掘到巨大挑战的机遇。
