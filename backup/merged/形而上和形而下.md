# 形而上和形而下

请注意,现代哲学中的"形而上学"是日本学者翻译西方哲学时引月用《易传》中的"形而上"而形成的一概念。

### 系辞

此段开头便说:

子曰: 书不尽言,言不尽意。然则圣人之意,其不可见乎?子曰:"圣人立象以尽意,设卦以尽情,系辞焉以尽其言。变而通之以尽利,鼓之舞之以尽神。

这是点明《易经》的**象、卦、辞和爻**的作用都是为了尽圣人之意。

最后才总结说:"**是故形而上者谓之道,形而下者谓之器。**

化而裁之谓之变,推而行之谓之通,为天下之民举而措之谓之事业。

#### **象、彖、爻**

**它们是组成一卦的重要因素。**

- 卦象，象征万物；
- 彖辞：裁断（一卦之义）；
- 爻：效法天下万物变动。

易经总共64卦，每卦六爻，一共384爻。**阴阳两种卦象是由爻位和爻数决定。**

### 西方

**形而上学研究存在的本质和基础，而形而下学研究存在的表象和现象。**

- 形而上学研究存在**的本质和本体，试图揭示它们的真正特征和属性**。
- 而形而下学则研究这些**本质和本体在现实中的表现，试图揭示它们的外部表象和现**象。
- 主观存在和客观存在。

它们之间的关系可以用“本质”和“表象”来描述。

# 叔本华：四重根，给了一个更具体的回答

人的行为动机是充足理由**律第四种形式，**人的行为必有动机、从动机得到解释，这便通向了他最著名的作品《作为意志和表象的世界》。

---

叔本华在《充足理由律的四重根》中论证“四重根”的完备性，主要基于对人类认知能力和表象世界的系统性划分，通过穷尽表象的基本类型及其对应的解释原则，证明四种充足理由律形式涵盖了所有可能的“理由”范畴。以下是其论证的核心逻辑：


一、表象的分类与充足理由律的本质
叔本华继承康德的“表象世界”理论，认为人类认知的对象是“表象”（分为直观表象与抽象表象），而充足理由律是“一切表象的普遍形式”，即每一种表象都必须在某种关系中被认知，这种关系即为其“理由”或“根据”。
他将表象分为四大类，每类对应一种独特的“充足理由律形式”，从而形成四重根：

1. 直观的、经验的表象（物理世界的对象）
2. 抽象表象（概念，即逻辑思维的对象）
3. 直观的、非经验的表象（时间与空间，数学的对象）
4. 内在表象（意志的自我意识，即动机与行为的关系）

二、四重根的具体界定与完备性论证

第一重根：生成的充足理由律（因果律）

- 适用范围：直观的、经验的表象（物理对象）。
- 核心原则：每个物理对象的变化（生成）都有其原因，即“因果律”。原因与结果的关系是时间中前后相继的必然联系。
- 论证作用：覆盖自然界的物理因果关系，解释经验世界的变化。

第二重根：认识的充足理由律（逻辑真理）

- 适用范围：抽象表象（概念）及其组合（判断）。
- 核心原则：判断的真理性需要逻辑理由，如三段论中的前提与结论关系，或概念间的包含、矛盾等逻辑关系。
- 论证作用：覆盖逻辑思维领域，解释判断的有效性依据。

第三重根：存在的充足理由律（数学关系）

- 适用范围：直观的、非经验的表象（时间与空间，作为先天直观形式）。
- 核心原则：时间与空间中的位置关系（如几何中的点、线、面关系，算术中的数的序列）构成“存在的理由”，即数学真理的必然性源于时空的先天结构。
- 论证作用：覆盖数学领域，解释先验综合知识的可能性。

第四重根：动机的充足理由律（动机与行为）

- 适用范围：内在表象（自我意识中的意志行为）。
- 核心原则：意志的每一次行动都有其动机，动机作为“理由”必然导致行为，体现为“目的因”或“动机因果性”。
- 论证作用：覆盖人类行为与意志领域，解释动机如何决定行动。
三、完备性的关键逻辑：穷尽表象的基本类型
叔本华认为，上述四重根的完备性源于其对表象世界的彻底分类：

从认知对象看：

- 外部世界（物理对象）→ 因果律（第一重）；
- 抽象思维（概念）→ 逻辑理由（第二重）；
- 先天直观形式（时空）→ 数学关系（第三重）；
- 内在意志（自我意识）→ 动机（第四重）。
这四类表象涵盖了人类认知的全部可能对象，无遗漏亦无重叠。

从“理由”的本质看：

- 因果律（物理变化的理由）、逻辑理由（判断的理由）、时空关系（存在的理由）、动机（行为的理由），分别对应“为何变化”“为何为真”“为何存在”“为何行动”，构成对“理由”的终极追问的完整回答。

对传统充足理由律的批判与细化：

- 叔本华指出，莱布尼茨、沃尔夫等前人对充足理由律的表述过于笼统，未区分不同领域的“理由”本质。他通过划分四重根，证明只有明确不同表象领域的特殊规律，才能避免混淆（如将因果律与逻辑理由混为一谈）。
- 若缺少任一重根，都会导致某类表象无法被解释（如否认动机的充足理由律，就无法解释意志行为的必然性）。


四、结论：四重根作为“先验框架”的完备性

- 叔本华的论证本质上是一种先验演绎：通过分析人类认知能力的边界（康德式的“批判哲学”路径），证明四重根是表象世界得以被理解的必要条件。
- 由于人类只能通过这四类表象认识世界，且每类表象必然遵循其对应的理由律，因此四重根是完备的、不可缩减的。
- 简言之，四重根的完备性源于其对“人类如何认识世界”这一问题的彻底拆解——从物理现象到逻辑思维，从数学真理到意志行为，所有可被认知的对象均已被纳入，且每种对象的解释原则均具有独特性和不可替代性。
