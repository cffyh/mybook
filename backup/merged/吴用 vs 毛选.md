# 吴用 vs 毛选

作者：遨游之舟

链接：[https://www.zhihu.com/question/361400163/answer/3041678857](https://www.zhihu.com/question/361400163/answer/3041678857)

读过《毛选》后，再去观照社会、历史或者小说演绎，真个是拿了个照妖镜，又如杀鸡用牛刀。

比如，以前看《水浒传》：吴用就是智多星，智多星就是吴用。

读了《毛选》以后看吴用：真个是“无用”。

作为梁山智囊，吴用之所以“无用”，是因为他的智是蛇鼠之智。有小聪明，无大智慧。

真正的智，是为梁山集团谋划奋斗纲领和路线方针，是搞顶层设计、搞战略研究。

就是《毛选》这一套。

毛主席就是党和人民军队的大军师大谋士:

研究中国社会各阶级图式，提出安定天下的“隆中策”；

研究建党建军的原则设计；

研究红色政权为什么能够存在；

研究星星之火如何燎原；

研究中国革命战争的战略问题；

研究打败日寇、走向胜利的道路方针；

研究延安整风，打造如身使臂、横扫四方的强大组织；

研究关起门来打狗占东北；

研究千里挺近大别山控中原；

研究世界头号强国美帝，为什么是纸老虎；

...

不仅如此。写出《实践论》《矛盾论》，明明白白告诉大家分析解决问题的传世心法。

......

以吴用反差对比毛主席，亦属不敬。

这里只为说明格局，探讨现实干事创业中如何安放我们的思考力。

吴用这个人，你说他聪明，他确实超过普通人。领导提目标，他马上拿方案，贯彻领导的意图和私心非常给力。

你说他有用？哪集、哪页纸你看见他研究过梁山的建设原则和远景发展？

一个组织的挫败乃至覆灭，不是战略有问题，就是策略有问题，这都离不开谋士。

而他，拿不出正经的战略和策略。

总结吴用的计谋案例，无非是上升到理论形态的江湖经验:盗、抢、骗。

内透阴毒。而且，往往给对手留下破绽。

如:

智取生辰纲，移花接木下药。

赚卢俊义上山，假扮算命老道。

浔阳楼事件，为救宋江，伪造蔡京书信。

逼朱仝、秦明等上梁山，杀人栽赃断其退路。

号称团队智多星，领导有需求，我就有办法，但是围绕组织的顶层设计和战略分析从来没有见解。

谁用谁败。除了黑社会，正经人谁用？

...

读《毛选》之前，并没有看出吴用的问题，只觉得是智多星，会出主意。

读了以后，复盘对比，方才大悟。

**顶层设计+战略分析，这就是《毛选》的格局**

——真正的知识分子的格局，搞组织的格局，创业和管理的格局，看社会的格局…思考和解决问题的格局。
