# 为什么有的人永远都学不好数学？

说句实在话，在当前教育模式下，你需要把数学当做文科学，背诵，记忆，重复，再重复，大量重复，最后熟练套用公式求解题目，千万不要自己思考。然后，你会发现成绩好了，题会做了，公式会用了。数学学习完成了。

这就是最优路线。不要轻信什么理解，图像，本质的话术，因为这些东西跟考试没有丝毫关系。不过，这些东西其实和数学没有丝毫关系。因为你看到的永远是一个黑盒子，不知道里面是什么。

当你开始对考试，做题感到厌烦的时候，反而，你会想要知道数学到底是什么。当你开始拒绝所有已知的规律，定理，解题方法，而选择自己去寻找他们的时候，你就明白了真正的数学是如何研究的。

为什么这么多的人学不好数学？因为太贪心了，紧抓着背诵过的东西，又想搞清楚为什么这些东西是对的。邯郸学步，形容的很贴切，当前教育模式就不是让你理解的，只是为了熟悉，然后使用，千万不要对此寄予过高的期待。而且，在长期的训练下，你的那种探索能力早已退化了。

我在网上见过许许多多数学还不错的人，他们的体会是，上学时从来没有体会到数学是什么，在实践，工作，生活中才逐渐明白什么是数学。而且，明白之后才发现一切都是数学。

数学从来不是解决问题的方法，就像魔法一样，Duang的一下出现不可思议的效果，而是**认识世界的方式，所见所及都可以用数学描述，时间长短，形状大小，距离方位，可行方式，用时多少，最优选择。**

而人们一直把数学视为工具，就像镰刀锄头一样，用完即弃，自然永远不可能领悟到数学的思想。
