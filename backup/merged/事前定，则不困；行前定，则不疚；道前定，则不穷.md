# 事前定，则不困；行前定，则不疚；道前定，则不穷

“凡事预则立，不预则废”这句话，各位可能都听过。但这里的“预”，到底包括哪些内容，倒不见得都真正深入思考过。按我们之前在[知乎 Live - 全新的实时问答](https://www.zhihu.com/lives/853976666937102336)里面讲的方法，我们来深入地拆一拆“预”的内涵。

对“预”最直觉的理解，就是准备。但如果对这句话的理解，只到“做什么事必须有准备”这个感性的层面，就实在有些浪费古人一番苦心了。所谓“有心栽花花不发，无心插柳柳成荫”，回顾个人经验，总有你做了准备而未成的事，也总有你无心却意外成了的事。但倘若你因此就否定了 “凡事预则立，不预则废”这个看似绝对的说辞，却也是误会。

“凡事预则立，不预则废”出自《礼记•中庸》，千百年来，作为经典流传。其生命力本身，就值得我们多花些时间，推敲一下，是什么样的“预”，有则使其事成，无则使其事毁。按照“察消长之往来，辨利害于疑似”的要求，我们需要先找到“预”在横纵向逻辑上的位置。

横向逻辑与纵向逻辑相比，更容易被忽略和混淆。

这就是一个典型例子。纵向来说，我们知道“预”的时序在做事之前，做事又在成事之前，因此得出了先要做“预”才能事成的结论。但如上文所述， “有心栽花花不成，无心插柳柳成荫”情况的客观存在，对这个结论形成了挑战。这个时候，与其分分钟倒戈，颠覆此前判断（俗称毁三观），不如试试“辨利害于疑似”，在纵向逻辑“预即准备”的基础上，细化一下横向逻辑。换言之，横向来说，准备本身也有丰富内涵，那什么样的准备才是预？

要回答这个问题，第一件事，就是回到原文。这句话还有后文，

“凡事预则立，不预则废；言前定，则不跲（jiá）；事前定，则不困；行前定，则不疚；道前定，则不穷”。

这几句，实际解释的就是什么叫”预”： 说话先有准备，就不会理亏词穷；做事先有准备，就不会遇困于窘迫；行事前有计划，就不会后悔莫及；道理能够提前理顺，就不会陷入绝境。然而， 至此，什么准备是预，还是不够清楚。要再细化下去，就必须要拆解逻辑链。

拆解一下上面提到的各种准备，再根据内涵合并同类项，可以发现，这里的预，包括素材置办、做事计划及应急预案。“言前定”，是说话准备好要说的内容，就不会哑口无言，这讲的是素材准备，和要做饭先买菜，要建楼先搬砖是一个道理。 “事前定”，做事要提前订好计划，这讲的是依计划行事，和按菜谱做饭，按图纸建楼是一个道理；“行前定”和“事前定”有一定部分的意义重合，重合的部分归入上一条，不重合的部分则是预案，俗称Plan B，这就相当于提前想好，饭煮糊了可以叫个外卖，建楼人手不够了可以再招，并依照预案和外卖店家和包工头打好招呼，以避免临时出事再“悔不该当初”。 这三点，都不难理解，最后那句“道前定，则不穷”，才是重点，也是这句话的精髓所在。

我之前在讲道术器的时候，说过道这个东西，看起来人人可懂，但实际上，必得有一定经历和思考的人才能真正明白个中深意。思考和经历无论哪个没到，都可能出现听了很多大道理，还是过不好自己这一生的情况。

所以，要理解最后这句“道前定，则不穷”，仅靠原文也是不够的了，还需要有一点博观和厚积，这里和大家分享一下我的理解过程。听过live[知乎 Live - 全新的实时问答](https://www.zhihu.com/lives/853976666937102336)的同学，应该知道，在做事始终的问题上，我还引过一句话，叫“思其始而成其终”，作为我对自身学习经历的一个总结和思考。这里我们就以此为切入，来还原对“道前定，则不穷”的理解过程。所谓思其始而成其终，最主要的意思，就是从开始，就要想到最后结果，才能成事。同样的意思，用比较官方的话来说，就是目标导向；用比较文艺的话来说，就是不忘初心，方得始终。不论你知道哪种说法，只要能在思考“道前定，则不穷”的时候调用出来，都是殊途同归，足够。

至此，我们可以开始逻辑链重建。把这句话，结合之前拆出来的那三个内涵来看，你会发现，目标导向可以作线，串联起三个看似独立的方面。物料置办，是根据最终目标倒推出来的；计划是围绕目标展开的；预案也是为了在特殊情况下仍然最大限度实现目标而存在的。如此，你就把“思其始而成其终”、“凡事预则立，不预则废”和“道前定，则不穷”联系起来了。各自代回原文，答案已经呼之欲出了：“预”即以目标为导向的准备，具体来说，是根据目标准备材料、制定计划和预案。

重建完了，我们再回去谈之前“有心栽花花不成，无心插柳柳成荫”的问题。有心栽花可知花才是目的，而最后花不成而柳立，说明准备的时候对花这个目的所需的物料条件、计划和预案不足，甚至目标方向出现了偏差，才意外得柳。得柳也不是凭空而来的结果，是物料条件、计划和预案错配而产生的意外结果。 此时再看“凡事预则立，不预则废”，大可不必全盘推翻，甚至更觉有理了。

这就是思考层次不同，对判断带来的影响。

思考流于肤浅，就容易被外界刺激毁三观，刺激如果过多，很可能出现三观来回摇摆的纠结。但如果思考深入到一定程度，就能够用理论解释现象，甚至形成足够强大、足够有包容性的理论体系。在这种情况下，外界刺激，不过是促进该体系发展、创新的推动力，并不会形成颠覆性力量，这种状态，俗称内心强大。

再回到“凡事预则立，不预则废”本身。经过刚才一番讨论，我们知道“预”在纵向逻辑上是准备，在横向逻辑上包含是以目标为导向的前提。换言之，这个“预”字，不止“准备”，更深一层的含义是“与目标相应的准备”。

如果只知道准备，却常忽略了其与目标的一致性，就会在做事的过程中迷失，出现刻舟求剑、缘木求鱼这类拘泥于错配物料、计划和预案的状况。

比如说，很多人想当然的认为，要实现学好英语的目的，就要背单词，于是开始准备单词书，制定背单词计划，规定自己每天背多少单词，如果按计划背单词还没有效果，那就Plan B报交钱上辅导班，让别人来为自己的成绩负责。最后付出了很多时间精力和财力，可能效果也并不如意，这就是忽略了准备与目标一致性的后果。具体解决方法，live和公众号文章都已经有清楚的介绍，这里不在重复，新关注的同学可以看《英语不好？可能是单词贝多了》或者看方法论相关的live课程[知乎 Live - 全新的实时问答](https://www.zhihu.com/lives/853976666937102336)。
