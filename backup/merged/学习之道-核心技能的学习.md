# 学习之道-核心技能的学习

[身心合一的奇迹力量](./身心合一的奇迹力量.md)

**别去预测结局,准备好随机应变且能够在高压下发挥的能力**



拆解学习之道

他的学习之道，不只是教你如何进行一项技能层面的练习，还包括了成长观念的改变以及心理层面的练习。

技术层面的练习固然重要，它决定着你能不能走，但成长观念的改变以及心理层面的练习决定着你能不能走得更好、更高和更远。所以本书的大部分文字都是在说成长观念以及心理层面的练习。

按照个人的理解，我对作者的学习之道拆解成以下三个部分。

一个是心态层面，一个是情绪调节层面，一个是技能练习层面，三个方面相辅相成，最终达到万物合一的状态。

3.1 首先是心态层面的改变

本书提到了以下几个重要的成长理论。

3.1.1 渐进理论

什么是『渐进理论』？

说到『渐进理论』，就必须得谈到与之对应的『整体理论』。

简单来说，具有『整体理论』的人，将成败归结于一种与生俱来、无法改变的能力水平，把自己的综合智力或技能水平看成是一**个固定的无法演变的整体，如『我很聪明』，当他们失败了，他们会急躁不安，容易放弃；**

而『渐进理论』的人会说『我之所以做到是因为我非常刻苦』，失败了会认为『我应该更努力一点才是』，他们的想法是世上无难事，只怕有心人，通过努力，一步步循序渐进，新手也能成为大师。

3.1.2 以退为进

什么是『以退为进』？

以退为进，是说在学习的时候，**要有先输后赢的心态，先要充分地释**放自己的自尊心，允许自己**东倒西歪，允许自己失败，允许自己退，从而来学习如何不做抵抗**。在输的过程中，主动地/不抵抗地接受这个"输"，可以**让人的恐惧感消失，进而有足够的精力去学习如何吸收这些失败，如何去应付这些失败，进而学会如何攻击。**（为何要学会"不抵抗"呢？因为抵抗的状态会让自己进入一种高度的压力状态，形成漩涡效应，容易因为一个错误引发更多的错误，最终走向失败。）

**以退为进和渐进理论也是相辅相成的，只有拥有"成功需要努**力"的这种心态，我们才会接受自己在某一段时间内处于一种被打击被瞧不起的状态。当我们接受这种低自尊的状态，认为这是我成功的路上必经的一步的时候，就不会真的难受了。大丈夫，能屈能伸。

3.1.3 化困境为优势

作者在学武术的时候有一次右手意外骨折，但右手对于习武之人来说非常重要。在这样的困境之下，他练习左手熟悉并精通武术动作，这让他形成了一种全新的武术交流概念：通过单手牵制住对方的双手，让对方陷入极大危险中，用另外一只手打败对方（这个原则适用于所有身体接触的运动项目）。后来作者在自己的武术生涯中，每次身体受伤，基本上他总会在受伤的第二天就回到训练场，都想着如何利用这种新情况来提高某些方面的技能。

有人说过一句话，你所获得的知识不是你自己的，而是属于全人类的。一方面，我们有义务去分享自己的收获，另一方面，你会发现不只是你一个人发现这些知识和智慧。优秀的人都会从把困境化为优势，不会错过任何一次困境。

3.2 心理层面的练习

为什么心理层面的练习很重要？

当人犯了一个小小的错误，错误发生了之后，人会习惯性地呆在之前的情感舒适区域，可是也会产生一种令人不安的预感，这样就会接踵而来的，像湍急漩涡一样，第二个错误，第三个错误甚至第四个错误，会导致一系列灾难性的后果，事情已经变得越来越糟了，所以在关键的时刻保持冷静是可以转败为胜的。

接下来的心理层面的练习就包括，包括初阶练习：如何保持冷静和自在从容的状态，进阶的练习：如何化情绪为力量，激发最佳状态。对于情绪，要先接受它，再利用它。

3.2.1 软区域

练习这样的软区域，主要是为了避免突发状况而分心。

什么是软区域？先说说啥是硬区域。当自己正专心地做手头上的事情的时候，可能有一些小事打断了自己，而这种专注状态的本质决定了自己的第一反应，比如可能会很紧张，身体紧绷，要阻止自己分心，这样就进入了一个硬区域，也就是说要求有一个空间让自己继续保持这种工作，而这种压力的状态就像一个干树枝，非常脆，很容易就会断裂。

软区域就是静静的高度的专注，看起来很放松，但精神世界却激流暗涌，平静接受眼前的事，将生命中的每一圈涟漪都融入自己创新思维过程中去。这就是软区域，软区域就像是一颗柔韧的叶草，在静风中左右摇摆，以顽强的生命力幸存下来。

因此我们要学着心平气和对待一切已发生的事，之后我们要学着将这些事情为我所用，最后我们要学着做到完成自给自足，创造出一个扰乱对手而不影响自己的环境。比如作者的下棋风格就是创造混乱局面，但他自己则通过软区域的训练可以做到冷静的对待混乱、不清晰的局面，较好地走出混乱，战胜对手。

3.2.2 自在从容

在每一个领域区分强者与弱者的标准，很大程度上取决于在危机关头是否能够保持清醒的头脑，从容自如。在比赛当中，如果一个选手表现的从容不迫，而另一个选手已经被心理因素摧垮，那么比赛的结果已经不言而喻了。

每件事都会有面临危机的可能，如果我们还想追求卓越，那么我们必须适应自我激励和控制的生活方式，从容应对所发生的一切，像呼吸一样自如。要想学会抓住最微妙的机会以致胜，唯一做的就是坦然接受痛苦，直到能够将让人麻木的头脑煎熬，当成是闲庭信步。

作者过去比赛中常制造麻烦，让他的对手因为压力而崩溃。他喜欢未知的局面，喜欢创造问题，当没有答案的时候他甘之如饴，而对手却被吓到了，如此一来他成了游戏的掌控者，随后他所设下的心理棋局变得更加复杂起来，胜利也就在手了。

3.2.3 化情绪为力量

在比赛中选手们都会遇到很多负面情绪如愤怒，恐惧等，这些负面情绪会有因为漩涡效应导致整个过程变得越来越差，开始崩溃，走向失败。

我们要学会将本能的反应转换成力量，最大程度地发掘深藏在内心的力量源泉，而不是逃避自己的情绪或是被他们完全控制。将正面的情绪转换成力量并不难，但是身处险境时，要把负面的情绪转化成力量则需要心态和练习。我们不应该被愤怒控制或者否认自己的愤怒，而是必须从某种程度上将愤怒引导到一种极度全神贯注的状态。

我们需要容忍混乱，并且不再为膨胀的情绪所控，如此便能够驾驭他们，甚至在可能的范围里发挥他们的作用。一旦你在压力下能够不被情绪所控制，并且能善于控制，你将会发现某些心理阶段可能会对你产生比其他人更大的激励作用。

承认现实，超越现实，鼓起勇气并善用之，我们必须有事实并非完美的心理准备。

3.2.4 释放压力

释放压力是提高生活中方方面面的一个最重要的根基，作者说这就是他全部学习方法的根基。

如果在日常生活中花几个月来练习面对和释放压力，那么你就会成为一个临危不惧，活力四射为人所信赖的选手。比如在象棋比赛中，不必时时刻刻都把自己放在一个高度集中的状态中，当对手思考的时候，没有必要强迫自己处于一个全新关注局势的状态，让大脑释放一些压力。

比如在身体的训练中，每一个部分都是围绕着一种或者另一种形式的施压和恢复。这样训炼者通过及时持续的努力，达到了肌肉增强的目的，休息的间隔时间也能逐步减少。锻炼一段时间之后，在身心疲惫之后，就能够在更短的时间内，更好地恢复自己。

在生活中的应用，包括如果在读书的时候目光涣散就把书放下，深呼吸一会，再拿起书立刻双眼有神；如果在工作中发现自己大脑呆滞没有活力就休息一会儿，洗把脸回来就面目全新。放松就是一眨眼的功夫，除了能够增加身心的活力，也向你开启了一些美妙的令人惊奇的窗户。有创造力的思想会从无意识的头脑中冒出来。你会更加适应自己的直觉，会在风格上更忠于自我。

**无意识的脑袋是一个大有潜力的工具，怎样在压力下放松则是挖掘这一潜力的关键**的第1步。

3.2.5 激发最佳状态

职业运动员都有一些固定的方法，不断地在比赛前将自己调整到最佳状态。但是如何找到那种固定的方法呢？

首先要思考自己在生活中有什么事情能够让你最接近内心的平静、达到注意力集中的最佳状态。让内心平静的活动是什么样的形式不重要，达到那种状态才最重要。

接下来就是制定一套固定的修炼程序，多练习，然后慢慢的对这个修炼的流程进行缩短，优化和调整，把它变成一个压缩过的固定修炼流程。让自己在时间紧急的情况下，能够采用这个压缩过的流程，在短时间内调动自己的最佳精神状态。

3.3 技能练习方法

3.3.1 划小圈

学习的基本原则就是要钻研微观的细节事物，来理解是什么促成了这些宏观上的问题。

什么是划小圈？重点是在对外部技能进行细微的培养。这个过程**指：挖掘技能的实质所在，然后有效地压缩技能的外在表现同时又紧紧围绕技能的内在实**质。

有一个例子就是英国象棋大师，他一般下棋的时候都着重在侧方位，而把中盘的控制拱手交给对方，他其实是知道如何控制中盘而又表现的和中盘毫无关系，最后却是他赢，他把圈划得如此之小，就连象棋大师也看不出来。这个小圈的意思是**他把他自己控制在一个比较小的位置上，但是他却通过这么小的一个区域，控制了别人更多的东**西。这就是重视细微的吸收和完善，这样比学到的招式数量多，要重要的多。

作者说他能成为顶尖选手并没有什么秘诀，而是对可能是基本技能的东西有更深的理解，每天都要学得更深一点，而不是更广一点，因为学到更深可以让我们潜力中那些看不到感受不到而又极具创造力的部分挖掘出来。

他的整个学习过程的核心在一定程度上是受到了罗伯特.波席格的**摩托车维护艺术这本**书的启发。主人公文德罗，要帮助一个女生来解放她的思维，这个女生需要写她所**居住的小镇的故事。他让她从教室外面那个剧院的前台左上方的砖开始写**，结果这个学生变得文思泉涌，停都停不下来。

3.3.2 让时间慢下来

比如作者在练习象棋的时候，不是从标准开局开始，而是从一个只有王和兵的空白棋盘开始学习这两个棋子该怎么走，然后不停的练习这两个棋子走法，一直到自己感觉到走得很舒服为止，接着再慢慢地单独学其他的棋子的走法，然后他就很快熟悉了，象棋中各个棋子的走法和价值的所在，他不用有意识的去想这些棋子，而是可以在看到棋子的外形的同时，还能想到他们的潜能。

为什么说是让时间慢下来？当一个人仔细地练习各个步骤，一遍又一遍的重复练习，完善自己的节奏和准确性，然后把各个动作融合起来，并且成百上千次地进行练习**。一段时间之后，这个技能的每个步骤都会在他脑海中发展成为越来越详细的细节，**虽然整个动作在客观的时间上看进行地非常的快，但是对于练习精通的这个人来说，却拥有着非常多的极其熟悉的步骤，就像时间慢下来了一样。当然，这一步的关键还是在于不断的练习。

3.3.3 揭开神秘的面纱——如何去操纵对手的意念

在象棋比赛当中，自己的一些表现都会影响对方的心情和判断。所以大多数人在比赛的时候都尽量不去表现出自己的情绪变化，不给对手了解自己并操纵自己的机会。而作者则努力地将这个特点为他所用，大大方方地把自己思维过程中的表情摆给对方看。**利用自己的情绪来支配这场博弈的基调，当然中间也会穿插一些故意做出的误导性的表情，来影响对方的行动**。比如在武术比赛中，他故意让对手的自大膨胀起来，然后掉入他的圈套。接下来利用他的这种自大来打败他。

作者在生活中也仔细的去观察对方的节奏，对方的生活习性跟比赛场上的作风也是一致的。比如说一个人在餐厅排队时等待表现出来焦躁，就说明这个人面对紧张压力缺乏耐心。

领悟这些心理交锋的技术基础是很关键的，操纵对方的意念实际上并没有什么神秘色彩，这只是和诸多技巧一样可以被逐渐培养的能力。

万物合一

调整好心态，做好练习，从容不迫，利用情绪与困境的力量，划小圈进行练习，操纵对方的意志，达到一种万物合一的状态，赢得比赛。
