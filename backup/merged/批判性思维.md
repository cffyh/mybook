# 批判性思维

（1）我们的世界是复杂的，难以简单把握的

从自然层面看，我们所存在的世界，大而复杂到难以想象。宇宙从138亿年前诞生至今，地球作为一个行星起源于46亿年前的原始[太阳星云](https://link.zhihu.com/?target=https%3A//baike.so.com/doc/5350595-5586051.html)，此后，地球系统由简单到复杂，各个组成部分的[演化](https://link.zhihu.com/?target=https%3A//baike.so.com/doc/24036930-24618727.html)既相互联系又相互影响。地球系统的运动及其带来的[地貌](https://link.zhihu.com/?target=https%3A//baike.so.com/doc/200760-212262.html)变迁、[生命](https://link.zhihu.com/?target=https%3A//baike.so.com/doc/2208567-30289284.html)现象和生命活动共同构成地球的[历史](https://link.zhihu.com/?target=https%3A//baike.so.com/doc/3451443-3631913.html)。

从社会层面看，国家193个，地区31个，人口76亿，按颜色区分有白种人、黄种人、黑种人、棕种人，2000多个民族，5000余种语言，意识形态种类繁多，有[无政府主义](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E6%2597%25A0%25E6%2594%25BF%25E5%25BA%259C%25E4%25B8%25BB%25E4%25B9%2589)、[资本主义](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E8%25B5%2584%25E6%259C%25AC%25E4%25B8%25BB%25E4%25B9%2589)、[社群主义](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E7%25A4%25BE%25E7%25BE%25A4%25E4%25B8%25BB%25E4%25B9%2589)、自由主义、基督教[民主主义](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E5%259F%25BA%25E7%259D%25A3%25E6%2595%2599%25E6%25B0%2591%25E4%25B8%25BB%25E4%25B8%25BB%25E4%25B9%2589)、[法西斯主义](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E6%25B3%2595%25E8%25A5%25BF%25E6%2596%25AF%25E4%25B8%25BB%25E4%25B9%2589)、[帝国主义](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E5%25B8%259D%25E5%259B%25BD%25E4%25B8%25BB%25E4%25B9%2589)、[民族主义](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E6%25B0%2591%25E6%2597%258F%25E4%25B8%25BB%25E4%25B9%2589)、[纳粹主义](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E7%25BA%25B3%25E7%25B2%25B9%25E4%25B8%25BB%25E4%25B9%2589)、[保守主义](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E4%25BF%259D%25E5%25AE%2588%25E4%25B8%25BB%25E4%25B9%2589)、[社会主义](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E7%25A4%25BE%25E4%25BC%259A%25E4%25B8%25BB%25E4%25B9%2589/296)、[社会民主主义](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E7%25A4%25BE%25E4%25BC%259A%25E6%25B0%2591%25E4%25B8%25BB%25E4%25B8%25BB%25E4%25B9%2589)等等，有五大宗教：犹太教、基督教、伊斯兰教、印度教和佛教。

从知识层面看，科学革命之后，尤其是到了21世纪计算机的广泛运用之后，根据不确定统计计算，全世界的知识总量,七到十年翻一番，人类所有的知识划可以分为五大门类：自然科学，农业科学，医药科学，工程与技术科学，人文与社会科学，每一个大门类下又可以分为无数的小类学科，知识量简直浩如烟海。因此，无论从自然、社会、知识三个层面来看，世界纷繁复杂，难以度量。

（2）我们的所得到的意见是纷乱的

世界纷繁复杂，让我们难以捉摸，但这还并不可怕，可怕的是，这个世界上的意见和道理，比世界本身更多更杂更乱。互联网的普及和手机的普遍使用，每个人都可以轻易的往网络上输出意见，也可以从网络上获取大量的信息，往往一件事情，你会得到完全相反却又都很有道理的道理，你应该听谁的，照谁的去做？毕竟说话不用负责任，做事是需要承担后果的，如果我们不加思考的盲目去做，我们就会变成一具提线木偶，过着毫无意义的人生。

（3）我们的思维存在缺陷

我们的思维存在惰性，更愿意被动式吸收信息。这样的思维方式因为类似于海绵被放到水中的反应—充分吸收水分，而被称为“海绵式思维”。海绵式思维的特点是，照单全收一个不落，优点是轻松快捷高效，但它的缺陷也是显著的，你无法判断什么该信什么不该信，最后就是读到什么就信什么。而要解决这个缺陷，可以尝试建立“淘金式思维”，所谓淘金式思维，是带着特定的态度去接受信息，即不断提问（发问）的态度，它强调一个提问、质疑、互动的过程，对所接触到的知识采取批判性吸收的方式，尽管更多时候，会花费掉我们的精力和时间，但我们认为，是值得的。

我们的思维还存在排他性，更愿意接受与自己相契合的信念，排斥与己不同的信念。当我们从全盘接受转化为懂得进行批判性接受的时候，又容易陷入到另一个困境，即排他性的思维方式，我们称之为“弱势批判性思维”，弱势批判性思维是利用批判性思维来捍卫自己当前的信念。如此，我们就会成为一个自包的系统，不断地自我完善，但边界也十分清晰，缺乏突破性。但如果我们希望能够追求更好的结论、更好的信念，以及更好的决定的话，我们就需要进化为“强势批判性思维”，即利用批判性思维来评估所有的断言和信念，尤其是对自己的断言和信念加以评估。

这，也许可以成为我们学习批判性思维的主要理由。

二、如何习得批判性思维？

所谓的批判性思维，其核心宗旨在于对一个意见、观点、道理进行“评价”，而一个道理要成其为道理，不是仅仅依靠一句简简单单的陈述即可完成，而是需要完整的论证结构，因此，准确的说，批判性思维的目标就是，对一个论证结构进行全面细致的评价。

需要评价一个论证结构，就需要清楚，一个论证包括了哪些核心要素？答案是，论题、结论和理由。所谓论题，即一个论证想要回答的问题，结论就是对该问题的回答，而理由，则是对该结论的支撑陈述。

（一）论题和结论

论证根据论题或结论的类型来区分，主要有两大类，一类是描述性论题，一类是规定性论题。描述性论题是指针对过去、现在或将来的各种描述的精确与否提出的问题。规定性论题是指针对什么该做什么不该做，什么是对什么是错，以及什么是好什么是坏所提出的问题。简而言之，前者确定的是是什么的问题，后者确定的是应该如何的问题。

之所以把二者放在一起讨论，是因为问题和答案是密不可分的，在绝大多数的论证里，立论者会明确的把问题和结论明示出来，而有些时候就不一定会明示问题，但一个论证是肯定会有结论的，因为论证的目标就是输出立论者的结论。所以，找到结论是关键。

如何找到结论？当我们准备寻找一个论证的结论时，可以先问自己一些问题，作者提出了什么论题？作者想证明什么？文章的主旨是什么？对这三个的问题的解答，就是结论。

从技巧上来讲，我们还可以根据一些线索对结论进行找寻，第一是找到论题，也就是作者明确表示出来的要准备论证的问题；第二，通过一些提示词找到结论，比如因此、表明、由此可知、由此得出、因此可以断定、我要说的重点是、显示出、证明、告诉我们、问题的实质是，等等；第三，一般来讲，结论往往出现在章节或段落的开头或结尾。

找到结论之后，我们就实现了评价一个论证的第一步，下一步则是寻找论证的理由。

（二）理由

评价一个论证，最核心的部分，就是对理由的彻底探究，我计划从如何找到理由、评断证据的效力、寻找替代理由三个方面对论证的理由进行探讨。

（1）如何找到理由？

何为理由？是作为支撑或证明结论的信念、证据、隐喻、类比和其他陈述。它包括事实、研究成果、生活实例、统计数据、专家或权威意见、当事人证言以及类比等诸多方面。理由，在一个论证结构里居于核心地位，一个论证的目的是阐述立论者的结论，而一个结论是否能够立得住，取决于支撑该结论的理由是否足够充分，而对论证理由的彻底考究，将是一个批判性思维者的主要工作，那么我们首先要做的，是找到一个论证里，立论者表达出来的诸多理由。

在寻找理由之前，我们需要问自己，我为什么要相信这个结论？并从这里开始启程我们的探究理由之路。

同理，从技巧上来讲，我们还可以根据一些线索对理由进行找寻。第一，通过一些提示词找到理由，比如：由于、因为这个原因、因为这个事实、鉴于、由以下材料支撑、因为证据显示、研究显示、第一，第二，第三，等等；第二，确定该论题是描述性还是规定性的，描述性结论的理由往往是直接证据，规定性结论的理由往往是一般性的规定性陈述或描述性的信念或原则。

（2）评断证据的效力

证据，是对事实断言的支撑。事实断言，是对“是什么”的陈述，它包括描述性的结论、用来证实描述性或规定性结论的理由、描述性的假设三类。因此来讲，证据是理由的核心部分，在一个描述性论证中，它负责支持结论以及支撑该结论的对事实进行断言的理由，在一个规定性论证中，负责支持理由，简而言之，证据是作为包括结论和理由之中对“是什么”的陈述的支撑。

要对证据进行评断，先需要找到证据之所在，如前所言，要寻找证据，首先需要找到论证结构中对事实的断言，而那些能够支持这些断言的陈述，即为证据。

证据的类型可以简单概括为8种，包括直觉、个人经历、典型案例、当事人证言、专家意见、个人观察、研究成果、类比。

在这八种类型的证据里面，效力最低是直觉、个人经历、典型案例、当事人证言、类比、个人观察。

直觉来自于个人的主观本能，即丹尼尔卡尼曼在《思考，快与慢》中的快思考，是不够可靠的；

个人经历仅代表个例，容易犯以偏概全的谬误；

典型案例同样来自于个例，需谨慎待之；

当事人证言，由于立场、利益、信息不透明的因素，可信度也不高；

类比可做参考，但需要考察立论者是否提供进一步证据支撑二者之间的相似程度的论据；

个人观察存在价值观和知识偏见，需审慎对待。

效力较高是专家意见和研究报告，前者存在可信度，我们需要对其做谨慎的考察，问一些批判性问题：此专家专业背景如何？是否对该问题进行多年可靠的研究？后者总体比较可靠，但仍然需要考察一系列问题：

1.研究质量有高低，要懂得筛选；

2.研究成果自相矛盾；

3.研究发现不能证明结论；

4.研究者自带主观要素；

5.歪曲或者简化研究结论；

6.研究的“事实”会随着时间的流逝而发生改变；

7.研究的人工程度怎么样，也会引起研究的变化；

8.对经济效益、社会地位、人身安全和其他因素的需求可能会影响到研究的结果。

另外，一个研究报告之中，如果涉及到样本统计，要考察样本的数量是否足够大、覆盖的类型是否足够广泛、采样是否随机等因素。

（3）寻找替代理由

我们在遇到立论者采用因果断言的时候，应该关注一个论证之中，是否还存在其他替代理由。所谓因果断言，就是使用证据支持立论者对某件事起因的一个断言。也就是说，当我们在一个论证中，发现立论者试图建立因果关系来支撑他的论证时，我们就应该思考一下，是否存在替代理由？

那么怎么办呢？可以通过几个技巧进行探究：

1. 确定是唯一的原因，还是原因之一？
2. 建立多重视角；
3. 组间差异是否存在其他变量？
4. 关注相关性与因果性的差异；
5. 关注前后关系与因果关系的差异；
6. 关注事件解释的多样性。

通过这些技巧，我们能够找到一个论证可能存在的问题，并能够对其作出恰当的评价。

三、还存在哪些被忽略的问题？

一个论证结构最为重要的三个要素，论题、结论、理由，已经得到了充分的探究，但这还不足以称其为具备了批判性思维，真正一个合格的批判性思维者，还需要对一个论证结构中，那些尚未明确表达出来，却又非常重要的部分进行进一步探究，这些部分主要有：假设、词语歧义、论证谬误，且一一分析。

（一）假设

所谓假设，就是一个信念，通常没有明说出来，而作者认为是理所当然，并用来证明外在的论证。主要有两大类假设，一类是价值观假设，一类是描述性假设。价值观假设，就是在特定情形下没有明说出来的喜欢一种价值观超过另一种价值观的倾向；描述性假设就是一种关于世界过去、现在和将来是怎么样的没有明说的信念。

我们为什么需要寻找假设呢？因为只有找到假设，我们才能够更好的接受并评价一个论证。我们只有知道立论者的价值观假设，才能够清楚他的站位和立场，并对他的理由和观点多一分理解；我们只有知道了立论者的描述性假设，才能够清楚他那些看似不证自明的理由背后仍然存在一个前提，也就能够准确的作出评断，下面我们需要寻找到假设的去处。

（1）寻找价值观假设线索如下：

1. 作者背景；
2. 论题中可能潜在的后果；
3. 类似的论争中存在的价值观假设；
4. 采取与作者相反立场，看看哪些价值观能够支持本立场；
5. 寻找常见的价值冲突。

（2）寻找描述性假设线索如下：

1.理由与结论之间的断裂口；

2.支撑理由却没有明说的说法；

3.把自己放到对立的反对立场;

4.对论题进一步学习了解。

（二）词语歧义

词语歧义，指一个（关键）词或短语可能存在多重含义的现象。要找到歧义，先找到关键词，线索如下：

1.以论题为线索，找关键词；

2.在理由和结论中找关键词；

3.注意抽象词语；

4.通过反串试图理解立论者对关键词的定义。

确定关键词是否存在歧义的线索如下：

1.查找字典标准定义；

2.注意上下文语境；

3.描绘该词所代表的具体景象。

（三）论证谬误

论证谬误，即论证中的欺骗手段，作者有可能利用这个欺骗手段来说服你采纳他的结论。主要存在以下几种类型：

1.人身攻击谬误

不讨论辩题，转而攻击立论者。

2.滑坡谬误

假设该方案成立后必将导致一系列不可控连锁事件发生，可实际上却有相关措施起到了预防作用。

3.追求完美解决方案谬误

认为该方案无法完美解决问题就要将其弃如敝履。

4.诉诸公众谬误

以大部分人支持某种观点作为支持自身观点的理由。

5.诉诸感情谬误

使用带强烈情感的语言来分散注意力，使人们忽略理由和证据，比如恐惧、爱国、怜悯、同情.

6.稻草人谬误

歪曲论点，使其变成另外一个莫须有的论题。

7.虚假的两难选择谬误

现实中存在多重方案，却假设只存在两种方案，非此即彼。

8.乱扣帽子谬误

错误的给相关特定事件作出一个概括并以为理解了本质。

9.计划谬误

低估了完成事情的进度，误以为可以按时按质完成。

10.光环效应谬误

给某人或某事带上情感光环，转移人们对理由和根据的关注度。

11.转移话题谬误

在甲命题正在被讨论时，突然插入乙命题，混淆视听。

12.循环论证谬误

该论证是在已经假设结论成立的前提下作出的。

四、总结

要建立批判性思维，首先是做好思想准备，意识到世界和信息的复杂性，以及人自身的思维惰性和排他性，进而明确建立批判性思维的重要性；其次，是把握住建立批判性思维的核心任务是对论证的有效评价，这需要从两个方面探究，一是对论证结构本身进行考察，二是对论证中容易被忽略的相关问题进行考察。只有通过这三步动作的具体实施，我们才可以说一句，我们开始进行批判性思考了。
