# 十大军事原则

⼀、十大军事原则

1947年底，解放军从战略防御到战略进攻的转化已经完成，但是战争最艰难的阶段并没有过 去。如果把解放战争⽐作爬⼭的话，最吃⼒的爬坡阶段还在继续。但毫⽆疑问的是，不管国⺠ 党还是共产党，双⽅都已经感觉到，拉开决战序幕的那⼀天已经为期不远了。

然⽽，就是在战争还处于这样焦灼、拉锯、严峻⽽⼜关键的时刻，⽑泽东将⼈⺠解放军的⼀系 列的战略战术进⾏了提炼和总结，提出了指导中国⼈⺠解放军作战的 **“⼗⼤军事原则”** ，并毫 ⽆顾虑的将其公之于众。

这⼀年的 12⽉25⽇，⽑泽东在陕北⽶脂县的杨家沟召开了⼀次重要的中央扩⼤会议，这次会议 上正式提出了“⼗⼤军事原则”。这“⼗⼤军事原则”已然不仅仅是军事指导，它更像是⼀种哲学 原理，其中蕴含着深刻的事物运⾏发展的普遍规律。

如此简单通俗，却⼜丰富深刻的原理，我们实在有必要将其摘录⼀下：

( 1)先打分散和孤⽴之敌，后打集中和强⼤之敌。

( 2)先取⼩城市、中等城市和⼴⼤乡村，后取⼤城市。

( 3)以歼灭敌⼈有⽣⼒量为主要⽬标，不以保守或夺取城市和地⽅为主要⽬标。保守或夺取城 市和地⽅，是歼灭敌⼈有⽣⼒量的结果，往往需要反复多次才能最后地保守或夺取之。

( 4)每战集中绝对优势兵⼒(两倍、三倍、四倍，有时甚⾄是五倍或六倍于敌之兵⼒)，四⾯ 包围敌⼈，⼒求全歼，不使漏⽹。在特殊情况下，则采⽤给敌以歼灭性打击的⽅法，即集中全 ⼒打敌正⾯及其⼀翼或两翼，求歼灭其⼀部、击溃其另⼀部的⽬的，以便我军能够迅速转移兵 ⼒歼击他部敌军。⼒求避免打那种得不偿失的或得失相当的消耗战。这样，在全体上，我们是 劣势(就数量来说)，但在每⼀个局部上，在每⼀个具体战役上，我们是绝对的优势，这就保 证了战役的胜利。随着时间的推移，我们就将在全体上转变为优势，直到歼灭⼀切敌⼈。

( 5)不打⽆准备之仗，不打⽆把握之仗，每战都应⼒求有准备，⼒求在敌我条件对⽐下有胜利 的把握。

(6)发扬勇敢战⽃、不怕牺牲、不怕疲劳和连续作战(即在短期内不休息地接连打⼏仗)的作 ⻛。

( 7)⼒求在运动中歼灭敌⼈。同时，注重阵地攻击战术，夺取敌⼈的据点和城市。

( 8)在攻城问题上，⼀切敌⼈守备薄弱的据点和城市，坚决夺取之。⼀切敌⼈有中等程度的守 备、⽽环境⼜许可加以夺取的据点和城市，相机夺取之。⼀切敌⼈守备强固的据点和城市，则 等候条件成熟时然后夺取之。

(9)以俘获敌⼈的全部武器和⼤部⼈员，补充⾃⼰。我军⼈⼒物⼒的来源，主要在前线。

(10)善于利⽤两个战役之间的间隙，休息和整训部队。休整的时间，⼀般地不要过⻓，尽可 能不使敌⼈获得喘息的时间。

⼗⼤军事原则，并不仅仅只是解放战争的产物，如⽑泽东 1958年时所说： “⼗⼤军事原则，是 根据⼗年内战、抗⽇战争、解放战争初期的经验，在解放战争进⼊反攻时期提出来的，是⻢克 思列宁主义普遍真理与中国⾰命战争实践相结合的产物。”

远在南京的⽼蒋内⼼受到了⼀万点暴击。在绝⼤多数⼈看来，战争的前景还远不够明朗，可是 共产党⼈却敢于在这个时候将其战略战术堂⽽皇之的公之于众，这完全就是对国⺠党军事战略 和战术的⾚裸裸的羞辱和鄙视。

如果羞辱和鄙视只是令蒋和其⼀众国⺠党官员感到屈辱的话，那么更加令他们感到痛苦和恐惧 的是，他们对于共产党⼈公开宣布的这⼗⼤军事原则简直束⼿⽆策。

有⼈疑惑为什么⽑泽东要将解放军的战略战术完全公开，其实关于这⼀点，⽑泽东解释得⾮常 清楚，他说：“蒋介⽯匪帮和美帝国主义的在华军事⼈员，熟知我们的这些军事⽅法。蒋介⽯曾 多次集训他们的将校，将我们的军事书籍和从战争中获得的⽂件发给他们研究，企图寻找对付 的⽅法，美国军事⼈员曾向蒋介⽯建议这样那样的消灭⼈⺠解放军的战略战术，并替蒋介⽯训 练军队，接济军事装备。但是所有这些努⼒，都不能挽救蒋介⽯匪帮的失败。 这是因为我们的 战略战术是建⽴在⼈⺠战争这个基础上的，任何反⼈⺠的军队都不能利⽤我们的战略战术。”

⼈家都把⾃⼰的战略战术甩你脸上来了，你不得回击⼀下以作表示吗？

蒋介⽯痛定思痛，他决定召集所有的国⺠党⾼官，提出⼀个切实可⾏的应对⽅案。你⽑泽东可 以把⾃⼰的战略战术总结为⼗⼤军事原则，那我也可以把我们的战略战术总结出⼀个更拉⻛的 名字，那就是—总体战。

⼆、总体战

你搞⼀个“⼗⼤”，我就搞⼀个“总体”。

事实上，蒋介⽯所谓的“总体战”，同样是与共产党军队⻓期作战⽽总结和发展出来的。还在 1947年11⽉4⽇的时候，蒋就指出，若要挽救危局，必须实⾏“总体战”。他在国⺠党国防部作 战会议上指出： “必须做到共产党军队的四个⻓处，即宣传、组织、主动和保密，这四个⻓处都 是属于精神⽅⾯的，决不是单纯凭武器和物质所能做到的。我们要赶上他，压倒他，也要从精 神⽅⾯努⼒，也就是先要从政治⼯作做起。”

如果只是从这段话听上去，不知道的可能还以为是某个被“⾚化”的国军将领说的。不得不说， 当全⾯战争打了⼀年多以后，不管是蒋本⼈还是国⺠党中的⼀众⾼级将领们，多数都已经收起 了战争之初时的⽆知与傲慢，开始虚⼼学习和研究共产党⼈的战略战术。蒋本⼈更是当众对他 的将领们说，国军常遭失败，甚⾄将领被俘的原因是：“共产党打的是⼈⺠战争，⽽我军打的是 单纯的军事戡乱，没有发挥政治的，经济的，思想的和军事的总体威⼒。”

蒋介⽯之所以那么说，是因为他已经从战争中看到了⼈⺠战争的切实威⼒，他也想学着共产党 ⼈，把单纯的“军事戡乱”，改为总体的“⼈⺠战争”，但是他⼜不能说⾃⼰也打的是“⼈⺠战 争”，所以便要求⾃⼰的将领们也学习共产党转变⾃⼰的单纯军事战略，改为集中全部⼒量的 “总体战”。

可是他们都忘记了⼀点，也不得不忘记的⼀点是：从 1927年国共分裂那⼀刻开始，它便意味着 国⺠党和共产党各⾃依靠的社会阶级已经清晰地进⾏了划分。⼀个依靠⼤地主、⼤资产阶级、 ⼤买办阶级的政党，如何站在少数⼈的⽴场上，去要求全体⼈⺠动员起来，打什么所谓的“总体 战”呢？

这个“总体战”，从外表和形式上，似乎也是要求发动全体⼈⺠，全体将领的协同作战威⼒。但 是它到底和真正的⼈⺠战争有什么区别？这个问题实在是太重要了。重要到如果我们不搞清楚 这个问题，就容易被欺骗，就永远不能知道，到底什么样的战争才是真正的⼈⺠战争。

那么，我们先看⼀下，国⺠党搞的这个“总体战”到底是什么？

它分为四个⽅⾯，军事战、政治战、经济战、思想战。

在军事上，国⺠党撤销了原来的战区司令官，⽽是把各战区的党、政、军权进⼀步实⾏⼀元化 的集中。实际上，就是将各战区权⼒更进⼀步集中起来，以⽅便在政治、军事、经济上的统⼀ 调动和部署。

在政治上，其基本内容是“强化保甲制度”—“组织⺠众，⼈必归户，户必归甲，甲必归保，⽽ 不遗漏⼀⼈，不分散⼀分⼒量。”政治战的⽬的是：全⾯控制⼀切⼈⼒物⼒，集中⼒量办⼤事， 团结⼀致对敌作战。

在经济上，其内容就⾮常有趣了，实际上在“总体战”提出之前，国⺠党就已经开始尝试了，这 项措施就是⼟地改⾰。国⺠党看到了共产党进⾏⼟地⾰命的巨⼤威⼒，于是也开始效仿，名 ⽈：实⾏孙中⼭平均地权的主张。可是国⺠党既然是⼀个代表地主阶级的政党，它就不可能真 正的损害地主阶级的利益，但是它⼜想拉拢群众，怎么办？只能幻想两边不得罪，搞出了⼀个 分期付款的⽅案。这个⽅案我们此前就讲过，它的⼤概意思就是给农⺠搞⼀个零⾸付买⼟地的 ⽅式，农⺠只需要耕种⼀定的年限，以粮⻝或⾦钱的⽅式分期付款给地主，将来就可以获得这 块⼟地。可是问题是，原本租种⼟地的模式没有任何变化，农⺠的收⼊还是那么点，甚⾄还在 下降，该交的租⾦还得照交，你即使给他们零⾸付，他们⼜哪⾥有分期偿还贷款的能⼒呢？⼀ 旦贷款还不起，最后可能连原本有的⼀点家当和⼟地也会被地主卷⾛。农⺠怎么可能从这样的 ⼟地政策中真正受益？所以，表⾯看谁也不得罪，其实不过是涸泽⽽渔的⼀个办法。

在思想上呢？已经毫⽆可取之处，不过是⼀⽅⾯继续加强“振奋精神，艰苦奋⽃”的宣传，另⼀ ⽅⾯通过特务系统压制⼀切“⺠怨沸腾，群众不满”的⾔论。

从国⺠党这套“总体战”的⽅案⾥，有的⼈可能看懂了，但是有的⼈可能还没有看懂，这个“总体 战”与“⼈⺠战争”到底有什么区别？

从形式上说，总体战也是发动全体将领和全体⼈⺠，集中全部⼒量的⼀种策略，但是为什么我

们说它本质上跟⼈⺠战争完全不同？



这⾥⾯有三个原则上的不同：

第⼀，“总体战”强调权⼒的绝对集中，以便统⼀管理，统⼀调动，统⼀执⾏。可问题是，他们

在强调集中的时候，却丝毫不提⺠主。⼈⺠战争是什么？⼈⺠战争也强调集中，不集中⼒量就

⽆法形成合⼒，就不够强⼤，但是⼈⺠战争的集中，是在⺠主基础上的集中。⽐如，就在国军部署着“总体战”的时候，⼈⺠解放军却开始了新式整军运动，这个整军运动⾥⼀个核⼼步骤就是开展军队⺠主。它包括发动⼠兵检查⼲部，推荐⼲部与建议撤销不称职的⼲部，由⼠兵委员会和军⼈代表会来进⾏。在战术研究中，发动⼠兵讨论战术策略，开“想办法会”，开“诸葛亮会”，甚⾄允许某些情况下⼠兵讨论作战命令。战役结束后，往往由⼠兵们来评指挥，评战术，评作⻛，肯定优点，指出缺点。在经济上，基层⼲部可以通过⼠兵委员会或经济委员会，协助⾸⻓管理伙⻝，监督经济开⽀，定期公布账⽬，防⽌贪污、浪费和侵占⼠兵利益。

有了这样的基层⺠主基础，再强调作战时的集中指挥，那就和单纯的权⼒集中有了原则性的不

同。

第⼆，共产党在发动⼈⺠的集体⼒量的时候，是通过什么⽅式呢？是党的领导。⽽国⺠党在发

动⼈⺠的集体⼒量的时候，⼜是通过什么⽅式呢？是党的强制。虽然他们也美其名⽈“领导”，

但依然改变不了强制这个事实。⽽强制的⼿段就是“保甲联防制度”。领导和强制表⾯也有相似

的地⽅，但实质的区别在哪呢？

强制遵循的原则是由党来包办代替⼀切，为践⾏这个原则，所以“总体战”的做法是“组织⺠众，

⼈必归户，户必归甲，甲必归保，⽽不遗漏⼀⼈，不分散⼀分⼒量。”

⽽领导遵循的原则是什么呢？是⼈⺠⾃⼰管理⾃⼰的原则。它坚决反对包办代替，不管是在⼟

地⾰命中，还是在接收城市以后，⽑泽东都始终强调党起的是组织群众，启发群众，教育群众

的作⽤，但绝不是包办代替，归根结底，群众的事情要群众⾃⼰去做。⽐如在解放军48年4⽉

初刚刚攻下洛阳的时候，怎么管理城市？⽑泽东提出的原则⾮常清楚：“城市已经属于⼈⺠，⼀

切应该以城市由⼈⺠⾃⼰负责管理的精神为出发点。”⸺《再攻洛阳后给洛阳前线指挥部的电

报》

所以领导和强制的实质有什么区别？区别在于，⼀个是群众的事情群众⾃⼰⼲，⼀个是群众的

事情我来命令他们⼲。

两者表⾯也有相似之处，但内在遵循的原则却是完全不同的。

第三，也是最为本质的区别。当“总体战”在强调个体利益服从总体利益，农⺠利益服从国家利

益，⼀切服从“总体”的时候，它到底在强调什么？实际上只是⼀点，他们是在要求农⺠的利

益，基层⼠兵的利益，学⽣的利益等等⼈⺠利益，服从⼤地主，⼤买办，⼤官僚阶级的利益。

正是因为如此，所以当它要求军队权⼒集中的时候，却不提出官兵平等，账⽬公开；当它要求

农⺠参与⼟地改⾰的时候，却不要求没收地主的⼟地；当它要求所有⼈艰苦奋⽃的时候，却⼜

不允许⼤家发表对它任何不利的⾔论。

⼈⺠战争也强调个⼈服从集体，强调不能搞⼩资产阶级的个⼈主义和⾃由主义。但是，这⾥⾯

的根本区别在于，⼈⺠战争中集体利益与个⼈利益没有根本⽭盾，集体的努⼒最终是集体获

益。这⾥⾯当然会有⼀些兼顾不全，照顾不周的地⽅，但是它绝不会有⼀个少数⼈形成的利益

群体作为⼀个团体或阶级在集体中存在。国⺠党的“总体战”就好像什么呢？就好像⽼板在要求员⼯艰苦奋⽃，振奋精神，脱下⻓衫，吃

苦耐劳，不计较个⼈名利的时候，这种牺牲和努⼒最终获益的是谁？是⽼板。⽼板与员⼯之

间，是作为两个利益不同的阶级群体⽽存在的，这个“总体”或“集体”⾥⾯，有⼀个少数⼈控制

和垄断着的利益管道，这个管道保障着群众和⼈⺠的牺牲、努⼒、奋⽃所获得的⼀切利益，都

只会流向那个少数⼈的阶级群体。这就是蒋介⽯集团总体战的实质。

所以“总体战”和“⼈⺠战争”的区别在哪⾥？就是以上三点。

⼀个是⺠主基础上的集中，另⼀个是毫⽆⺠主的集中；

⼀个是以⼈⺠⾃⼰管理⾃⼰为原则的领导，另⼀个是以包办代替为原则的强制。

⼀个是集体利益与个⼈利益的⼀致性，为集体努⼒归根结底也是为⾃⼰努⼒；另⼀个则是存在

着⼀个与⼈⺠利益根本不⼀致的利益阶级，为“集体”努⼒，根本上不过是为那个少数⼈的利益

阶级努⼒。

这就是为什么当蒋介⽯天天喊着振奋精神，重拾信⼼，坚定意志，⼀切从精神上努⼒的时候，

其结果却是军⼼溃散，⼠⽓低落，⽃志丧失的根本原因。

这也是为什么⽑主席说：“我们的战略战术是建⽴在⼈⺠战争这个基础上的，任何反⼈⺠的军队

都不能利⽤我们的战略战术。”
