# 为什么表达是枢纽

#### 人试图主动的去改造世界，以有涯应对无涯。目的-表达-**互动**。

从解决问题角度看，**表达清楚了，才是思考清楚了；然后再进一步，交流清楚了，事情才能推动了。**

# 表达的核心

#### 逻辑—批判性思维—表达 是保障质量和深度优势的利器。

- 围绕着目标，把点串成几个不同角度的链条，只抽取和目标相关的部分，而不是试图涵盖全部的信息；

表达的核心是使用某种语言，从无限的世界中按照目的抽取提炼信息，然后将世界重新组建。

因为需要“重新组建”，所以需要**记忆提取和生成力**。类似**[原理+应用：基本现象之一](./原理+应用：基本现象之一.md)，与世界“互动”，输入和输出结合。**

- **不是全面的还原世界，也不是将所有想到的描述出来**；
    - 与[天才：仙风道骨](./天才：仙风道骨.md)冲突。
    - 分清楚写给自己借助于纸笔思考和互动交流两种情况。
    - 而这两种情况也可以相互促进，**输出带动输入，但没有输入就没有输出。**
- **而是互动**，**围绕目的发现所谓的本质**，并通过非常**严密的组织结构表现**出来。[什么是逻辑链拆解、重建、再表达](./什么是逻辑链拆解、重建、再表达.md)，说的也是类似意思。

将一个物理问题使用数学语言表达出来，基本就解决了。
