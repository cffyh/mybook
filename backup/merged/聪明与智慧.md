# 聪明与智慧

作者：晓梦butterfly

链接：[https://www.zhihu.com/question/55333539/answer/2377081430](https://www.zhihu.com/question/55333539/answer/2377081430)

[天才：仙风道骨](./天才：仙风道骨.md)

[仙风道骨和实干家](./仙风道骨和实干家.md)

聪明的人很容易看出来，因为聪明的人说话办事都很精明，聪明的人在世俗中很擅长处理事务性的事情，例如什么事情一说就明白，学习上不费劲，工作上也很容易找准目标，与人交流沟通时也面面俱到，在生活中容易讨价还价，不容易被别人占便宜，很容易得到现实意义上的好处。

聪明的人很容易看到自己的利益，然后想方设法的去获取，处事喜欢占上风，这些在众人的眼中都是聪明人的表现，从外貌上来看，聪明人往往衣着得体，眼睛比一般的人有神采，头脑很灵活，言语上狡辩度高，更甚者属于老奸巨猾。聪明人动手能力很强，思维很活跃，处事目的性也很强，很容易学到生存的技术和技能。但是技术和技能都属于术的层面。

而相反真正有智慧的人是着重于道的方面，大智若愚，他们可能**不太擅长事务性的工作，在生活中也不能左右逢源，甚至看上去呆呆的，吃了亏也不争辩，也没有狡黠的目光，也不擅长于争取现实的利益，**眼睛是心灵的窗户，多少都会透露出信息，无论一个人是善良的或势利的，恐惧的等，都会从眼神暴露出来，当一个人的大脑和关注力集中在某些事情上面，眼神和面相都会因此而改变。

一个人的事情分前世今世来世，聪明的人过多的把注意力和精力放在了今世具体的事情上，自然无法获得大的智慧，**可能很容易解决实际问题，但是却无法获得大的成就，而拥有智慧的人会去思考很多形而上的问题，关注宇宙自然以及未来等更深刻的问题，而在面对现实生活中的问题时通常却显笨拙**。但因此他们缺少现实中外界的关注，默默的在自己的领域耕耘，最后能达到很高的成就。

[同归于尽的癫狂：究竟是遭到校方迫害，还是钻牛角尖的妒忌？](https://mp.weixin.qq.com/s/JPls24xLEry3ebkB1C0D0Q)

当一个人自以为很聪明，自以为看透了世事，也同样容易做出错误的判断，你所以为的事实未必就是真的事实，你以为的虚假也未必是真的虚假，而事情的真假又在不停的变换中，祸之，福之所依，福之，祸之所伏。今天肯定的事可能明天就会否定，明天否定的事情可能后天又会觉得是真相，不停的在虚幻的妄念中跋涉。

自以为聪明通透的人容易看透一切，丧失热情，丧失前进的动力，沉浸在自己编织的思想牢笼里，嘲笑别人看不穿，这样反而让自己陷落在局限中，沉浸在自己编织的梦境中，无法看到更广阔的世界，容易聪明反被聪明误，一不小心浪费了时间，做出错误的判断，让自己更加烦恼，无法进一步靠近智慧真理。
