# 结构化、体系化方法

结构化、体系化方法

- 围绕目标，结构化的分解或推演，形成体系，并且有重点
    - 有利于高效完整的进行思考、安排、表达；
    - 一种将复杂分解为简单的方法；
    - 抽象上升到具体的一种方法。

因为事物的本质可以用系统来表达，

所以结构化、体**系化方法，可以接近事物的本质，而且相对完整的**表达事物，

但我们掌握的是无限世界中的有限信息，所以体系得有主轴，得有方向。

在系统中，有主次之分，如何处理？

- 主次是聚焦和关注的关系；
- 虽然力量聚焦在主要部分，但要时刻关注次要部分对主要部分的影响，次要部分转化为关键点的变化。





结构，是关系，是秩序，是循环的单元。

复杂都可被分解，如果当前不能被分解，说明太混乱，得从不同的尺度去处理，找到其秩序。
