# 面试注意点

### 一、字节跳动面试流程

根据你所面的岗位和职级，有所差别；一般情况下有6轮，大致流程如下：

一面：部门同级

二面：直属Leader

三面：交叉面

四面：二级leader

五面：HRBP

六面：HR（谈薪）

### 二、字节跳动各轮面试内容分享

### **（1）一面**

**面试官**：部门同级

**面试重点：**初筛，判断匹配度

每段工作经历的重点内容、负责指标、与岗位相关的项目，离职原因，职业规划等；

**面试时长：**40-60分钟

**常见面试问题：**

1、自我介绍

2、介绍下你的工作经历

3、介绍一下某段工作，与目标岗位匹配的工作（会追问：核心指标，核心策略，具体方案）

4、某一个项目是怎么做的？（与目标岗位相关的）

5、为什么现在考虑换工作

6、有什么问题要问我的

**补充说明：**

一面面试难度不大，重点是判断与岗位的匹配度，因此只要将个人工作讲清楚即可。

但以下事项需要做好准备，尤其是第三点，有助于你做好后续面试策略。

1. 确认面试官角色
2. 可以加面试官联系方式
3. 确定岗位的三大信息：kpi、核心指标、目前的业务难点
4. 做好面试复盘

**（2）二面**

**面试官：直属leader**

**面试重点：**匹配度+初筛面发现的有顾虑的点

工作内容深挖，与岗位相关的项目、核心优势，风险点排查，比如与这个岗位不匹配的点、欠缺的能力等；

**面试时长：**40-60分钟；

**常见面试问题：**

1、自我介绍

2、介绍一下某段工作，与岗位匹配的工作（会追问：核心指标，核心策略，具体落地方案）

3、你的核心优势是什么

4、你的职业规划

5、我们这个业务与你之前有XX不同，你有什么想法，你会怎么做

6、有什么问题要问我的

**补充说明：**

二面面试难度一般，面试官提问会比一轮更深入，重点关注部分会问的更细；

二面前后建议做好以下准备：

1. 重点准备一面面试官对你感兴趣的部分
2. 重点准备一面面试官对你质疑的部分
3. 做好面试复盘
4. 可以加面试官微信，如果气场和，可有助于你后续面试

**（3）三面、交叉面，**

**面试官：兄弟部门leader，与直属leader平级**

**面试重点：**

（1）工作经验了解：过往工作经验、代表项目、核心能力

（2）横向能力检验：与面试官协同部分需要具备能力的验证

（3）部门的协同：面试过程中的业务协同性、业务视角等

**面试时长：**30-40分钟左右

**面试案例：**

1、自我介绍

2、介绍一下你的工作经历

3、你的核心优势是什么

4、分享一个你成功的项目（或者就简历中他感兴趣的项目）

5、你过往如何与XX部门协同

**补充说明：**

这一轮，面试难度较大，也取决于你所遇到的面试官，有的面试官会问的很细，有些会问一些简历中他感兴趣的部分。做产品运营的同学特别要注意，极大概率遇到产品面，要注意做好面试准备。有不少运营的同学遇到产品面，气场不对，挂了。

这一轮面试前，做好以下准备：

1. 面试前提前确认面试官角色
2. 了解交叉面面试官所在部门主要业务
3. 了解面试官所在部门业务与目标岗位部门业务直接的业务流程和协同关系

**（4）四面，大部门负责人面**

**面试官：大部门负责人/总结**

**面试重点：**

（1）重点内容深挖：重点项目、重点能力、

（2）格局、深度：看业务的高度、专业能力的深度

（3）目标岗位理解及思考：对目标岗位的理解，以及面临的问题，是否有初步方案

（5）能力顾虑点排查：前几轮面试反馈的问题及顾虑点再次排查

**面试时长：**30-40分钟

**常见面试问题：**

1、 为什么选这个岗位

2、 介绍下每段业务做了啥

3、 抛了一个问题，举个案例

4、 从行业、专业垂直领域提一个大的问题（这个提问会从你与岗位不匹配的角度提）

**补充说明：**

这一轮面试难度较大，而且很多同学会受到面试官气场的影响，紧张发挥不好。大部门负责人会从更高的一些角度提问，而且经常会有一些出其不意的问题，来考察候选人的应变、思考框架、视角等。比如抛出一个假设、一个问题，来问你的思考和解决方案。

**（5）五面：HRBP面**

**面试重点：**

（1）专业度了解：工作、项目的了解

（2）职业规划及求职动机：离职原因、职业规划

（3）价值观判断：个人价值观、软性能力

（4）隐藏基础信息挖掘：个人、工作，薪资等基础信息

**面试时长：**30-40分钟

**常见面试问题：**

1、为什么考虑换工作；

2、职业规划是什么；

3、对下一份工作的期望

4、什么渠道了解到了岗位信息，目前对公司和业务有哪些了解

5、你的优势/劣势是什么

6、你的优点/缺点是什么

7、过往工作中对接流程、沟通能力、项目，工作中的困难点

**补充说明：**

这一轮，还是需要好好准备，字节的HRBP面，会有很大部分是业务问题，有些HR依然会问的很细、很深入。当然也有一些比较亲民，面试氛围像聊天一样，但是始终要记得，你是在面试，不能被表现麻痹，务必注意：

1、 做好前几轮复盘，前几轮被重点提问的点，再次做好梳理；

2、 切忌”得意忘形”，有些HR比较“能聊”，切忌话匣子打太开；

**（6）六面：HR谈薪面**

进入到这一轮，那么恭喜你，意味着你已经通过前面的面试啦！

这一轮，很简单，不涉及业务。

重点是聊薪资，面试，入职时间、OFFER、背调等相关的事情啦！
