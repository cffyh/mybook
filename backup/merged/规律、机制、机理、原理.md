# 规律、机制、机理、原理

四个词核心都指向 “事物运行的内在逻辑”，但侧重从 “宏观应用” 到 “微观本质” 逐层细化，区别如下：
**规律（Law/Rule）**

- 最宏观、普适的 “客观必然性”，是事物重复出现的稳定联系，不依赖具体场景。
- 示例：自然规律（四季更替）、市场规律（供需影响价格）。

**机制（Mechanism）**

- 侧重 “系统 / 组织层面的运作框架”，是为实现特定目标设计的**流程、规则或结构（偏应用）。**
- 示例：公司 “人才激励机制”（靠规则 / 流程推动目标）、人体 “体温调节机制”（器官协同的运作体系）。

**机理（Mechanism/Principle）**

- 侧重 “微观 / 科学层面的因果逻辑”，是事物运行的**具体原理、细节过程**（偏理论）。
- 示例：药物 “降压机理”（分子如何作用于血管）、手机 “充电慢机理”（电池化学反应的异常细节）。

**原理（Principle）**

- 介于 “规律” 和 “机理” 之间，指 “事物运行的核心依据 / 基本道理”，可宏观可微观，但更强调 “基础性”。
- 示例：宏观（杠杆原理 —— 力学的基础道理）、微观（疫苗 “免疫原理”—— 激发抗体的核心逻辑）。
