# 学科竞赛

链接：[https://www.zhihu.com/question/348296109/answer/1914288599](https://www.zhihu.com/question/348296109/answer/1914288599)

鲁省物理复赛前十名兼美国前十物理系“数学物理”方向博士同学回答一下。

数学竞赛难度更大一些，但是很难说数学物理哪个更测试智力。

数学竞赛的解题过程，很像是二战期间德国军队被苏联红军包围之后的突围策略，就是派出小股部队试探性地攻击苏军的包围圈的各个位置，寻找薄弱环节，然后集中主力全力突破。

所以，数学竞赛的解题需要主动性地寻找问题的突破口，其本质，是考察联想和抽象思维能力。解题需要大脑高度模式识别技能，把未知题目与已知解答建立相似性，所谓“旁征博引”，数学家和文学家很相似。

物理竞赛的解题过程，很像是诊断一台出了故障的机器，利用已有的现象，做各种逻辑推理和分析，寻找背后的根源。

寻找物理本质的过程，需要一些思想实验或者极端推理（例如伽利略为证明牛顿第一定律而做的思想实验），其本质是考察洞察力，和建模能力。

物理本质就是能看破现象，看穿本质，用公式来代替现象，需要强大的数学抽象能力。“看破红尘”，物理学家有点跟佛学家有点相似。我美国的室友，北大物院过来的，房间里好多佛经。

        对于一般学渣而言，物理更难一些，因为数学的大多数概念本身并不抽象，甚至非常具体，只要上完课就可以做出题，做不出来也是因为技巧不够。而很多物理概念本身，如果没有微积分基础，是很难理解的(例如向心加速度)，上完课很可能两眼一抹黑。

 从学习方法上，**高中数学甚至竞赛数学的学习需要记忆大量的定理，公式，速记式，甚至中间结果。**但是，学渣们，甚至包括很多学霸女生们在初中一直靠记忆来学物理。

到了高中，却很惊奇地发现物理是高中唯一的一门几乎“不需要任何记忆”的学科。学生必须真正理解其中的物理实质。例如说，记下 F = ma是没用的，解题的时候还是不会做。更要命是，这个只有三个字母和一个等号的公式，很多场景下居然还是错的！

2018年物理竞赛决赛中有一道题目，就是让学生证明在非相对论条件下，物体没有受力的方向上，居然还有加速度！这么毁三观的题目，让你发现记忆对于物理完全无效。因此你必须“真正”理解物理，而不是背下任何公式。 就算是大学物理的麦克斯韦方程组4个方程，也不需要多少记忆，因为它可以用2个简单方程概括之，那就是 d F =0和 d⋆F=⋆Jd，记忆量不超过牛顿第二定律。而磁场无源公式∇⋅B=0多少年一直被人怀疑是错的，可以说没有任何物理公式可以让你“依靠“，没有任何物理定律无条件成立。因此物理是高中唯一一门不允许“糊弄”的学科。

记忆对于物理来说，也不能说毫无意义。有时候记住一些东西可以少一些推导过程。中国最好的大学物理研究生面试的时候，曾经问过极坐标下的加速度方程。提问者的本意是希望面试者从 出发推导出来，但是我直接默写出结果 。这样当然可以给老师留下印象，但是从物理学角度，从基本原理出发，推导出一系列的结论，才是学习物理学的正确姿势。记忆力是个学习优势，但是单纯依赖记忆对物理学却是有害的。

对于竞赛而言，数学更难一些。数学竞赛题的难度是没有上限的，只要命题者愿意，可以出任何难度的题目，答题者可以给出任何不同于标准答案的解答，只要符合数学逻辑。但是物理竞赛题的难度存在人类物理学认知的上限，且理论题目必须符合物理实验结果，命题者只能从人类目前已有的物理知识中选择相关题目，答题者的解答必须有已有物理定律做为依托，自由发挥的空间不大。换句话说，最难的数学题的新颖解答可以写一篇论文发表，而最难的物理题再新颖的解答也不可能超过已经发表的论文。有些imo题目甚至大学专业数学教授也难以作答。而物理竞赛几乎难不倒“顶级”(还是需要智商的)大学物理专业高年级大学生。任何物理难题，都可以在某一本高等教材或者论文中找到命题依据和解答思路。数学就未必了。

说得更深刻一点，数学是一门理论科学，必须“制造问题，解决问题，总结解法，归纳理论”，才能进步。

历史上，数学大部分分支就是从解题中取得进步的，人类为了解决一元高次方程的求根问题，发现了对称，提出了群论和群表示论。人类为了解决曲线面积和曲面体积问题，提出了积分理论。人类在解决各种数学猜想的过程中，推动了数学的进步。

**解题，对数学这门学科具有极其重要的意义。而**物理学，则是从实验观测中与已知理论发生矛盾而取得进步的。更干脆地说，物理就是一门实验科学，理论的进展必须有实验为依托。最典型的就是黑体辐射这朵乌云，就是从发现实验结果与韦恩公式和斯特藩定律矛盾开始提出了量子辐射理论。

本质上说，**数学竞赛的解题，有可能是对未知方法的探索。**

而物理竞赛的所有解题，**必定是对已知理论的应用。**

从备考难度来说，数学的准备量远大于物理。高中数学竞赛的考试内容，如平几，不等式，其实很多都是初中数学级别的内容。甚至费马小定理也都是小学学过的互质。但即使是这样的范围，也足以出来高难度题目。也正因为如此，数学竞赛缺少章法，备考者需要做**极其浩瀚的题目，然后从中自己领悟规律和思维范式**。准备者应当从初中就开始。 

物理竞赛则是有明确的章法，把大学微积分学完，把大学物理四小力学系统性学完，基本上就只剩下一些解题技巧需要学习了。

所谓的物理难题，在高视角下很可能是简单题。2016年之前，物理竞赛盛行微元法，微积分一旦开闸，很多题目也没有考察难度了。

例如这道物理竞赛名题：蚂蚁速度与距离成反比，求爬行时间。 一旦用了微积分，这道题都可以直接心算答案。16届物理复赛最后一题，是一个求轨迹的题目。当时那个年代，微积分还没有开闸，因此标准解答非常有技巧性。在高视角下，该轨迹的微分方程是 。。。读者可以作积分，分析是什么轨迹。

从获奖难度看，数学竞赛相对于物理竞赛，群众参与度更高，竞争更激烈，获奖更困难。

物理竞赛相对于数学竞赛的备考，需要更好的家庭和学校经济条件，更烧钱。数学竞赛更适合自学和全民普及。

1，数学竞赛主要的学习手段是**做题和归纳总结**，而这与高考数学的学习手段类似。

 2，物理竞赛的主要学习手段是自学大学数学和物理，这需要很强的兴趣来支持，和超人的自学领悟能力。

 3，物理竞赛考察动手能力，数学竞赛不需要。

从贴近大学要求看，物理竞赛的好处在于竞赛学的东西(高数，大物)是大学理工科大部分专业的必修课内容，提前学习不会浪费。然而，中学生数学竞赛中的很多内容，仅仅对于思维锻炼有好处，对于大学课程来说，可能直接帮助较少。因此物理竞赛生更容易适应大学课程。有个别国际数学竞赛金牌，大学期间普通物理考试挂科，而没有毕业。但是从来没听说过哪个物理竞赛选手在大学微积分考试中挂科。我们中学班里的一个女生高考数学满分，但是平时学习，一旦见到多个小球碰撞的题目就眩晕脸红。

换句话说，数学好的物理不一定好，但是**物理好的数学不会差。**

以本人为例，本人获得中学生物理竞赛省一，然后大学生数学竞赛决赛一等奖。因此说，从知识储备来看，物理竞赛生强大的数理基础最贴近大学招生要求。从解决问题的能力来看，数学竞赛的挑战未知问题的精神最能预测大学毕业生未来的发展。因此，大学招生办最喜欢数学和物理竞赛生，是有原因的。

从对个人科研能力预测性来看，数学竞赛比物理竞赛好很多。数学竞赛的思维方式，是把一个未知问题转化成已知问题，这个跟数学科研很相似。

物理竞赛的题，从物理学角度看，逼格不高，跟《物理学在工程技术中的应用》这本书的例子非常相似。因此物理竞赛的表现，更能预测学生在工科大学的电子系，或者土木系的表现。 20年前的物理竞赛非常像智力测试，出来不少脑筋急转弯的题(以静电势方面的题居多)。最近几年，物理竞赛也出现了几道纯粹的物理题，例如上面提到的2018年决赛的第七题，就很物理化。但是总体上，物理竞赛更像是给工科大学，而不是理科大学选拔优秀学生。

说句题外话，高考成绩更能预测学生在大学的写作和英语水平。

四科竞赛题目难度比值，数学，物理，化学，生物，估测 10:8:5:1

四科竞赛备战学习量比值，数学，物理，化学，生物，估测 5:3:3:2

数理两科竞赛的相同之处就是都需要长时间积累，而积累需要强烈的兴趣和一定的天赋，光靠上大学的功利性想法，把竞赛当成高考，恐怕很难熬下去，取得成绩。准备竞赛消耗的脑力，恐怕足够考高考状元了。物理竞赛需要看的书，除了四小力学，还有大学微积分，外加物理实验（而高考是不考实验的）。这些数学物理书书加起来，已经远超过中学理科综合的学习量了。

\-------关于OI ------

我高中的时候曾经参加过一阵子OI培训。

目击OI难度应该介于物理与化学之间。OI竞赛对于数学的考察比较尴尬，有时候会出现用力过猛反而喧宾夺主的情况。比如说我听说某个公司考察算法，出了一道题，写程序求 ∑i=110001n(n+1)\sum_{i=1}^{1000}\frac{1}{n(n+1)}, 本意是想考个递归算法。结果用力过猛，有人直接写 return 1−11000+1 1 - \frac{1}{1000+1} 。结果什么算法都没考到。

因为涉及到算法设计，而算法设计更测试智力，而非记忆力。当然也有套路可循，例如动态规划，二叉树算法, memorization， 图遍历，dijkstra 路径算法等。

五科难度比值 10:8:5:1:6

OI的主要问题不在于试题难度，而在于OI竞赛对教练依赖度高，而聘请一个教练相当于从阿里巴巴和腾讯挖人，聘请成本远高于普通编制的老师。因此竞赛参与者明显局限在发达地区，大部分贫困家庭和落后学校缺乏计算设备和网络资源，根本没有尝试的机会。因此参与者少，竞争比较小。有些发达地区学校将其当作升学捷径。

\------关于生物------

生物竞赛更像是大学生物专业的预科考试，甚至是研究生入学考试。生物竞赛对记忆力要求极高，对知识面有要求，也需要对学科有浓厚兴趣。

“强度”这事看的是多少人竞争吧。这事几门学科竞赛都差不多。但是不同竞赛的方向还是不一样的。

高级别的数学竞赛训练大体上还是刷题，只是题目更难、技巧更精妙，可以说是“解题训练”的巅峰。

这是因为数学竞赛限制了[高观点数学](https://www.zhihu.com/search?q=%E9%AB%98%E8%A7%82%E7%82%B9%E6%95%B0%E5%AD%A6&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1644276041%7D)的运用，而集中考验选手短时间内运用初等数学技巧的能力。[物理竞赛](https://www.zhihu.com/search?q=%E7%89%A9%E7%90%86%E7%AB%9E%E8%B5%9B&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1644276041%7D)的技巧难度则是有上限的，到集训队级别，大致是一半时间学大学课程，一半时间刷题。很多初看之下千头万绪的经典难题，在高观点下就会变得清晰、甚至简单很多。

做个不见得恰当的类比：数学竞赛类似于短跑，规则非常简单，但比较考验天赋和头脑的爆发力；而物理竞赛类似于足球，顶级选手里高矮胖瘦都有，有人就是靠天下第一的技巧所向披靡，也有人技巧一般、全靠大局观和洞察力而战无不胜，很难说哪一项能力就一定是最重要的。相比之下，**物理竞赛训练更接近大学、甚至科研阶段的学习模式和节奏。所以少有物理学家骂ipho的，但数学家不喜欢[imo](https://www.zhihu.com/search?q=imo&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1644276041%7D)的还是有不少。**

最后多说一点。国内的中学教育非常强调解题，这有国情和历史缘由在，但也造成了一个非常不好的影响，那就是导致了许多人对技巧的盲目崇拜。这个问题下的许多答主，倾向于把”智力“和”解题能力“视为等同，大体也是这种崇拜的体现。这里我想引用一段菲尔兹奖得主，拓扑学大师[威廉·瑟斯顿](https://www.zhihu.com/search?q=%E5%A8%81%E5%BB%89%C2%B7%E7%91%9F%E6%96%AF%E9%A1%BF&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1644276041%7D)（William Thurston）的话：

跟（智力）早熟相关的另一件事，是把数学看成一项比赛、甚至体育竞技项目的流行趋势。在各色（地区级）高中数学联赛里，数个高中组织队伍参赛，定期聚在一起，在一小时左右的时间里比拼解题水平。

在州级、国家级、乃至国际级别上也有类似的竞赛。对擅长竞赛的选手来说，竞赛体验可谓是寓教于乐，但这些竞赛也不乏缺点。它们强化了一种观念：一个人的”数学基因“是先天决定的。这些竞赛鼓励牺牲思考深度和广度以换取速度，强调需要找到隐蔽技巧才能解答的谜题，**忽视了重视系统性和可持续解决方案的真实问题。这**就让许多思考速度**没那么快、技巧没那么娴熟、但可能更擅长慢慢解决问题的人对数学望而**却步。一些顶尖竞赛选手确实成为了优秀的数学家，然而许多顶尖数学家却并不擅长竞赛数学。

迅速的思维对数学自然有益，但也无非是有益的素质之一。对不想成为数学家的人来说，竞赛数学的那些技巧恐怕更没用处。这些比赛有点像拼写大赛。优秀的拼写跟优秀的写作能力有一些联系，然而地区拼写大赛的冠军，并不见得就有成为优秀作家的才华，反之，一些好作家的拼写却算不上很好。如果大众把拼写和写作混淆在一起，许多潜在的好作家就要对写作望而却步了。

总之，高**强度、快速的技巧性思考确实是智力的一部分，但远不是智力的全部内容。实际上作为一个科研工作者，我**所见到的真正的[数理学科](https://www.zhihu.com/search?q=%E6%95%B0%E7%90%86%E5%AD%A6%E7%A7%91&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1644276041%7D)的大师（领域领袖、甚至诺奖级别），有思维非常敏捷严谨的，也有反应稍慢、甚至经常犯错的，而他们的共性，是知识非常广博、思维极其深刻。

数学和物理竞赛考察的人脑能力的方面不太一样。

数学竞赛考察的是观察和联想能力，难点在于对抽象黑箱的内在结构的洞察，往往是和已知白箱的类比，而且往往**只有些许的相似，通过组合所有的相似处，寻找出一个解决路径。**

物理竞赛的题目一般逻辑推理难度不大，考察是是数学建模能力，要求的是比较好的大脑的并发计算能力——你需要**同时载入不同情景才能对一个复杂抽象物理问题**选择恰当切入点完成建模，**这点和OI是一样的，因此信息竞赛和物理竞赛更相**似。

如果大脑并发计算能力很强，那么搞物理竞赛优势很大且上升速度很快。但是，无论你洞察力有多么惊人，搞数学竞赛都需要漫长的数学成熟度积累期。

**一般来说，我们容易把数学竞赛考的那个洞察力理解为智力，在这个考察层面上，数学竞赛是物理竞赛几十倍**。如果说是综合难度，那么见仁见智，数学应该略大于物理。

总结：

物理竞赛的相对卡点在于工作记忆容量，这个更多地程度上依赖于先天。

数学竞赛的相对卡点在于模式识别算法，需要通过长时间的解题中的思考来形成自己的数学特征识别模式，因此是一种隐性的经验积累，数学竞赛是一种**隐性的文科竞赛。**

绝对难度上，数学竞赛要比物理竞赛大得多，准备的实际工程量浩瀚得多。
