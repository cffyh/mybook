# 词语的替代

如果不用本质这个词语，

可以使用底层逻辑、第一性原理

还能够：

### 一、日常口语（聊天、生活最顺口）

1. 根子
2. 关键
3. 要害
4. 本心
5. 本来面目
6. 根本问题
7. 矛盾点
8. 不冲突
9. 核心诉求

### 二、理性思考（偏认知、做事，更落地）

1. 核心
2. 内核
3. 根本
4. 本源
5. 实质

### 三、书面 / 正式表达（写东西、讲观点）

1. **真实需求**
2. 根本要义
3. 核心内涵
4. 根本所在
5. 真实内核

### 四、解决问题 / 抓重点（找原因、做决策）

1. 根因
2. 核心逻辑
3. 底层规律
4. 根本原则
