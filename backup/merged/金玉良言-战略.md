# 如何制定战略？

- 战略思考本质上是最困难的，但不易检验，容易陷入夹生不熟的境地。
    - 战略的第一基础是以改变现实世界为目标，实际情况是中轴线。
        - 战略必须建立在真实情况的基础上，求真务实是必要条件。
        - 哪些是真，哪些是实，论点和结论是否正确(逻辑合理)，⇒ 批判性的思考，批判之后发现世界不是非黑即白的，⇒ 辩证性的思考。
- 本质是取舍，就是选择和坚持。有所不为才能有所为。很难：
    - 时间上：以终为始；
    - 空间上：整体利益大于局部利益；
        - 24/坚持并不难,改变也不难;难的是知道什么该坚持,什么该改变,何时该坚持,何时该改变。轻重缓急

# 战略的不同阶段

- 战略尝试期必须躬身入局，才有手感，能快速判断。
- 成长期：故事清楚，招到更多的人。关键的战略洞察，变成一小部分人的共识。
- 成熟期：效率，把一个战略分解成可以运营的商业模式。

# 战略需要的能力

- 战略很大程度上是一个系统性思考，是一个模型的建构过程。
    - 这中间需要抽象思考能力，不是逻辑思考能力。从表象抽象出背后的结构。
    - 没有逻辑，就缺乏正确性保证，一切依赖逻辑。而通常的逻辑，过于精确，就失去了直观。这在数学里面可以找到例子，没有逻辑就没有数学。数学的创造依赖直观洞察。

**just do it和战略性的思考并不冲突。**

- 区分多数简单情况和少数需要战略性思考的情况。区分不清楚，就会既没有全局高度，又没有简单踏实。

# 选择与坚持

坚持  =  定力和信念 = 心力(世界观 + 真实的体验)

- 先天的遗传 + 后天的际遇，决定了该坚持什么，也就是扬长避短。
- 坚持 => 积累 => 力量。

选择 = 智慧（洞察 + 取舍）和 机遇

- 有了坚持，才有取舍，才有了选择的依据。选择什么是由客观情况 + 主观的坚持决定。
- 选择依赖于sense，关键选择决定了大势，这就是选择最重要的原因。

[四渡赤水的高明之处在哪里？](https://www.zhihu.com/question/328298066/answer/3146771898)

四渡赤水不是单纯军事上的战争，还是政治层面的战争和信息战，尽管任何战争都有政治考量，都要千方百计获取敌方情报信息，但这里不一样。

据说西点军校曾通过计算机程序模拟四渡赤水，10次都无一例外的失败了。毕竟是3万被40万围剿。

**博弈中，通常任何一个策略都有对手策略**，除了随机策略。

教员在四渡赤水中采用的正是随机策略，于是在云贵川三省间纵横捭阖，让十倍于己的围追堵截之兵疲于奔跑，不断赢得主动权，三番跳出包围圈。

有以下几点：

① 深刻的政治洞察：国民党中央与地方的貌合神离

这是运用一切策略的大前提。教员不仅对蒋的驱虎吞狼之计胸中了然，并且对这一点有着深刻洞察，并将其归结为中国革命发展和取得胜利的根本前提和依据。教员在《中国的红色政权为什么能够存在？》以及《中国革命战争的战略问题》中都详细阐述了这一点。这就是战略前瞻力和战略洞察力。

② 前瞻的战略规划

当时的中央红军有3个可能的策略目标：

a. 北上与红四方面会合

b. 东去与红二六军团会合

c. 黔北以遵义为中心开辟根据地.

三策确定了总体框架。a策略为优先策略，c策略在后来的会议中被否决了，因为贵州太穷了，所谓“人无三分银，地无三尺平，天无三日晴”。但不妨碍以此来迷惑对方，声东击西。而**为了实现a策略，教员在遵义时就考虑如何调动云南军阀龙云。**

③ **坚定的核心目标：战争主动权**

其他所有具体目标都置于这个核心目标之下，作为实现核心目标的可能手段。不在乎一城的得失，和一役的成败。与**坚定的目标结合的是灵活的策略**，而不是固执。土城失利，迅速撤出战斗，一渡赤水。一渡之后，扎西再度被围，果断转移，二渡赤水。二渡后，反对打打鼓新场的王家烈，因为路程远，变数多，而决策鲁班场打周浑元，不求胜，只求势，为三渡。

④ 敏锐的信息判断

兵不厌诈，你来我往的情报战在战争中很平常。但**抓住一个信息来调整整个部署，自上而下重新谋划一场战役实则罕见。**教员可是曾经专门派部队搜集当地报纸等信息来源。足见其对信息的重视。上面的随机策略的随机可不是任性随意，而是持**续纳入最新信息的寻优策略，是贝叶斯决策模式**。这里的信息战在于对信息的敏感性，并将其作为战略的输入。整个战役转折点就在**抓住蒋来贵阳亲自坐镇指挥**这个信息。攻敌于必救，从此扭转战局。

回头看。体现了《孙子兵法》中的

① 致人而不致于人 （多方以误之）

在调动敌军的过程中来调动自己的兵力。牢牢把握战争的主动性，形式上被动时，就在内容上把握主动性。

对双方的力量有着清醒的认识，不冒险，不拼命。把战略和战役战术分的非常清楚。有如康德将理性和知性的严格划分。教员对战略和战役战术上的对立统一的认识是天花板。战略上藐视敌人，战术上重视敌人。战略上以少胜多，战术上以多胜少。战略上以一当十，战术上以十当一。这正是以弱胜强的底层逻辑。
