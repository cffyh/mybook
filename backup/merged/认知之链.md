# 认知之链

- [规律 vs 现象，本质和表象。](./一些基础哲学概念.md#规律-vs-现象本质和表象)
- 问题是矛盾的表现形式；
- 矛盾（作用力）驱动变化，规律是变化过程中重复出现的模式；

---

从现象上升到理论

- 高效高质量的理清复杂的现实：
    - 逻辑和批判性思维 -> 结构化思维；
    - 分解、分类、链条。
- 提炼现象，匹配理论（相似相异是核心）；
- 透过现象看本质：现象 -> （表象 —> 问题 -> 矛盾 -> 规律） -> 本质 。

抽象和具体：最上层的具体是实际(客观现象)，最底层的抽象是概念

- 在最底层和最上层间有n个中间层；
    - 中间结论，中间层重要性，让链条串起来。
- 概念是从实际（客观现象）中提炼出来的，是抽象上升到具体的起点。
    - 另外，概念通常是作为已知，不需要新提炼。因为我们总是在一定的历史和已知上解决问题，不是从空白开始。
