# 二阶控制论

控制论从提出之初就不是一个可被化约的同质性理论，而是一个有着多种话语和声部相互竞争的场域。

但是，若撇开40年代以来这些复杂交织的文化史和思想史脉络，仅从结构的层面来梳理的话，大致可分为两支：

其一以维纳为代表，他所做的传送器设计工作，考虑的依然是如何将物质性实体进行远程传输的问题。

其二是从香农的信息论。

一直到贝特森等人试图将机械领域的控制论理念引入心理学和人类学，在自然世界-文化世界之间架桥，控制论由此从物质性转向了对非物质性的强调。

以上，大致是一阶控制论的两种形态。也可对应于旧唯物主义和新唯物主义之间的区分。

前者与传统人文主义形成相反相成的关系，二者都坚持在物质与精神、实体同虚拟之间进行二元对立与划分。

而后者，即新唯物主义，以海尔斯哈拉维为代表，则主张上述二者之间边界彻底的消融与抹除，但这也会导致重新倒退回一元论的危险（德勒兹主义）。

卢曼同冯·福斯特等人提出的二阶控制论或又称新控制论或可对这一僵局提供理论上的解决方案。

他们一方面反对工程学版本的控制论的“封闭系统”观，同时也反对一厢情愿的人文主义者所梦想的无限的“开放性系统”。

简言之，各种系统，无论是物理的、生物的、文化的还是技术的，它们都是半自治的，也即“具有一定边界的开放性”，他们的自组织的封闭性恰恰是对环境开放的前提条件，同样的，在外界环境中汲取异质性信息也是维持其内部进行自我超越运动的前提。（对这一问题，西蒙栋在其博士论文中也说得较清楚，他表述为“内群体/外群体的相对性”、“分化和整合的辩证关系”，“非连续的连续性”等命题）。此种扭结的悖论关系并非德勒兹主义式的“流动边界”可以简单概括的。

 二阶控制论本质是，一种在**具体操作过程中的自我参照和自我指涉**，用系统论的说法，主体只能是在具体环境中被环境所构成的主体。

但这并不意味着，向环境无限敞开和融入，而是需要我们越来越多地考虑环境的强大影响。

环境的日趋复杂化，对系统提出了更高的要求，**系统必须持续不断地在与其他系统和结构耦合过程当中“回过头”来自我观察**，据此引入认知操作的功能，**在边界上与其他系统进行有选择性的谈判，才能维持内系统的稳定性**。（参见西蒙栋“个体化”概念）

或许有人要**争辩说，哲学早就给出过更简化版本的说**法：主体是在异化运动的过程当中形成自己的，但这显然**不如控制论那样更具有时代相关性和现实相关性**。

另外，**“系统”也显然要比“主体”的适用范围广泛得多**，并且更恰切。且这种说法也无助于解释在**复杂技术环境下能动性与非自主性的全新关系。**



福斯特认为「**控制论本身就是一种认识论，**甚至说任何完整的认识论都可以看做是某种形式的控制论」。因为**认识论并非关于知识本身，而是关于如何获取知识**。这也能在一定程度上解释控制论的模型框架为何能够应用于从自然科学到人文科学等诸多学科。

# 维纳的控制论

物理系统和统计物理对于群有深入的研究，该学科将我们所说的一致性看作是一种测量，众多具有属性可度量性质的粒子组合而成的整体可以表现出的某种测度如何在变化中保持不变性是该方向的一个研究重点。

控制论向我们深入阐述了信息、反馈、震荡、博弈、谐波与同步等在物体的相互作用中，无论是机械系统、电子系统还是神经系统都扮演的核心角色。这些研究和相应的学术和工程手段，尤其是非线性系统方面的黑盒处理手段都对复杂可控制系统的实现提供重要的工具，但这些工具没能直接给出智能的实现方式，而是更多应用在形式化之后的系统之中，我们的目标便是要在控制论大厦之上，厘清黑盒控制的重要理论基础和实验方向，以实现符号推理学习系统对大量黑箱系统的自动控制能力。

# 黑箱理论

我们更加关心人类的黑箱控制理论，系统2对系统1的控制就是意识机制对大量黑箱的控制过程。因而我们的方向上，更加偏重于如何理解神经网络操纵神经网络，而不是人类如何用形式化系统操作神经网络。

我们可以从ashby box去了解相关的理论，即W. Ross Ashby 的基本非平凡机器。

# 循环因果与反馈

控制论最重要的思想是循环因果与反馈，智能体从外界获得信息，并对外界发出作用，作用的结果又变成新的信息，如此循环久久。这种视角在前馈和机械论的世界还是陌生的，你会发现即使在今天的神经网络中也是陌生的，现代深度神经网络的反馈机制仍然很初级，做的稍好的是RNN到GPT-3等时序相关模型。

# 二阶控制论

二阶控制论又称控制论的控制论（Cybernetics of Cybernetics）。鉴于我们所提出的世界未必是客观的，而心智本身尤其是符号化能力及其局限决定的科学能力受限问题，让我们认为二阶控制论在方向上存在正确性。

武建峰说“一阶控制论向二阶控制论的转变蕴含了认知科学从计算主义到生成主义的范式转向”[28]，阿什比认为“这种‘约束’是观察者与事物之间的某种关系，任何一种特殊约束的属性将依赖于真实事物和观察者。”我们后面要讨论的“单一力是认识论问题”的论断是相呼应的。

为了回应阿什比的挑战，福尔斯特建立了二阶控制论采用了生成主义的观点。这为后来的激进建构论的认知心理学和认知生物学奠定了坚实的基础。

“从根本上反对表征主义的认识论，某种程度上延续了休谟和康德的哲学思想。它在本体论上采取了一种与唯物论相对立的立场，即认为被观察的世界是由观察者建构起来的而不是相反”[28]。

二阶控制论的核心要点与我们所设想的神经网络控制神经网络是同一的内涵。当然，如果二阶控制论具有实践性，则必须要解决我们AGI所面临的种种问题，尤其是符号学习与结构化能力构建问题。
