# 68联想

曹烽开始研究棋局，想通过围棋定式与定式之间的搭配，找到解决开放性和复杂性的方案。曹烽慢慢地感觉到，**模块与模块之间联系并不是逻辑结构，更不能靠直觉经验。曹烽找不到合适的词，只好称之为联想。**

复杂系统是这样构成的：刚性的逻辑结构和直觉交错，就像电器和人一样，再复杂的电器都有逻辑有电路图，而稚嫩的儿童，都拥有机器不能相比的能力。人与电器的互动，也是一种模块，可以快速地对环境刺激产生应变，比如开关收音机、调台等。但刚性模块与直觉模块之间的联系，会构成很复杂系统，就是一个开放的世界。比如，你不能确定这个人拿着电器要走到哪里去，去和另外哪一个电器或哪一个人交流。

这种不确定的交流，就是曹烽称之为联想。

围棋就是一个复杂系统。定式是模块，**定式之间的联系不能单靠逻辑，也不是单靠直觉，可能一会是逻辑的，一会是直觉的，或者说不清的，就与联想有关。**

曹烽陷入一种痴迷的状态，不仅对七连的活动不感兴趣，甚至忘记《连报》的出版，好在期末考试来临，《连报》停了也很正常。但曹烽天天拉着刘天说这些让人初听似乎明白、再想彻底晕菜的话题，很烦。

刘天小心地问：“这些东西有什么用吗？”

曹烽翻白眼，那意思就是说，想不到刘天这么庸俗。

刘天提醒，期末考试，七连有任务的。曹烽这才解释，搞明白推理与直觉、模块和联想的关系，学习方法论的问题就解决了，还怕什么期末考试。

“你的人生问题就全部解决了！”曹烽一字一句地说。

“我现在不想讨论人生问题，只想期末考好一点。”刘天不受诱惑，他愿意死盯曹烽找感觉，却不愿意曹烽反过来死盯他。

“做数学题，需要通过**推理和直觉反复应用，达到模块化运作，能快速解决很多问题，成为学霸。而物理不行，物理只能一个模块**一模块地学习，力学不能解决电磁学的问题，电磁学不能用于原子核，但这些**模块之间，有内在联系。如果就靠联想，发现了，打通了，就形成了更大的模块，光电统一就**是例子，这其中就会产生发明和创造，就是科研方向。这不是人生大问题？”曹烽还在喋喋不休，刘天实在受不了，转身走开了。

曹烽追上来，刘天反问一个现实问题：“你能告诉我期末数学和物理的应对策略吗？”

曹烽想了想，好像机器人突然之间被问到人类情感一样，满脸的迷乱，短路的眼神导致神经系统发热，肌肉收缩变形，躯体弯了下来。

刘天看到了异样，忙去扶他，曹烽终于喃喃地说了一句：“我的头好乱，什么都干不进去。我还是去想想12个乒乓球吧。”

这个时候让曹烽背圆周率100位，估计能够挣到钱。

曹烽独自一人离开人群，向后山走去。后山后面，林中空地，有一块大石头，就像曹烽的家一样，他去那里休息休息也好。刘天想：就让他休息吧，我不能休息，我要奋斗，要考出更好的成绩来。

这后半期的感觉，刘天十分痛快，他就像找到什么宝，莫名其妙地感觉到学习的乐趣。他觉得现在学习就像在织一张大网，每天多干一点活，网就大一点，结实一点，没事查查漏洞，查到就能补救。他感到进步特别明显，急需一次大型考试来证明。头一次，刘天渴望考试能够早点来临。

刘天承认，他的进步，七连特别是曹烽起了很大的作用，语言和文字这两课感觉有点深，除了在作文方面用起来见效果以外，其它科目还不明显；但有关记忆和遗忘的这两课，对他太有用了。原来记不下的东西，刘天只能死啃乱背，就算昏头背下来了，睡一觉又忘一半。现在他的手法多，记忆加法做起来很愉快，根本不可能有背不下来的东西，只要愿意，有足够多的时间，刘天觉得背圆周率100位，也是很轻松的，给他时间，他敢背新华字典。

给我一个支点一个扛杆，就能撬动整个地球。对，正是阿基米德找到真理的感觉。

至于曹烽后来说到了推理和直觉，现在谈到模块与联想，刘天似懂非懂，而且感觉这主要是理科方面需要，刘天的理科是强科，提升空间有限，再加上时间有限，所以根本也就没兴趣听下去。何况曹烽自己还没想明白呢。

所以，努力吧！期末考试对曹烽无所谓，对刘天来说却是一场真正的竞赛。

二班的其他同学也是如此。

对于差生，关系到明年高三还有没有机会上；对于优生，毕业会考成绩关系到高考时的保送和推荐；对于普通学生，如刘天，关系到你会不会突然犯个致命错误，然后就被提前送入社会了。曹烽少考一门课都没事，而刘天，两门发挥失常就危险，超过三门就要出事，能不紧张吗？

全班如此。能来上晚自习的都来了，学生多了四分之一；能回家复习的几乎都不住校了，宿舍少了三分之一。里里外外，人流起伏很大，凭空制造了紧张空气，压得人喘不过气来。

而曹烽，他却在准备竞赛，两大竞赛等着他呢，先是数学竞赛后有物理竞赛，任何一门竞赛，如果能够考进复试，曹烽就能离开青云中学，去省城参加培训；如果复试再考好，就会去北京培训，拼国际比赛名额。曹烽复试的机会很大，只要复试，他就算脱颖而出了，就被大学提前录取啦。

所以，曹烽和刘天是两个世界的人，这一点，人盯人时一定永远不要忘记。

何况曹烽还是一个永远都让人迷惑的神人，即使这样重大的比赛，曹烽都没有放在心上，一天到晚弄他的推理和直觉，模块和联想。

刘天是明智的，不能陪曹烽胡闹下去。那样的结果是两个都损失了。曹烽一个人，没了听众，估计冷静了，反而容易走出来吧。

实事证明，刘天幼稚了一回。曹烽在强烈欲望的驱使下，陷入深深的孤独，他一言不发地吃饭睡觉，沉默中有绝望的表情，高兴时露出怪异的笑容，标准的走火入魔状态。要不是他时时手里都有数学物理的书，估计可以直接送医院了。这状态一直持续到期末考试前一晚，熄灯好久了，几天不说话的曹烽收好书刚爬上床，突然喊：“黄永庆。”

黄永庆没理他，大家都没理他，曹烽咚一声跳下床，跑到黄永庆床上又喊：“黄永庆，醒醒。”他喊醒半睡的黄永庆，只是为了说12个乒乓球的答案有了，而且还比较简单。他非要让黄永庆听听，黄永庆气得直告饶：“老大，明天考语文，你吓得我又忘了两篇古文了。”

“啊？明天考语文？”曹烽是这种口气。

“啊，明天不是考语文吗？”被床晃醒的徐逊莫名地跟着吓了一跳。

没完全睡着的苗家伟干笑两声，没答话，翻了个身，继续睡。

曹烽爬上床，兴奋得睡不着。他翻来复去地翻身，让下面的徐逊十分恼火，他喊曹烽不要乱动。曹烽看徐逊没睡着，探出头来说：“13个乒乓球也没有问题。”

没人理他，只听见他自言自语地那儿说着：“40个称四次，121个称五次，364个称六次，所有3N+1都行……”

过一会儿曹烽坐起来，打着手电开始写着画着，刘天睡了一觉醒来，发现曹烽还在那里写着，过去问他为什么还不睡，曹烽说反正睡不着，起来给《连报》再写一篇稿，当练习作文了。题目就是《联想：模块的万有引力》

“搞了半天，还是一道数学题！”曹烽像说梦话一样，最后嚷嚷了一句，睡去。

第二天，曹烽就跟没事人一样恢复了正常，和大家一起进考场，答起了语文题。

刘天一看作文题，极为惋惜：以“———的联想”为题，写一篇文章，要求……几个小时前，曹烽就在写同名的作文，刘天也没多看一眼，也没多问一句，太亏了。

一天又一天，连续三天的考试，宿舍的人越来越少。最后一天黄永庆都不来了，也没见曹烽拦着他讲12个乒乓球的事。

曹烽真有定力，说放下就放下，说考试就考试，整个就是一个机器人。

考完了，大家特别放松，曹烽还是没有提乒乓球的事，这就让刘天觉得不明白了。他感觉自己考得不错，既想放松一把，又想让曹烽开心一下，更想知道一下这题到底有多难。他去问曹烽怎么做出来的。

曹烽说：“和你讲不明白的。”

“有那么难吗？”他追问。

“不难，只是中间一步被常识套住了。”曹烽说。

“什么意思？”

“天平是可称三种信息的，就像二维人不理解三维世界一样，三维人也不理解四维时空。”

“算了。那晚上说3N+1是什么意思。”

“那个意思就是如果称四次，就可以增加到40个球……给你365个球，称六次，就能找到那个次品，我觉得一年365天，每周六天，加一礼拜天，这个数据好像很巧合……”

“算了，我不问这个了，作文你写的什么？”刘天换了话题。

“记忆的联想。”曹烽答。

果然是这个题，曹烽会在作文里写三十六计的《无师表》，写集体背《阿房宫赋》，谈推背图和圆周率，扯计算机和12个乒乓球，这作文，得多少分呢？这家伙，运气也太好了吧。他试着问曹烽是不是写这些内容了。

“你傻呀！写这些东西？你都听不懂，语文老师看得懂？”曹烽白了一眼他，继续为《连报》写稿。

只是，这次曹烽写了又撕，撕了又写，反反复复地痛苦着，终于没有写成稿。

曹烽这回又疯了，他的大脑本来是很先进的模块，一联想就乱了套了。

他是被联想整疯的。
