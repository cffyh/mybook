# 抽象上升到具体

马克思主义认为，理论体系中的“细胞”概念是理解复杂系统的基础单元。

它是一个相对的概念，意味着在不同的理论体系和研究领域中，“细胞”可以有不同的表现形式和含义。在一种理论体系中作为基础和起点的“细胞”，在另一种体系中可能是成熟的或发展了的形态。这种相对性体现了马克思主义辩证法的思想，即事物是普遍联系和发展的。

在马克思主义哲学中，经济范畴如商品和货币，不仅仅是抽象的概念，它们在现实世界中有着具体的物质形态和作用。马克思批判了蒲鲁东的唯心主义经济学观点，强调经济范畴是现实经济关系的反映，它们的存在和真实性是建立在这些关系存在的基础之上的。

此外，马克思主义强调，任何科学理论中的基本范畴和概念，无论是化学元素、生物细胞、语义学的词、构词学的字母，还是数学集合中的元素，都有其现实世界的对应物和原型。这种唯物主义的方法论，**体现了马克思主义对知识与实践密切联系的重视，以及对理论必须基于并能够解释物质世界的坚持。**

通过这种方法，马克思主义旨在揭示社会和自然现象的内在联系和发展规律，从而为人类社会的进步提供理论指导。





**数学、物理、化学等有一套自有的运行规律，技术也如此-《技术的本质》，他们都和人、和实事求是关系不大。作何解呢？**

- 他们都是相对简单和纯粹的领域，他们和人、社会在更底层的系统、控制层次统一。
- 他们又能和政治社会经济互补，认知和技术是工具，行动实践是载体，改造环境以利于生存是目的。

### 抽象上升到具体，这个上升的难度在哪里？

- 如何上升：以有限生成无限，思维状态由抽象(简单、低级)到具体（复杂、高级）的过渡
    - 分解，分类，链条。
    - 重复，递归，组合。
    - 相互作用，转化。需要用我们大脑来"**生成**"本不存在的世界。
- 对比：理解和认识，也需要生成。但抽取本质概念是一个信息精简的过程，而抽象上升到具体，是一个信息丰富的过程。对精细和精确要求更高。
    - 精确和精细了，就涉及到有限和无限的矛盾，那么“目标”就非常重要了，让注意力聚焦而且坚定。

抽象上升到具体，除了有通过逻辑的方式以外，还有如艺术一样的，

通过直观的方式，直达事物的本质，然后通过具体的形态表现出来。

- 从本质到具体也要经历解构，然后丰富，需要利用技巧。
- 只是具体路径不同，逻辑的方式是推演，艺术的方式是手艺。
