# 特征变换

在机器学习和数据科学领域，特征变换是数据预处理的一个重要步骤，它可以帮助提升模型的性能。以下是一些常用的特征变换算子：

1. **标准化（Standardization）**：
    - 目标：使特征数据具有零均值和单位方差。
    - 公式：(z = \frac{x - \mu}{\sigma})，其中 (x) 是原始特征，(\mu) 是均值，(\sigma) 是标准差。
2. **归一化（Normalization）**：
    - 目标：将特征值缩放到一个固定范围，通常为 [0, 1]。
    - 公式：(x_{norm} = \frac{x - x_{min}}{x_{max} - x_{min}})，其中 (x_{min}) 和 (x_{max}) 分别是最小值和最大值。
3. **幂变换（Power Transformation）**：
    - 目标：改变数据分布，减少偏斜。
    - 示例：Box-Cox 变换、Yeo-Johnson 变换。
4. **对数变换（Log Transformation）**：
    - 目标：压缩大值范围，减少数据的右偏态。
    - 公式：(log(x))，其中 (x) 是原始特征值。
5. **交互特征（Interaction Features）**：
    - 目标：构造特征间的组合特征，捕捉特征间的交互作用。
    - 示例：(x_1 \times x_2)，(x_1 / x_2)。
6. **多项式特征（Polynomial Features）**：
    - 目标：构造原始特征的多项式组合。
    - 示例：(x_1^2, x_1x_2, x_2^2)。
7. **编码变换（Encoding Transformations）**：
    - 目标：将非数值型特征转换为机器学习算法可处理的数值型特征。
    - 类型：独热编码（One-Hot Encoding）、标签编码（Label Encoding）等。
8. **缩放变换（Scaling Transformations）**：
    - 目标：调整特征的量级，使其在相同的范围内。
    - 示例：最大绝对值缩放（Max Abs Scaling）、最小-最大缩放（Min-Max Scaling）等。
9. **离散化（Binning）**：
    - 目标：将连续特征转换为分类特征。
    - 方法：等宽 binning、等频 binning、基于决策树的 binning。
10. **缺失值处理（Handling Missing Values）**：
- 目标：处理或估算缺失的特征值。
- 方法：删除缺失值、填充缺失值（如使用均值、中位数等）。

这些变换算子可以单独使用，也可以组合使用，以适应不同的数据集和机器学习任务。在实际应用中，选择哪种变换技术往往需要根据数据的性质和分布、模型的类型以及模型的性能来决定。
