# 玄学拉回现实

# 前言

1、读书、做事需务实，我之前写过一篇文章提到机会很重要，还有一些佛学的理念，然后有些读者会惯性的往玄学带，这是我不提倡的。

“一命二运三风水”，试问一下，你能改变什么，又能改变多少呢？这句话有一定的道理，但是很空，对很多人没有实质帮助的。我认为人们会常常把玄学挂在嘴边从而感觉高人一等，或者感觉说完这些会让别人觉得你十分聪明，就像我们吹牛逼的时候只要是你盲区的地方你就会觉得对方很厉害一样的道理。

但是试问一下，这些真的帮到你了吗？所以从玄学拉回现实务实～

佛学、道德经都是讲述无神论，只不过在传播过程中，会有地域特色，或者我们对那种精神的追逐、向往，从而映射出某个相。

不是非黑即白

上面讲述读书、做事要务实，但是不代表玄学里面的东西就不存在，对于自己未知的东西要敬畏，我想表达的意思是真正有意义的东西，或者讲你能改变的东西只能是脚踏实地的东西。

还是那个例子**，你是水里的鱼，却天天想着天上飞的东西，没有什么帮助的～**

2、读书不能全盘接受，需要思考

网上或者广为传播的书籍里面，像曾国藩有家书、冰鉴，包括我们看的一些很经典的书籍，里面常常在告诫我们的修德，或者一些品行。我认为这是没错的，这是人立足之本，它之所以能够被广为流传，我认为是符合我们的价值观，还有我们提倡的精神。举个例子，大部分主流宗教就提倡行善、宽容。

这些东西做人必学的，**但是如果你想有所作为，这远远不够的。**我尝试在网上查了一下曾国藩最高的职位相当今天的部级，一个学历好点的终其一生也就是处级，可想而知这是多么厉害的人物，他需要处理的事情是十分棘手的，**如果你只停留在上面讲的德、品行，远远不够的，那只是做人基本素养。所以我认为读书需要多琢磨，不能人云亦云。**

# 《道德经》

我在微信读书看了《道德经》，我的感受是：这是一本通过格物致知来推导人道的书籍，格什么呢，格天地生存之道，通过最原始的启发来领悟人应该怎么处事。

整本书是老子在讲述自己对于生存之道、养生之道、治理之道的看法，提到无为而治，上善若水，顺势而为等经典的思想，但是它是作为一个比较理想的层次去谈的，具体到某些场景需要我们灵活变通，那么我们走近去了解一下这本书对我们的启发是什么。

其实就是跨界应用，这也是很多书籍提到，或者很多大佬会讲到的跨界应用能力，当你对原理透彻之后，能够跨过当前场景进行应用，这是一项非常重要的能力。

摘录内容

1、道可道，非常道；名可名，非常名；

很多人会习惯往玄学靠，我们从上面的定义可以知道，老子通过格物来推导，那么世间的规则怎么可以用一个词或者一句话来概括呢对吧，只能模糊讲，它其实就是一种势。

怎么形容呢？

**这股力量推动事物变化，推动整体前进，这股势孕育着世间生命，有着变化的趋势**。

2、天地长久的生存道理

天地包容了世间万物，包括好的坏的，孕育世间生命，而它没有因此而炫耀，忘身而身存，这是天地能够长久的秘诀。那么“对应天地不仁，以万物为刍狗”，自然也就好理解了，天地之间它有一定的规则，或者说运作的规律，在这些规则里面没有对东西做特殊的识别，一视同仁。而正是这种近乎原始的“道”，带动生态系统的变化，从而不断演变，生生不息～

张磊：“河水不争先，争的是滔滔不绝”，也是道家的思想，正是这种互相转化，互相推进，才能让这个系统长久进行着，这里面的推手就是“道”。

在商业上能不争吗，肯定是不行的，但这里更多的**是表达长久、源源不断的意思，这也体现《道德经》是一种辩证思维，要想长久就是不断更新迭代**，而不是为了争而争。

对于我们什么启示？

无常是常，当人在走上坡路的时候容易得意忘形，居功自傲，而这种上升趋势并不是一直持续的，水满则溢，容易栽跟头。对于处于高层的人，会更加谨慎，水它滋润、孕育了万物，按理来讲功劳非常高的，而它却处在很低的姿态，所以不被人所嫉妒，所针对，最终得以长久。

曾仕强：“一般厉害的人出场的时候都十分的低调，没有人看得出他的厉害”，当然对点我觉得分情况来讲，厉害的人之所以低调，是怕树大招风，跟上面水低位是一样道理的。但是对于现在的年轻人成事，我觉得需要学会造势，你都低调到了尘埃里面了，没有大佬赏识，你觉得还有出头机会吗对吧。所以低调需要讲究火候，不是什么时候都要低调。

比如说刘备，为啥到哪里开头就是一个刘皇叔，你只有把名头立起来了，才好做事，而至于低调那是自保的时候用的，学会变通。

3、无为

按我理解是不瞎操作，而是去参与，去推动这种势，这才是有意义的。

举个例子，我遇到非常经典的情况，就是家长教育孩子的时候，靠着嘴去言传，这个效果非常低下的，就是你说了几年，但是真正有帮助孩子改变的东西很少。

所谓的无为**，不要去做没有意义的动作，而是你了解这个结果是怎么产生的，原理是怎样的，然后去参与这个过程推动事情的前进。**比如说水在一个大气压下100摄氏度会沸腾，你一直在旁边喊着根本没有什么用，只有参与这个加热过程，水才有可能沸腾，这是自己作为榜样去教导孩子，如果你自己都做不好还一直指导别人，没有什么根本的作用的。

曾国藩《冰鉴》讲，君子临事的态度：入局，不管什么结果都义无反顾承受，你只有参与到事情中，成为这种推动事情里面的一股力量，成为一种势，才能真正改变点什么。

4、知其雄，守其雌

书中德篇也提到了大成若缺，花是半开的时候最好的，当全部绽放的时候也意味着凋零的开始；还有“反者道之动”，事物总是朝着自己的反面运动。我们可以理解世间为了保持这个活力，需要彼此转换才能有生机。

对于我们什么启示？

按一个循环来讲，当我们从低处的时候，保持乐观，因为已经是低处了，再低也没低到哪里对吧，当我们处于高处，保持谨慎，因为处理不当随时也会往下走。

# 我的感受

我很明显感觉《道德经》这本书从格物致知角度来讲感悟的，“人法地，地法天，天法道，道法自然”，人应该跟随地运作规律，地应该跟随天...

1、格物致知的缺陷，或者局限

你观察天地、水的规律来总结，这些力量是多么强大，水可滋润万物，亦可剥夺所有。如果人从这些强大力量角度去学习，会因为自身能力不足而有所偏差。举个例子，东施学着西施样子，结果更丑了。

所以我们需要吸收对自己有用的部分，在低位的时候保持乐观，在高位的时候保持谨慎。

2、顺势而为

所谓的“道”，就是推动各种东西运作的那个无形的手，你可以理解为“势”，就像水往低处流，那么会不会往上走呢，也有水蒸气，正是这些循环给予了生机。

顺势而为，是很重要的一门课，首先你要了解是怎么运作的规律，当古人不知道重力或者没有这个概念的时候，诞生了阴阳，类似阴气下降，阳气上升，当你无法准确表达的时候，靠着比喻、抽象；然后了解这种运作规律之后，参与进去其中成为势的一部分，去推动到我们想要的结果，就是临时态度入局，以及无为里面勿瞎操作，你遵从原理基础去推动顺势而为。

3、没有绝对的真理

我看有些视频号会讲孔子老年的时候演讲老子的书籍，表达道德经才是高于这些礼教的。如果从认知的角度，《道德经》是比较高的，就像你研究人性，跟你研究人的行为，一个从本质来搞，一个从表象来整，层次不一样的。

如果从应用来讲，实践出真理**，丛林法则才是最终主宰着运作，有个政治家提出了棒棒糖跟锤子理念，还有大炮范围即真理，大量的实践表明了人类社会需要规则的**，《道德经》的“道”，是**真正强者才有能力实施的，天地、水都是非常强大的力量，**如果三体里面的二向箔存在的话，这些力量怎么保持低调呢？瞬间从三维压成二维。
