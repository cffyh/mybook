# 思维模型梳理

从空间分布上，区分为微观思维和宏观思维两大块，所谓微观，主要体现在点线面层次，所谓宏观，主要体现在立体层次。

1. 在点这个位置，我们要学会切换不同的位点，或者说叫做切换不同的站位点，也就是不同的观看事物的视角，进而能够尽可能的看清楚事物的各个角度上的样子，做到全面的看问题，这种思维我们叫做“换位思维”。
2. 在线这个位置，我们要学会找到这条线的踪迹，而这条线，可以是时间序列上的，也可以是逻辑序列上的，从哪出发呢？当然是我们当下所站的这个点作为起始，如果你要找线的前端，就要通过“5why法”，往回找原因；如果你要找线的后端，就要通过“5so法”，往后寻结果，这种思维我们叫做“思维逻辑链”。
3. 而在面这个位置，我们要学会从一个面跳到另一个面上，当我们需要建立知识体系或者对一个事物进行认识的时候，就需要将散落的抽象的点和线，通过将其建构成图像、图表或者是一个画面，只有这样，所有的点和线才能够构成一个整体，不仅能够更好的方便我们输出，也可以更好地让我们输入内化。这种思维方法，我们叫做“可视化”。

以上说的是微观上的点、线、面，下面我将特别说一下在体这个层级上的思维方式。

1. 从立体层次，我们先说点和线的结合，我们在分析设计一个系统的时候，不是胡编乱造的，而是有序的建构，如何有序的建构？就需要利用到我们前面学到的“思维逻辑链”，将系统里的关键环节，通过逻辑顺序连接起来，形成一个流程，但此时，建构流程，与前面的思维逻辑链有所不同，思维逻辑链要的只是把系统里面的逻辑链条给找到，但我们现在是需要把这根链子用起来，怎么使用呢？这条线不是一条干巴巴的线，而是一个个模块按照逻辑顺序连接起来的生产线，这些模块，我们可以当做是一个个关键的“点”，我们如果是需要优化整个系统，就需要在点和点与点之间的线段上做文章，把每一个点和线分立出来各自进行优化后回归到系统内部的时候，由于整条逻辑线是一个串联路线，分立点的变化是能够起到相加甚至相乘的效果的，换句话说，每一个环节点的一点点微弱的优化，都会在整个系统上呈现全流程的优化，这样的一种思维过程，就叫做“流程思维”。

流程思维，还只是一种线性的推动，或者说只是一种有方向性的线性推动，但是我们的现实其实是非常复杂的，它远远不只是简单的点或线甚至是面，而是一个在系统内部方向交叉空间位置复杂的网络系统，面对这样的复杂问题，我们就需要从系统上看世界。

2. 在一个复杂的系统内部，所有的元素（点）之间会形成相互关系（线），重要的问题在于，点与点之间的这条线，不再是简单的单向性的箭头，而是双向性箭头，那么这样的点和线又会构成一个复杂的面形网格，但是请注意，这个面形网格不止一个，而且还不是欧式几何的面，而是非欧几何的面，是曲面，面与面之间，也是通过点和线乃至面进行交叉链接，构成一个复杂的“体”，这个体，我们称之为“系统”。而我们如果需要真正的了解真实的世界，就需要建立“系统模型”，这是最接近于真实世界的模型体，通过不断地进行剖析和量化，利用“换位”找到不同的“点”，再利用“思维逻辑链”找到点与点之间的“线”，最后利用“可视化”将整个系统真正的具体化、模型化建构出来，这就叫做“系统思维”。
3. 系统，是最接近世界本身的模型体，它涵盖了动量和静量，是一个涵盖所有事物的模型，但在具体的建构过程中，如果更多的关注“动”的量，或者说能动的生命体，我们就可以把这个系统看成一个生态圈，你的视野里就会呈现所有的自发能动的“生命”，这些生命在系统内里根据自身的生存需要，各自找寻自己生存出路，它们在不断地运动着，这些运动不断地作用于其他生命乃至无机的生态圈实体，当我们眼里呈现这样的场景的时候，你不仅能够理解万物，甚至于能够体会到“道”的运行，而此时你将自身置身于其中，不管我们是选择斗争也好、合作共生也好，你都知道怎么样去与整个系统“相处”，从而不会逆天道而行，这样子的话，我们就算是建构了“生态思维”。
4. 我们看见了整个系统的运行图景之后，体会到“道”的运行，请注意“运行”两个字，建立系统乃至于生态系统的时候，我们只是看见系统的“空间化”，但是我们还知道，世界是在变化的，它在时间上是发展的运行着的，那么这个系统中道在时间序列上的运行方向，我们称之为“势”，通过找到系统的势，我们就能够“顺势而为”，顺应天道而行，在文中作者列举了“马太效应”、“科技进步”、“人口规律”等几种规律，可作为具体的操作指南，但我们可以看见这些规律的本质在于：决定性因素对系统的损益，是一个正反馈效应，将不断形成递归循环，构成叠加效应。这种思维，我们称之为“大势思维”。
5. 那么，我们在这样的一个复杂系统里，除了认识到系统的“空间化”和“时间化”之外，还有什么应该需要掌握的呢？当然有，我们前面说的“生态思维”，看的是如何“相处”。所谓相处，更多强调的是合作关系，但合作和竞争是辩证统一的，合作是为了在更高层级上进行竞争，竞争又能够保证合作体的构成。在竞争的这个方面，我们就得学会“兵法思维”。

这里面有几个方向问题需要掌握，第一，“先胜后战”，所谓先胜后战，也就是孙子兵法的“计”，计，不是阴谋诡计的计，而是计算，就是要计算你在竞争中胜算可能性，有胜算才战，无胜算则不战，在具体操作中，古代用的是一种经验性的对比性估算，而现代由于数学的发展，我们可以使用概率学上的“凯利公式”，对胜算可能性进行数学建模，能够更好地指导我们的活动。

第二，“致人而不致于人”。这里面的核心要义是，你得找到系统里最重要的“位置”，前面已经说过，系统是一个立体性网络，但是这个网络不是每一个点、线、面的地位是平均分布的，有的位置重要、有的位置不重要，有的位置更具有决定性作用，有的作用则微乎其微，重要的位置，是能够对其他位置进行作用、限制、牵制的，也就是“致人的”，而有些位置是被作用、被限制、被牵制的，也就是“致于人的”，如果说我们需要在系统里更好的生存和发展，就需要到“致人的”的位置上去，而少到 “致于人的”位置上去。

第三，“胜可知，不可为”。我们通过计算，掌握到敌我双方的局势之后，确定到自己在系统里的生态位之后，你需要让自己置身于“致人的”位置，并掌握事物发展的“概率”，我们只要顺势而为，做到自己的极致，由此我们就立住了，我们就“不可胜”了，立于不败之地了，至于能不能胜，我们需要看对方会不会犯错误，就需要“等待”，也就是用到“以慢制快思维”，把眼光放长远，不要计较眼前的得失，而是把生命看成一场棋局，眼前一棋一子只是时间和空间上的局部，只要我们领会了道、体会到了规律、把握了趋势、确定到了位置，剩下的事情，就是持之以恒的把自己做好，剩下的一切，就留给时间就好了。因此所谓胜可知，强调的是知道如何做到不败，不可为，指的是留给时间。

最后，用叶修的结语结束：你我都在命运的局中出生，但谁会在樊笼里终老？《深度思维》讲述的不仅是思维的技术与格局，也是人生的策略和对平庸命运的抗争。
