# 如何写PPT

写PPT 是基于图形符号将**概念+关系可视化的过程，具体如何操作呢？**

### 概念展开

### 组装

将基本单元组装起来，来刻画全景，或完整的表达一个主题。

如何组装，看成一个工业产品：**内在运作机制串联起**构件 → 模块 → 成品，将其**挖掘并展示**出来。[逻辑展现与艺术表现](./逻辑展现与艺术表现.md)

组装是递归的，单页是一个相对完整的全景或者主题，全部页面再整体组成一个完整故事。

### 展示

与一篇文章类比，文章是一维的，PPT是二维可视化。

- [我们的思维本质是“网状”的，而语言表达却必须是“线性”。](craftdocs://open?blockId=1B5400AC-2046-4C10-B8D7-F7C86849B007&spaceId=44a12fe2-554c-518c-5b22-cbde920acd16)
- **一维（文章/口语）：** 是线性的、时间性的。听众必须从头跟到尾，**在脑海中自行构建逻辑框架和图像**。信息传递效率低，且容易遗忘。
- **二维平面（PPT）：** 
    - **形状与线条：** **直观**地**表达**流程、循环、包含、对立等“**关系**”。
    - **位置：** 上下左右体现层级、顺序和关联。
    - **大小：** 标题大，正文小，突出重点。
    - **色彩：** 区分不同类别、强调关键数据、营造情绪氛围。

### 量化

> [行为（运动）的结果是现象；主动运用（弱者道之用）⇒ 客观—>主观（主观能动性）。](craftdocs://open?blockId=08C59B5C-AB32-4C27-8C0C-A350ED827B2C&spaceId=44a12fe2-554c-518c-5b22-cbde920acd16)

1. 量化→ 精确的比较和计算，给后续动作（判断-决策-下一步action）确定性的信息和依据。
2. 分为过程和结果。
3. 信息量丰富： 精确→ 确定性高 → 信息量大。
