# **关于拉马努金的笔记本**

[https://gitcode.com/open-source-toolkit/9c76f/?utm_source=tools_gitcode&index=top&type=card&](https://gitcode.com/open-source-toolkit/9c76f/?utm_source=tools_gitcode&index=top&type=card&)

2019年数学界值得记住的一件事是拉马努金最权威的《笔记》在大陆出版，足足九大本，4000多页（当然不是原始的笔记。据说《哥德尔文集》也可能被译成中文，打算以后买回家膜拜一下）。拉马努金只活了32岁，如果他活64岁，情况会怎样，这书的页码会翻个倍、变成8000多页吗？考虑到拉马努金不可能在10岁前进行高水平数学研究，所以很可能翻倍也不止，至少10000页吧！

我查了一些书龄超30年（即一代人，这就除去炒作，比较客观）的豆瓣高分作品：《红楼梦》9.6-，《费曼物理学讲义》9.5，《计算机程序设计艺术》9.4+，《圣经》《哥德尔、艾舍尔、巴赫——集异璧之大成》9.4，《西游记》《三国演义》《古今数学思想》9.2，《自然哲学的数学原理》《几何原本》《从一到无穷大》《人类简史》9.1±，《时间简史》8.8，《生命是什么》8.8-，《水浒传》8.6+，《物种起源》《自私的基因》8.6-。也许高于8.5分的都是现象级图书，可遇而不可求。

霍金的《时间简史》无疑是上层佳作，但若非作者残疾，它会同数十本科普书挤在一起，不会太引人注目，就其特色而言比不上《哥德尔、艾舍尔、巴赫》，而《拉马努金笔记本》其原创性、独特性又要高出一个层次。我总感觉《哥德尔、艾舍尔、巴赫》尽管是本奇书，仍是人写出来的。格罗滕迪克的《代数几何原理》、高德纳的《计算机程序设计艺术》是大师巨作，当然这些书里很多工作也是在前人基础上整理、总结出来的。而《拉马努金笔记本》才更像天外来物，但愿有足够多的人评论，也必然是豆瓣高分吧。

这些书连同其他一些深奥的数学、物理学、哲学乃至少量佛学书（我对佛教义理有点兴趣），如今都安静地"躺在"我家里，有时我也会好奇地问自己：读懂它们需要花多少年？

我可能永远不会去读懂它们。买这些书，也算是向天才、大师致敬的一种方式吧（当然我买的绝大多数书比较通俗）。

外貌、体育、战争、数学都属于硬实力的比拼。中国古代的历史小说，像《三国》《西汉演义》《水浒》《岳飞传》之类，虽有英雄排座次，也不像《说唐》那样吹第一英雄好汉比第二名强到不知哪里去。真实生活中，我们固然也可以见到第一名明显强于第二的情形，如短跑名将博尔特，当其一骑绝尘、甩开第二名十多米冲向终点并大幅打破世界纪录，我们感叹以后恐不能再有这样的外星人出现；但有人指出数学能力的差异可能最为明显。现在习惯用学霸来描述那些尖子生，那么拉马努金又是什么样的存在呢？这位被哈代誉为当世最强数学能力的人物，竟然是从梦中得到大量高深数学公式（其实估计是不知不觉自己推导出来的），可谓前无古人后无来者。也正是这个原因，多年来，有几百篇论文研究他的思维奥秘，没得出什么有价值的结论。

某种程度上数学家是最强大的一群人，但并未受到广泛认可，相比之下，认可六块腹肌的人可多了去。对于感官刺激的纵容，也算是世俗社会的一大特产吧。

但也许，拉马努金及其成就若能广为人知，那么世间的抱怨和自欺至少减掉一半。

不是吗？他才华逆天，却命运多舛、英年早逝，在英国期间也因生活不适而自杀（未遂），如今头像却差点上了英镑。芸芸大众的日子可能比他还好过一点，又谈不上什么才华和贡献，有啥好抱怨的。

不是吗？他和爱因斯坦一样传奇，却少了很多"毒性"：爱因斯坦名气太大传播广泛，有人想效仿，当然最终大多没什么好结果（跟爱因斯坦本人无关），陈景润更甚。而拉马努金的女神托梦一看就没法效仿，反而让大家更清晰地认识自己。

试想如果当年拉马努金一直待在印度，或许能活得更久，但在数学史上完全可能湮没无闻。他的选择使其从尘埃变成了流星，流星虽短暂，却光华四射，永载数学史册。

拉马努金离世已整整100年了。天才已逝，思想长留。而小市民们追求的及时行乐大行其道，也终究不过在虚幻中自我安慰而已（当然，这也完全可以理解）
