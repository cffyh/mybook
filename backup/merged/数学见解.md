# 数学见解

为啥微积分、线性代数、概率是高等数学中基础中的基础，有几个深刻的原因：

- 初等数学是常量的数学，解决较直观的数和形的问题。当问题朝着更复杂的方向发展，最核心基础的便是几门学科代表的几个角度的拓展了：
    - 微积分，函数，研究变化；
    - 线性代数，计算的对象需要拓展到多维；
    - 概率，刻画世界本质复杂性带来的不确定性。
- 所以，三门学科是三大高等数学的基础，而不是较独立的领域，作为基础，在应用中是经常同时出现的。

高等数学相对于初等数学，在复杂和深刻性有了质变。但数学的核心依旧没变：

- 一门描述模式的语言
- 一种思维方式，而且影响到科学
    - 转化、简单生成复杂的演绎推理、概念和技能的双重属性。
    - 算法：将问题描述、建模。

数学与数学之外的世界

- 数学是最纯净的系统，最接近本质；
- 相似相异
    - 不同粒度下的抽象，有相似的规律，数学世界和现实世界里的规律可以互相启发。
    - 同时，不同抽象粒度的世界，有自己世界下的不同规律。
    - 在相似中寻找不同点，而这不同点又能从别处找到相似，这样又回到递归重复感觉。

有限和无限之间互相的转化，在技能掌握上，体现的淋漓尽致：

招式是有限的，但变化是无限的；

人要掌握变化，只能通过具体的例子。但人精力有限，能学习的例子是有限的，但人能够通过有限的例子，提炼出有限的招式。

数学包含了技能、概念，还有符号-演绎思想，一种转化问题有效方法。

另外的两个矛盾转化，复杂和简单、抽象和实际也有体现。
