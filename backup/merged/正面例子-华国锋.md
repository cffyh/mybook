# 正面例子-华国锋

1976年9月9日，伟大领袖毛主席去北京去世，很多人以为主席没有留下遗言。实际上，毛主席早就给华国锋留下了三句话，这三句话分别是：“一切照过去的方针办；大胆去做，不要怕；你办事，我放心！”

这三句话被华国锋奉为“座右铭”，也成了毛主席选择华国锋为“接班人”的一个证明，当时中央人才济济，毛主席为何要选择年轻又没有军事经验的华国锋为“接班人”呢？

1976年1月，周总理去世后，毛主席推荐华国锋接任周总理，主持国务院的工作，消息一传出来，国际上纷纷对华国锋的身份好奇不已。

在此之前，华国锋没有频频在高层出镜，外国报纸将华国锋称为“中国政坛的一匹黑马”，正如他们描述的那般，异军突起的华国锋确实是一匹“黑马”。

华国锋出生于1921年的山西交城，本名苏铸，参加了抗日救国先锋队以后，才将自己的名字改为“华国锋”，以此来表示自己的革命决心。

抗日战争全面爆发后，华国锋不顾家人反对离开了交城，前往前线抗日，当时华国锋才17岁，因为上过学、有文化，在队伍里担任敌后宣传工作，在他的宣传下，越来越多的爱国青年加入到了抗日队伍中，从这个时期就可以看出，华国锋拥有很强的宣传组织能力。

新中国成立前夕，华国锋跟随南下工作队来到了毛主席的故乡湖南，没想到他在湖南一干就是21年。

华国锋将湖南视作自己的“第二故乡”，一股脑投身于湖南的建设和发展当中。

华国锋来到湘潭的时候，湘潭还是一个落后的山区县，连条像样的公路都没有， 汽车都开不进来，为了改变湘潭积贫积弱的现状，华国锋带领县委工作小组跑遍了湘潭每一个村落，开展农业合作化运动，提高各地粮食产量，只有老百姓富了，湘潭才富，富了才有钱修路。

1955年，华国锋第一次见到了毛主席，这次见面给华国锋留下了难以磨灭的印象，也为华国锋后续的发展奠定了基础。

老实忠厚、脚踏实地，是华国锋留给毛主席的第一印象，也成了主席选择华国锋为接班人的重要原因之一。

华国锋的眼光非常长远，当时他就看出，发展农业生产是带动湖南经济发展的重要原因之一，他非常重视袁隆平的“杂交水稻技术”，为了支持袁隆平他力排众议，拨款150万元用于杂交水稻的推广工作。如今看来，华国锋的这个选择实在是太正确了，直接改变了全国人民的生活。

人品好、政治眼光长远，这都是毛主席选择华国锋为“接班人”的原因。除了这两点以外，华国锋有一个优点，那就是拥有丰富的基层工作经验，且年龄也合适。

毛主席认为，只有这样的干部才是真正沉下去的好干部，将来才能脚踏实地地为人民做实事，不管他站在多高的位置上，心里都装着人民，这才是主席想要的好干部。
