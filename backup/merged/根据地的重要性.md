# 根据地的重要性

教员在土地革命时期，有一项非常重要的创举，就是[根据地建设](https://zhida.zhihu.com/search?content_id=254714347&content_type=Article&match_order=1&q=%E6%A0%B9%E6%8D%AE%E5%9C%B0%E5%BB%BA%E8%AE%BE&zhida_source=entity)。

大革命之后，白色恐怖笼罩，革命陷入低谷，这个时候，苏联的经验，已经完全不适用啦。苏联的经验是城市斗争，但这个时候，城市的反动势力空前强大，相比之下，革命势力非常弱小，根本打不下来，留下来搞搞地下工作，也全是徒劳，革命者不怕死，但这样毫无意义地牺牲，实在没有必要。

革命的出路在哪里？苏联人没有办法，革命者也没有办法，只有教员，指向了农村。现在看来，此举理所应当，但在当时，没有人理解，因为其他人根本不了解农民阶级中蕴含的力量，而教员，因为开办过农民讲习所，非常清楚这种力量，也知道怎么运用这股力量。教员就是这样，不受任何教条束缚，目光极其敏锐，看准的事情没有错过。

但即使方向正确，实施起来也不容易，因为，你要发动农民，那你就得了解历代农民起义的特征，基本上就是[流寇战术](https://zhida.zhihu.com/search?content_id=254714347&content_type=Article&match_order=1&q=%E6%B5%81%E5%AF%87%E6%88%98%E6%9C%AF&zhida_source=entity)，跑来跑去，让官军抓不住，如果固定在一个地方，很容易被剿灭，所以，农民起义必须在官方统治基础千疮百孔的情况下，才能发展壮大，要不然，跑着跑着就跑不动了。很显然，当时的反动统治，虽然军阀林立，但个个手握重兵，一帮流寇，肯定掀起不了风浪。更何况，我党不是一个盲动的组织，有着自己的理想和信念，流寇只能搞破坏不能搞建设，这也不是我们想要的。所以，我们要反对流寇思想，搞根据地。

前面说了，力量弱小的一方，如果不跑来跑去，而是固定在一个地方，很容易被强大者所剿灭，教员为什么不怕呢？因为军阀林立。军阀林立的坏处是，你跑的话，没有多少地方可以跑，因为每个军阀都会捍卫自己的核心地盘，但固定下来，反而有地方可以固定，**那就是，几个军阀的交界处，三不管地带，说白了，就是利用敌人之间的矛盾，在缝隙中生存**。而且，**考虑到军阀之间还可能打仗，趁他们打仗无暇顾及的机会，我们的根据地还可以扩大**。这就是教员的特点，具体情况具体分析，既然这种情况全世界都没有，就中国有，那中国就要在现有条件下找到最优解，学别人是没得学的，因为情况不一样。

因为只是发现根据地模式走得通，还体现不出教员的水平，教员要在这小小的根据地里，实现他理想的雏形。

首先就是土地革命，中国必须搞土地革命，往近了说，“打土豪分田地”可以释放农民的积极性，为革命提供持续的兵源；往远了说，中国社会需要完成农业社会向工业社会的转型，土地集中在地主手里，是不行的，虽然把土地分给农民，本质上还是私有制，但为将来土地公有制打下了基础，不把地主阶级消灭掉，中国的工业化转型就不可能实现，反正早晚要消灭，革命的过程中顺便消灭掉，以后的麻烦少很多。其实，蒋光头也知道必须动土地，但窃取政权之后，发现地主阶级是他的统治基础，所以完全打消了想法，蒋光头成就的上限，也就是一个腐朽落后的反动军阀罢了，因为他没有意愿也没有能力解决根本问题。有根据地，才能搞土地革命，如果跑来跑去，你分土地给农民，农民也不敢要，因为，你跑了，地主就要打回来了，分地的农民会遭受报复，所以，你不能跑，得留下来。

然后，就是第二个问题，如何定义军队和群众的关系。教员要建立的是一支新型的人民军队，这支军队是真正的[人民子弟兵](https://zhida.zhihu.com/search?content_id=254714347&content_type=Article&match_order=1&q=%E4%BA%BA%E6%B0%91%E5%AD%90%E5%BC%9F%E5%85%B5&zhida_source=entity)，和群众是鱼和水的关系，但要想实现这一点，不能只有思想教育，得落实到实践中。为了根据地的生存，这支军队也必须和人民群众融为一体，在后来历次[反围剿](https://zhida.zhihu.com/search?content_id=254714347&content_type=Article&match_order=1&q=%E5%8F%8D%E5%9B%B4%E5%89%BF&zhida_source=entity)中，群众的帮助，是胜利的重要保证，这样就进一步强化了这支军队的信念，相信并依靠群众就能胜利，否则就要失败。

最后是第三个问题，如何建立政权。我党就是在根据地里，尤其是后来建成的定都[瑞金](https://zhida.zhihu.com/search?content_id=254714347&content_type=Article&match_order=1&q=%E7%91%9E%E9%87%91&zhida_source=entity)的中央根据地里，学会了如何用新型方式治理国家，为将来接管政权打下了坚实的基础，这一点被很多人忽略了。如果一直流动作战，就算打下天下，也不一定立即学会如何治理，明末李自成，在这方面就吃了大亏。

总之，根据地建设，是教员的重要成果之一，其意义之大，应该引起我们的重视。这一成果，对创业者的启示是：

1、公司一定要找到自己赖以生存的业务，长期深耕，不要什么机会好，就抓什么，那就犯了流寇主义的错误。

2、公司小不可怕，只要能利用强大的竞争对手之间的矛盾，仍然可以找到生存空间，并进一步壮大。

3、要和用户建立良好的关系，这是胜利的重要保证。怎么强调“以用户为中心”都不为过，同时，让大家尝到甜头，对用户好，公司就能成功，反之就会失败，从而把这样的信念强化并固定下来。
