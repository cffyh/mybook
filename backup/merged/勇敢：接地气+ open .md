# **进入状态**

进入状态，**高强度用脑、**争分夺秒但有**节奏**的干起来。

- # 什么是进入状态

进入状态隐含是一种和谐，运转良好的状态。

这个状态，是一个建模仿真效率和质量皆高的状态。



人进入某种持续状态，在此之前先做热身，这跟体育锻炼是一个道理。

不知道大家看过柴油发动机或者手扶拖拉机启动时候的情景没有，是需要人大力持续地手摇把手狠狠地摇上好些圈的，当摇到一定程度，才算正式进入状态，正常运转了

就像打乒乓球，每次热身都要来回抽球。

踢足球，每次热身都要用各种部位颠球。人的运动系统会依据这些热身运动的反馈进行调整，这样正式运动起来才会流畅
