# 理，智，定，力：金刚之法

一个人的成长过程是痛苦的，因为要从一个什么都不懂的白纸逐渐变成**习惯于社会规则，接受自然规律的人。**

这个转变的过程就是人的种种经历。**人们喜欢的是促进，帮助，和谐，但是，批评，否定，指责，压制也比比皆是。**

在这对立的两面中，一个人的行为，思想，以及话语被**改变和塑造**，最终塑造出来的就是相对稳定不变的人格，或者说性格。

那么，要改变性格，就要从这两方面入手，一方面，吸收好的因素，另一方面，排除坏的因素，所谓的去恶扬善。吸收好的方面，通常来说问题不大，因为每个人都有进取之心，而排除坏的因素则是非常困难，因为坏的定义是相对的，他往往被视为一种好用的工具与武器，用来敲打其他人，你对我产生了不利的影响，那你就是坏的，所以我要教你做事，非常有趣，一个善良的人是人人可欺的，为什么呢？因为缺乏对抗坏的力量，但是，什么才是对抗坏的力量呢？善没有力量吗？

这就是我打算讲的东西，如何获得善的力量，击碎坏的攻击？

这部分属于力的范畴，理，智，定，力是四种强大的工具，用来塑造和改造自己以及环境，

理在于**思**，思维，学习，分别，

智在于**无**，无我，无人，无差别，

定在于**安**，安于劫难，安于痛苦，安于混乱，

力则是**破**，破封锁，破邪念，破局限。

这四种德全部获取之后，这个人就具有了无坚不摧的矛，和无物可破的盾，**而矛和盾在智的无分别中合为一体，在理的指导下直击关键，这就是金刚之法。**

金刚之法属于秘法，何为秘法？只有满足了条件的人才有学习的资格。什么条件呢？可以承载金刚的身心。就像钻石一样，他是钻石原矿一点一点雕琢出来的，如果使用的是普通的木炭，根本就雕刻不了，而使用钢铁，则又过于坚硬，雕刻不动。所以，合适的材料是必须的。

然后如何做呢？**那就是要承受砸的力量，刻的痛苦，以及具有保持结构不散的忍耐力与定力，如此，千锤百炼，终成钻石。**

所以，**批评，攻击，否定，压制都是砸，把他们一一接下来，就可以去掉钻石原矿上很多顽固的杂质**，将钻石闪耀的一面显露出来，闻过则喜，面刺则赏，受辱则心思不动，这都是极难极难做到的大丈夫行。

这就是金刚练就之法。那么如果能做到，有什么好处呢？毕竟没有好处的事谁会干呢？**好处就是自在**，不因善恶取分别，善可以成就一个人，恶也可以成就一个人，关键在于度，什么程度为宜，这个度就要在自己承受这些恶的锤炼过程中体会，所谓己所不欲，勿施于人。就像医生需要先在自己身上练习针刺，然后才能在病人身上实践。

所以，**金刚法对于恶劣环境中人的身心成长基本上是必备的，因为打击无处不在，松懈一丝，自己就会被击碎，但是，人总有懈怠的时候，所以，单纯扛的下打击，不算本事，抗的云淡风轻，不以为意，才是本事，到最后，甚至于都不知道是打击还是按摩了。这就是自在之境。**

用现在的话来说，金刚就是抗压，无论是心理压力，还是物质压力，甚至于语言暴力，精神暴力，以及身体暴力，都拿得下。这样的人往往被称为硬汉，决心坚固得难以想象。以至于一个人就能应对一群人，一个人就能吓退一群人。这种勇气倘若用在正道上，就是大众的中流砥柱，可以守护很大一片地方。守护自己，家人，在乎的人，善良的人。

世界就是这么奇怪，尤其是环境恶劣的地方，**善良的人必须自己足够强大才能把自己的善良挥洒出去。**很多恶人是不可能被感化的，因为他们以力为尊，谁的力量大，就服气谁，那么要教化他们就需要满足其愿望，用足够的力量击碎他们的傲慢，这就是社会强制力，对付以力为尊的人最好的方式。

其实，不仅仅是社会治理，在人际交往之中，也处处能看到这种欺软怕硬，攀炎附势之人，他们本身没有多少力量，却攀附依赖于某种强大的存在而狐假虎威，让人看着就不舒服，但是又畏惧于其背后的存在而不敢与争锋。

如何做呢？那就是硬，但是这种硬，很难很难的，背后有很多力量在交锋，势力，权力，财力，领导力，人脉力，有一样扛不住，就完蛋了。但其实，一时成败无所谓，因为此刻的退缩并不意味着结束，积蓄力量之后，总会再次遇见的，到时候再一举击溃就是了。

金刚的目的是为了善，这是很关键的，因为不受控制的力危害更大。但是，善其实也是相对的，所以，缺乏了智慧，仅有力也是不完全的。
