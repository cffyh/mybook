# 物理数学

作者：Changkai Zhang

链接：[https://www.zhihu.com/question/433954281/answer/1676673806](https://www.zhihu.com/question/433954281/answer/1676673806)

这个问题可以说是年年都有人来问，我记得我大约三年前写过一个类似的回答。不过既然三年过去了，我就再过来回答一次。很多刚刚学了微积分形式下的经典力学，或者是微分几何语言下的广义相对论的同学都会有一种感觉，觉得这种基于严格数学的物理学理论优美而又可靠，仿佛这套理论支配下的一切都可以用数学上的一种标准流程给推导出来，进而产生了一种错觉，觉得数学是物理理论的一种基础，没有严格数学基础的物理理论是不可靠的。

然而，严格的数学基础和一套可靠的物理理论之间的关系是非常微妙的。原因在于数学是一个非常宽泛的概念，里面包含分析、代数、几何等等等等。所以这里面的问题在于，数学基础需要严格到什么程度才能支撑一套可靠的物理学理论。

为了说明这一点，我需要自定义两个概念：代数结构与分析构造。举个栗子，积分满足线性性 (linearity)：

∫αf(x)+βg(x)dx=α∫f(x)dx+β∫g(x)dx\int \alpha f(x) + \beta g(x),\mathrm dx = \alpha \int f(x),\mathrm dx + \beta\int g(x),\mathrm dx\int \alpha f(x) + \beta g(x),\mathrm dx = \alpha \int f(x),\mathrm dx + \beta\int g(x),\mathrm dx

这是积分的代数结构，除此之外另一个著名的代数结构是分部积分律。而你把积分用 Riemann 积分的定义写出来：

∫f(x)dx=limΔxi→0∑if(xi)Δxi\int f(x),\mathrm dx = \lim_{\Delta x_i\rightarrow 0} \sum_{i} f(x_i)\Delta x_i\int f(x),\mathrm dx = \lim_{\Delta x_i\rightarrow 0} \sum_{i} f(x_i)\Delta x_i

这是积分的分析构造。总的来说，代数结构就是某个数学概念满足的一系列规则，而分析构造则是通过一系列流程去构造一个数学概念。

接下来我要问的问题是，你觉得一个积分的代数结构更基本，还是分析构造更基本？你通常情况下应该会回答我分析构造更基本，对吧？理由很简单，你在定义了 Riemann 积分后，你可以推导出积分的线性性，反过来却不行。是的，这是一个相当充分的的理由。但是，接下来我要告诉你几件事情：

首先，Riemann 积分只是积分的一种分析构造。除此之外你还可以定义 Lebesgue 积分，这是一种完全不同的分析构造，然而，它还是满足线性性。不仅如此，如果你愿意，你还可以把积分看成 de Rham 复形上的上同调。甚至，在广义函数理论里，广义函数的积分压根就没有分析构造。然而，在这所有不同的情形中，积分都满足同一种代数结构：线性性。

现在，你再倒回头思考一下，是代数结构更基本，还是分析构造更基本？

你会发现这样一个事实，就是不管你用哪一种定义，真正决定一个数学运算能不能被称为积分的，不是它的分析构造，而是它的代数结构。这一点在广义函数理论里面体现的淋漓尽致。我们在定义广义函数的积分时，唯一的要求就是广义函数的积分需要满足积分的线性性和分部积分律。我们甚至通过分部积分律来定义 delta 函数的导数。

所以这就是我要讲的很重要的一个观点：代数结构比分析构造更基本。而真正给予了物理理论自洽性和可靠性的其实只是数学理论中的代数结构而已。你仔细回想一下你在做物理中的问题时，你真的有关心过把函数分段求值然后取极限这种事吗？没有，你只是根据一系列积分的代数性质去做了计算而已。诚然，一套完整的分析构造是必然带来自洽的代数结构，但反过来却不一定，一套自洽的代数结构未必一定有分析构造。一个典型的例子就是量子场论。量子场论迄今为止也没有发展出一套令人满意的分析构造。不过这不影响它成为高能粒子物理的理论基础，并获得了巨大的成功。

物理理论只关心代数结构这件事其实并不难理解。从目的上讲，物理理论需要做到的无非就是在给定条件的情况下输出可观测量。要做到这一点，你只需要一套规则，把输入的条件根据规则转化为输出的可观测量即可。至于这规则背后从数学上是如何构造的，不重要。有当然好，没有也不是什么大不了的事，你唯一需要保证的是这套规则是自洽的即可。

这是理论物理与数学的一个重要差别。对于数学而言，我们有一套公理，所有的结论都是根据公理推导而成。而对于理论物理，我们有的则是一个 Lagrangian 或者 Hamiltonian，通过 Lagrange 方程 (经典) 或者路径积分 (量子) 的方式获得可观测量。我们不关心究竟应该怎样去给某个 Lagrangian 去找分析构造。在理论物理中，数学是服务者的角色。这个 Lagrangian 才是物理学的“公理”。如果你哪天发现传统的分析构造会 fail 掉你的 Lagrangian，那么不是你要做的不是去放弃这个 Lagrangian，而是命令数学家重新搞出一套分析构造让它能 work。因为只有这个 Lagrangian 里才有物理，它的分析构造只是数学。

因此如果你只关心物理的话，你唯一需要关注的就是代数结构。事实上，你以后还会发现许多物理书里用一些看上去明显不严格的表达方式。你会发现物理学家想分部积分就分部积分，想交换极限与积分的顺序就交换，完全不管会不会出事，只要物理上看上去合理。这不是什么不专业的行为。恰恰相反，这是很多理论物理学家的必备技能。如果数学家对此提出异议，你应该叫他们想办法重新搞一套理论出来让这些疯狂的操作变得合理。

记住，数学只是我们用来描述物理的语言而已。

2025年博士即将毕业之际重新回过头来看以前的回答。

博士六年期间选了9门数学系博士生的课（因为老密会开很多专题课），对这个问题有了更深层次的理解。

做理论物理多了解一些数学结构是好事。

但是物理人学数学最重要的不在于了解每个定理证明的细节（我现在[反函数存在定理](https://zhida.zhihu.com/search?content_id=331924988&content_type=Answer&match_order=1&q=%E5%8F%8D%E5%87%BD%E6%95%B0%E5%AD%98%E5%9C%A8%E5%AE%9A%E7%90%86&zhida_source=entity)怎么证明都不知道，但是学[微分流形](https://zhida.zhihu.com/search?content_id=331924988&content_type=Answer&match_order=1&q=%E5%BE%AE%E5%88%86%E6%B5%81%E5%BD%A2&zhida_source=entity)的时候这个定理用了不下二十次），而是你知道每个定理里的每个前提条件是什么意思，结论是什么意思，这个定理本质上阐述了怎么一回事，这个定理该怎么用，别人是为什么要建立这么一个定理。而这些理解，是大部分人读一遍书学不到的，而往往读到第三遍时才能领悟到其中的真谛。当然还有个捷径，就是office hour的时候疯狂向老师问问题，然后多跟懂的人交流。

我举个例子，我2019年读本科的时候在复旦上过[群论](https://zhida.zhihu.com/search?content_id=331924988&content_type=Answer&match_order=1&q=%E7%BE%A4%E8%AE%BA&zhida_source=entity)，甚至还拿了A，但是那个时候群论里的技巧和定理如同天神下凡，让我摸不着头脑。

后来我读了Serge Lang GTM 211的相关章节（不建议阅读）以及Serre的GTM 42（同样，谨慎选择），才意识到物理学用的群论，本质上就是为了把对大系统的研究，利用对称群，约化到对小的系统的研究。而物理学中用到的有限群里的技巧，核心只有4个。第一，群上取平均，这个技巧是用来构造不变量的。第二，[舒尔引理](https://zhida.zhihu.com/search?content_id=331924988&content_type=Answer&match_order=1&q=%E8%88%92%E5%B0%94%E5%BC%95%E7%90%86&zhida_source=entity)，它是告诉你不可约表示之间有什么关系的。第三，[正则表示](https://zhida.zhihu.com/search?content_id=331924988&content_type=Answer&match_order=1&q=%E6%AD%A3%E5%88%99%E8%A1%A8%E7%A4%BA&zhida_source=entity)/群环，它告诉你一个有限群的最自然的非平凡表示应该长什么样。第四，[特征标](https://zhida.zhihu.com/search?content_id=331924988&content_type=Answer&match_order=1&q=%E7%89%B9%E5%BE%81%E6%A0%87&zhida_source=entity)，他可以用来刻画不可约表示。了解了上面的这个核心观点和下面的四个技巧之后，我顺利的发了第一篇利用群论建立双曲晶格上的[布洛赫定理](https://zhida.zhihu.com/search?content_id=331924988&content_type=Answer&match_order=1&q=%E5%B8%83%E6%B4%9B%E8%B5%AB%E5%AE%9A%E7%90%86&zhida_source=entity)的文章。

总而言之，物理人学数学，不应该是追求百分之百的严谨（书上说的定理你要信），更重要的在于对数学概念以及结构的深刻理解：对某类数学知识或者技巧能干什么不能干什么做到心里有数。从我个人经历来看，这些更深层次更高观点的理解对做物理科研是有帮助的。

以下是2021年的回答

追求严谨和数学知识也不是一件坏事吧，如果你以后想做理论，那数学知识肯定是知道的越多越好。

在这里我想举两个例子，第一个例子是我本科学习数学的例子。当时复旦自科要求的必修课是高数，但是我听了第一节课就果断退课了，因为我感觉高数里很多东西讲的都模模糊糊，不能让我满意，然后就跑去数学系上数学分析。当然第一个学期学数学分析确实挺痛苦的，因为柯西和魏而斯特拉斯他们搞的那一套epsilon-delta语言对于初学者来说用起来的确很麻烦。但是后面学到[黎曼积分](https://zhida.zhihu.com/search?content_id=331924988&content_type=Answer&match_order=1&q=%E9%BB%8E%E6%9B%BC%E7%A7%AF%E5%88%86&zhida_source=entity)什么的，数学上的图像就建立起来了，做题的时候慢慢开始学会先建立大的图像，然后再用精细的语言去把论证过程写清楚。你习惯了这一套严格思考的方法之后，才有资格说建立起数学的直觉。当然这一套方法对我学物理确实起到过负面作用，比如说在学之后的高阶课程的时候，能明显发现老师或者教科书在骗你，或者说发现他们逻辑的漏洞，这的确是一件痛苦的事，但是当你把这些漏洞填上之后（也许要过上几个月甚至一两年），你的物理图像就会比别人清晰很多。当然夹带私货的说两句分析类的课程对物理来说（除了复分析）用处好像不是特别大，如果对数学有兴趣几何和代数类的课程会更有帮助，这些东西更像是给物理理论提供了骨架，而且（学过数学分析）学这些东西的时候你大概率不会有那种强迫自己变严谨的感受了。

第二个是这个学期学[广相](https://zhida.zhihu.com/search?content_id=331924988&content_type=Answer&match_order=1&q=%E5%B9%BF%E7%9B%B8&zhida_source=entity)的例子。我是想不明白不懂[黎曼几何](https://zhida.zhihu.com/search?content_id=331924988&content_type=Answer&match_order=1&q=%E9%BB%8E%E6%9B%BC%E5%87%A0%E4%BD%95&zhida_source=entity)的人能对广相产生什么物理直觉，按那些入门的教科书学一个学期的话无非是会算一些黑洞引力波的东西，但是你对理论理解的深度肯定是不够的，毕竟爱因斯坦创立广相的时候是懂黎曼几何的。

最后提一句物理直觉这个东西很虚无缥缈，就我浅薄的理解来看物理直觉的建立有两种方式，一个是你对一个现象/概念反复推敲带来理解深度上的提升进而对这个现象/概念有了直觉；另外一个就是实验现象见得多了积累下来的直觉。（当然还有少数天才一生下来就可以把自己想象成一个电子，这是例外）对于第一种建立图像的方式，数学上的严谨是有帮助的，因为这会让你在反复推敲自己的论证过程中加深理解。

以[凝聚态实验物理](https://zhida.zhihu.com/search?content_id=335601175&content_type=Answer&match_order=1&q=%E5%87%9D%E8%81%9A%E6%80%81%E5%AE%9E%E9%AA%8C%E7%89%A9%E7%90%86&zhida_source=entity)在读phd的视角答一下题:

喜欢数学很正常，我大一的时候也不满足于高数跟线性代数，也花过时间去看[数学分析](https://zhida.zhihu.com/search?content_id=335601175&content_type=Answer&match_order=1&q=%E6%95%B0%E5%AD%A6%E5%88%86%E6%9E%90&zhida_source=entity)，觉得室友在那看广相，什么[微分流形](https://zhida.zhihu.com/search?content_id=335601175&content_type=Answer&match_order=1&q=%E5%BE%AE%E5%88%86%E6%B5%81%E5%BD%A2&zhida_source=entity)看起来贼炫酷。

这个病治好也挺简单，你要意识到物理世界的宽阔。数学某种程度上作为物理学的语言，打好基础固然重要，但是对于物理的学习来说，够用即可。我在学完数理方法之后，就屁颠屁颠去刷四大力学了，这多有趣啊。当然后面我进了一个做[分数量子霍尔](https://zhida.zhihu.com/search?content_id=335601175&content_type=Answer&match_order=1&q=%E5%88%86%E6%95%B0%E9%87%8F%E5%AD%90%E9%9C%8D%E5%B0%94&zhida_source=entity)的组开始对凝聚态感兴趣，于是[多体物理](https://zhida.zhihu.com/search?content_id=335601175&content_type=Answer&match_order=1&q=%E5%A4%9A%E4%BD%93%E7%89%A9%E7%90%86&zhida_source=entity)走起，[固体理论](https://zhida.zhihu.com/search?content_id=335601175&content_type=Answer&match_order=1&q=%E5%9B%BA%E4%BD%93%E7%90%86%E8%AE%BA&zhida_source=entity)走起，[群论](https://zhida.zhihu.com/search?content_id=335601175&content_type=Answer&match_order=1&q=%E7%BE%A4%E8%AE%BA&zhida_source=entity)走起。

另一种可能性是，数学的严谨本身吸引了你，那需要说明的是，物理区别于数学的最大特点是实验学科。理论再天花乱坠，没有实验的验证只能是空想。物理的严谨不一定体现在他的理论框架上(个人觉得学好物理意味着你能正确的支配"近似")，而是体现在理论是否正确的解释了实验观测，又是否，实验观测到了理论预言的现象。所以，可以想想，你喜欢哪种严谨。

最后加点私货。个人觉得要学好物理，数学很重要。但是在follow完那些推导步骤，真的融会贯通之后，数学细节其实不一定记得真切。所谓掌握物理图像，就是数学计算其实记不得了(大雾)

严谨的科研当然需要严谨的数学计算。但是，决定你想project，决定下一步怎么做的时候，很多时候还是物理直觉在起作用。
