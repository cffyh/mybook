# [学习哲学的前提条件是什么？](https://www.zhihu.com/question/442222174/answer/1753426525)



我的认知过程是从数学物理，到西方古典哲学，最后回归到中国思想。

为什么会形成这样一个“玩家”？

如果让我自己总结一个原因的话，我想应该是一种“敏感”，这种敏感使得我可以从很多事物中找到乐趣。

从数理化到文史哲再到体音美，甚至连刷盘子、贴发票这种活都能让我乐在其中。



医学，武术，饮食都受到哲学的影响，虽然是潜移默化的。为什么存在潜移默化，因为传承演化和层次规律是普遍存在。

- 人类所有活动都潜移默化受哲学影响。人与世界，区分开来，人要认识世界，改造世界，而人本来就是世界的部分，人拥有了意志才有了区别。
- 作为专门学科，其研究有高峰有低谷，内容和方式不断变化。当前我们到了一个高峰，不需要神秘性和宗教，就能对世界有一个完整的解释。 进化论，系统论，控制论，马克思将实践和社会引入哲学。。。 “古典哲学黑格尔已经终结了，现代哲学已经被福柯终结了，现在是后现代哲学时代，啥叫后现代，就是没有支点的时代，哲学不再是它自己，它已经不存在任何的起点，它被分散到政治，经济，逻辑，伦理各个方面，但是它却不再是他自己”
- 哲学和其他基础学科一样，虽然其任务没有全部完成，但已经到了无处不在而又好像停滞了的阶段。大众化，应用化非常有必要。

学习哲学的前提条件是什么？

是科学。

科学是理性加实证，这是人类目前为止认识世界的最优途径，没有之一。

那为什么还要学哲学呢，学科学不就完了？

因为科学能解决的问题实在太少了，少的可怜且增长缓慢，满足不了我们的好奇心，满足不了我们对美的渴望。

学习科学的前提条件是什么？

是数学。

数学是科学的语言，语言不通，别人说什么都看不懂，怎么学呢？

学习数学的前提条件是什么？

是形式逻辑。

也就是基于概念的演绎，不能违反[排中律](https://www.zhihu.com/search?q=%E6%8E%92%E4%B8%AD%E5%BE%8B&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1753426525%7D)、同一律、矛盾律，这是数学这门语言的基本语法。

学习[形式逻辑](https://www.zhihu.com/search?q=%E5%BD%A2%E5%BC%8F%E9%80%BB%E8%BE%91&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1753426525%7D)的前提条件是什么？

是自然语言。

形式逻辑是自然语言的高度抽象，是自然语言表达的严谨形式。

学习自然语言的前提条件是什么？

是母语。

一个中国人连中国话都说不利索，很难说他能够清晰的理解概念的定义。概念都不清楚，何谈逻辑呢？

学习母语的前提条件是什么？

是缄口。

不都说学语言要多说吗？怎么到你这变成[缄口](https://www.zhihu.com/search?q=%E7%BC%84%E5%8F%A3&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1753426525%7D)了呢？

学外语要多说，母语则不用，因为我们就在语境里，说的已经够多了，这叫习得，除非流落荒岛几年，否则是忘不了的。

反而是要谨言，少去与人争辩，努力发现自己的问题，然后落实到“如何做”。而不是找别人的问题，大家的概念定义不同、讨论的目的不同，最后只能是[鸡同鸭讲](https://www.zhihu.com/search?q=%E9%B8%A1%E5%90%8C%E9%B8%AD%E8%AE%B2&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1753426525%7D)。

咱们什么时候见到辩论能辩论出一个解决方案的？

辩论不但是对自身理性的摧残，更是对人类理性的侮辱。

谨言才能慎行，然后才轮得到[修齐治平](https://www.zhihu.com/search?q=%E4%BF%AE%E9%BD%90%E6%B2%BB%E5%B9%B3&search_source=Entity&hybrid_search_source=Entity&hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A1753426525%7D)吧。
