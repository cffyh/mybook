# 2023年你的数学研究或学习有什么收获和感悟？

作者：半数o阿白

链接：[https://www.zhihu.com/question/629305334/answer/3308894126](https://www.zhihu.com/question/629305334/answer/3308894126)

这一年真是进步惊人，甚至于不可思议。大概是开窍了吧，不过，经过了将近五年时间，也称不上聪明。

年初，关注的是寻找新的数学体系解决非线性数学问题，由此探索了很多从未接触过的领域，数论，图论，环论，模论，然后，提出了图观点的说法，即将数学表示为图形，通过几何直觉提取图形中的典型模式，作为认识的基础，在这样的基础上重新建立数学理论，经过数月实践，这一套方法被总结为[直觉数学理论](https://zhida.zhihu.com/search?content_id=631696590&content_type=Answer&match_order=1&q=%E7%9B%B4%E8%A7%89%E6%95%B0%E5%AD%A6%E7%90%86%E8%AE%BA&zhida_source=entity)，结构即为图形。

随后仿佛开挂一般，原本看不懂的理论变得非常形象，原本理解不了的思想可以通过图形表示，数学结构从未如此清晰的展现在面前，借助于这种视觉手段，我回望整个数学世界，发现所有的数学对象都可以表示为图，而目前几乎所有的数学理论都是建立在自然数结构([良序定理](https://zhida.zhihu.com/search?content_id=631696590&content_type=Answer&match_order=1&q=%E8%89%AF%E5%BA%8F%E5%AE%9A%E7%90%86&zhida_source=entity))与[有向无环图](https://zhida.zhihu.com/search?content_id=631696590&content_type=Answer&match_order=1&q=%E6%9C%89%E5%90%91%E6%97%A0%E7%8E%AF%E5%9B%BE&zhida_source=entity)(有限表示)上的，有环图(无穷非周期)似乎缺乏对应的数学理论，于是在思考很长时间后，突然灵感出现，明白了有向环图就是非线性数学，无穷个圆的叠加，由此给出了非线性数学三部曲，相当得意之作。我自认为明白了数学的本源。

后来，想要寻找未来的数学，于是从纯粹现象出发探究数学对象的基本组织形式，点，点集，序列，多重序列，有向图，多重有向图，由此发现现有的数学理论都依赖于数学对象的表示，对象如何表示，理论如何发展，这就是最一般的数学表示论，表示之后就有运算，点集代数，序列代数，幂集代数，图代数，范畴代数，灵感之下，给出了点，函数，函子，函子态射这样的四阶模型表示理论，开启了计算与表示复杂度的探索，发现有限数学的复杂度超乎想象，最后以完全数学为结束。具体做法为选取数学对象表示形式，穷举所有对象，转化为图形，通过直觉识别关键特征，建立数学理论。给出的观点是，未来的代数学，是以二元运算集，原群，半群，群，双二元运算集，环，模，三二元运算集，幂集代数为基本构造的有限数学。

接着，借由[伽罗华理论](https://zhida.zhihu.com/search?content_id=631696590&content_type=Answer&match_order=1&q=%E4%BC%BD%E7%BD%97%E5%8D%8E%E7%90%86%E8%AE%BA&zhida_source=entity)为引子，我开始尝试把我的认识与已有的数学理论结合，由此开始各种理论的重新建立与对应工作，比如分析学的代数化，比如代数学的图形化，效果相当不错，认识经过实践的检验，总算是站稳了脚跟。

之后，才明白，我所发展出来的方法实际上已经出现，属于[动力系统](https://zhida.zhihu.com/search?content_id=631696590&content_type=Answer&match_order=1&q=%E5%8A%A8%E5%8A%9B%E7%B3%BB%E7%BB%9F&zhida_source=entity)，[组合数学](https://zhida.zhihu.com/search?content_id=631696590&content_type=Answer&match_order=1&q=%E7%BB%84%E5%90%88%E6%95%B0%E5%AD%A6&zhida_source=entity)的内容。看起来，白费力气了，实际上，正是这波澜壮阔的实践与探索打下来深厚的基础，学会一种方法和独立发展出这种方法是完全不同的。

这种差异相当有趣，所以，继续探索数学认知，数学应用，数学理论，时代局限的关系，这就是[数学认知理论](https://zhida.zhihu.com/search?content_id=631696590&content_type=Answer&match_order=1&q=%E6%95%B0%E5%AD%A6%E8%AE%A4%E7%9F%A5%E7%90%86%E8%AE%BA&zhida_source=entity)。

现象→表示→理论→实践→现象

学习数学可以改变一个人的世界观，世界观改变，一切都会变，随着对世界的认识越来越深刻，考察的对象越来越一般，个体的智力会发生显著的跃变。数学天才对于周围现象的认知与普通人完全不同，天赋是一种认知差异。普通人想要改善天赋，极度困难，痛苦折磨，却仍有微小可能通过大量时间与资料的浸润改变思维习惯，学习数学的过程中就是对个人认知的调整，那么假如一个人可以在认知的顶端俯瞰所有的数学现象，这样的人是完全无法想象的存在。由此，我也在假想古代的圣人是否就是如此，具有了远超时代局限的认知，所以才能留下超越时空的真言。虽然他们各有所专，其道理殊途同归。数学认知理论就介绍到此。

最近，大概是学了[代数几何](https://zhida.zhihu.com/search?content_id=631696590&content_type=Answer&match_order=1&q=%E4%BB%A3%E6%95%B0%E5%87%A0%E4%BD%95&zhida_source=entity)的基本语言，明白了层论，函子理论，拓扑理论的基础性与普适性，想要在这种认知层次上回顾一下泛函分析的内容，毕竟，函数理论是下一阶段数学发展的重点。

虽然，中间歪到物理上，认为粒子就是函数，用来理解量子力学的奇异性质，不过，没打算深入，毕竟，现代物理很难通过实践检验，各种新角度重新建立物理学的烂活都不新鲜了。
