# 一、层次、聚合度和颗粒度

它们之间的关系：

- 层次的对象是业务问题，数据无高低层次，而问题有高低层次。
- 详细级别及问题对应的维度，简称层次。
- 问题的答案是数据聚合，聚合就是将明细汇总的过程；聚合度越高，对应的问题层次就越高。
- 聚合**度和颗粒度描述问题“详细级别”的高低，二者**正好相反。

聚合：

- 数据分析的重要工作就是用一个数值描述样本的总体特征，这个由多变少的数据计算过程，被统称为“聚合”。

颗粒度：

- 聚合度越高，颗粒度越低，相反，聚合度越低，颗粒度越高。
- 最高的**聚合只有一个数值；最低的聚合，也就是最高**的颗粒度。
- 数据分析的背后是问题所在的层次（详细级别），计算**的过程就是数据的各种聚合的与解聚。**

# 二、字段

- 根据数据属性可以把字段分为分类和量化两类；
- 根据数据值是否连续又可以把字段分为连续和离散两种。
- 维度描述“问题是什么”，度量描述“答案有多少”

字段的两种特征：离散和连续

- 判断字段内的数据是否有必然的先后关系，是理解连续和离散字段的最简单的方式。
- 抽象方式：看字段内任何两个元素之间是否可以且分为更小的单元。
- 离散字段通常生成标题，而连续字段生成坐标轴。

## 连续：

- 字段是连续的，指字段内的数据有先后顺序。
- 两类连续字段：日期和数字。
- 表达连续数据的最佳方法是把数据放在坐标轴上。
- 有先后顺序和能无限切分，是连续字段的基本特征。

## 离散：

- 表示离散字段，就是一组数据的排列，可以称之为“列表”和“标题”。
