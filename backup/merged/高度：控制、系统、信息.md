# 高度：控制、系统、信息

### 1、从认识世界的角度，系统论是核心，控制是目标，信息是描述。

概念：抽象间隔。**系统是一组紧密耦合的变量。那些边界圈定的范围，就是概念的范围，就是系统边界。**

- 系统这个概念，是极好的抽象，可以用来表达世界的本质
- 世界虽然不等于系统，但系统能描述世界。世界无限，描述有限，我们需要的就是，在无限中寻找有限
    - 例如，如何描述人类？
        - 人的本质是一切社会关系的总和。
        - 从生物角度，人体由八大生命系统组成。
- 系**统的核心是关系**，关系的描述即为结构。

**从系统论的角度，万事万物被抽象成系统。关系为核心，具体形态被略去。**

- 就像物体被抽象成质点，但揭示的是本质规律；
- 从具体中提炼出抽象，看山不是山，看水不是水；
- 最后，从抽象**上升到具体**，看山还是山，看水还是水。

### 2、从改变世界的角度，控制论为核心，系统是对象，信息是手段。

- 矛盾驱动变化，规律是变化过程中重复出现的模式。
- 控制
    - 控制论和进化论不矛盾，而且在迭代这个核心理念上集中体现。
    - 控制论和进化论可以统一，都是描述系统变化的基本规律。

### 3、从生命与世界的关系的角度，学习是生命的本能和主线，信息是中心。

学习和 认识世界、改变世界之间的关系

- 学习是更加具体的认识和改变世界，本质上就是知行合一的。
- 广义的学习是更广泛的概念，认识世界和改变世界主体是人，广义的学习拓展到一般生命体上。
