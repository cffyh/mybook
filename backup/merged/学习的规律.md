# 学习的传统观念

1.学习的传统观念

我们通常认为学习是关于知识获取的过程，一个积累和掌握已知事实和理论的过程。这种观念将知识视为静态的实体，学习则成为对这些实体的简单复制。我们被教导去相信，**掌握知识的标准在于能够准确无误地复述和应用这些知识。然而，这种对知识的理解忽略了知识本身的动态性和复杂性**，以及学习过程中个体理解的主观性。

# 学习知识的四个层次

一般认为，从低到高,A、B -> C -> D。实际还存在 A、B -> D ,特别是对于那些概念认知类的知识。

了解（A）：对所列知识内容有初步的认识，会在有关的问题中进行识别和直接应用。

理解（B）：对所列知识内容有理性的认识，能够解释、举例或变形、推断，并能利用所列的知识解决简单问题。

掌握（C）：对所列知识内容有深刻的理性认识，形成技能，并能利用所列知识解决有关问题。

灵活和综合运用（D）：系统地把握知识的内在联系，并能运用相关知识分析、解决比较综合的问题。

2. 缺乏主体性的学习状态：

当我们将学习简化为对知识内容的复制，我们实际上忽略了学习过程中的主体性。这种学习状态是一种被动的接受，一种没有自我参与的“空杯心态”。我们被鼓励去放空自己，以便接纳更多的知识。但这种态度，虽然表面上看似谦虚，实则可能导致个体失去自我反思和批判性思考的能力，从而变得依赖于外部知识，缺乏内在的判断力。

3. 虚心学习与弱者心态的关联：

长期的虚心学习可能导致个体逐渐失去自信，变得过分依赖他人的观点和答案。这种心态下，个体在遇到问题时，往往首先寻求外部的指导和帮助，而不是尝试自己解决问题。这种依赖性反映了一种没有主体性的弱者心态，个体不再是知识的共同创造者，而是成为了知识的消费者。

# **将网整理为链，这就是最优方法。**

知识体系是一个庞大的网络，各种概念，定理，事件，案例交错复杂，如果没有找到其中的主线，很容易就被压垮掉了，学习这个概念需要以另一个概念为前提，学习另一个概念又要学习第三个概念，结果，不断溯源，会发现自己需要面对一个极长的学习链条，还没有开始学习，就已经有了放弃之心。

那么为什么有人可以快速入门，快速学习呢？因为他们找到了一条最短的学习链条，只需要掌握少数几个概念，就能覆盖大多数知识点，由此事半功倍，学习速度飞快。

那么如何找到这样的链条呢？这就是一个复杂问题了。

有些人是触类旁通，也就是说他首先在其他知识体系中掌握了多种典型知识结构，概念结构，然后通过思维，验证，反思，将这些结构映射到了这一个领域的知识体系中，由此，根据已有的学习经验，可以判断出哪些概念重要，哪些不重要，从而快速找到学习主线。

有些人是实践经验丰富，他们在生活实践中实际体会到了那些概念重要，那些不重要，所以可以直接锁定关键，不受其他概念的干扰，这就是用中学，很多搞研究的人就具有这种才能，知道什么是关键，什么不是关键。

还有人是结识了好的老师和引路人，有了这些人的帮助，也能快速找到重点，看起来就是跟班学习一个月，不如跟着优秀的老师复习两三天。这在各种考试中经常出现。不过问题在于，这些人没有实践经验，只是听别人的道理，所以往往在没有老师的帮助下就完全搞不清楚前进方向了。这就是速成所带来的弊端，想要走出老师的阴影，开辟自己的道路相当困难。正如那句话，学我者生，似我者亡，老师的作用是引路，而不是无微不至的帮助，不经风雨，怎见彩虹。

上面就是典型的几种方法。通常来说第一类人天赋很高，智力超越，难以模仿，第二类人花时间很多，功夫很深，也很难做到，第三类人就是普通人的道路，效果也很显著，用来应付学习是没有问题了，但是想要深入研究，困难会大很多，往往会被自己所学的知识困住，不敢突破，由此基本上没有能力创新，一直处于跟随的地位。

# 世界的本质和思维规律

系统这个概念，是极好的抽象，可以用来表达世界的本质；

        世界的本质是系统状态的转化：系统结构的变化导致系统从一个状态向另一个状态的转化。

        结构是是什么？系统的核心是关系，关系的描述即为结构；

         结构决定状态、状态决定性质、性质决定相互作用，老结构因相互作用而转化为新结构，这在化学科学中集中体现，而这个规律是世界本质规律。

        矛盾（力）驱动变化，规律是变化过程中重复出现的模式；
