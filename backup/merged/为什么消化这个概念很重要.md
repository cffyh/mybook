# 为什么消化这个概念很重要

客观现象：

- 不存在吃啥长啥，即使如器官移植这种hack手段，需要通过抑制异物排斥机制。

消化：通过机械破碎，化学反应转化，将外部食物分解为营养素。

吸收：利用营养素，按照自身身体的需求，转化成自己身体的一部分

学习知识也是如此？

- 需要采用各种手段，将知识分解。然后再通过练习，关联，总结等手段内化。

为什么物质世界的食物过程，和心理世界的学习的知识过程，会如此的相似？

- 因为本质相同，一个系统，利用外部环境的输入来，改变自身系统；
- 储藏食物，发现食物来源，不等于食物的吸收利用；
    - 不要试图通过表面记忆，将食物储藏起来，来代替知识的内化。
- 因为存在了系统，那么就区分了系统自身和外部环境。所以身体自我识别，心理上有自我意识。

记忆与内化

- 记忆对于人脑非常特别，它贯穿于整个学习的过程。所以让人误以为学习就是记忆。
- 当走向另外一个极端，不重视记忆，那就是一场空。
    - 知道，了解，仅仅是作为储备。
    - 不仅得记住，而且要非常精确，还必须经得住逻辑批判性的考验。
        - 需要通过讲述、写作、检查等手段，好奇心很难起作用了。
        - 所以数学的定量思维就很关键了，它彻底保证了精确性。描述 + 计算成为获取和应用知识的最核心手段。
- 所以，记忆的多样化是关键。
    - 模仿，熟练，自动提取规律。是一个持续生长的过程，不是理解与否的问题。
    - 深度参与
    - 多场合回忆
    - 。。。
    - 分析、理解、记忆是一个综合性的过程和现象。
