# 关于笔记

链接：[https://www.zhihu.com/question/33971405/answer/992929411](https://www.zhihu.com/question/33971405/answer/992929411)



本质原因：只要理解了基础定义和推导原理，其实并不需要记忆什么东西，开局一个定义，剩下的全靠推。

学习最关键的是理解基本概念和掌握使用规律，而不是死记硬背各种公式定理和花里胡哨的小套路。

不论是解决纸面上的题目还是工作生活中的实际问题，拿到问题的第一反应就是找到与之对应的基本概念，再根据具体问题选择适当的方法。绝大多数简单和中等难度的问题往往都是一眼就能看出来需要使用的知识范围和具体方法，自然没有什么需要勾勾画画的地方。只有很难的题目，需要做很复杂的变型，或者对考察的内容隐藏得很深，才需要尝试几种具体的方法找到适用的思路。

学习数理科目的关键是理解概念和把握规律，第一步是理解，也就是弄懂“是什么”“什么样”“为什么”，都是很本质的东西，这里面解释说明性质的逻辑链条非常重要，最好的办法是跟着老师的思路走，只要脑筋转过弯，当场理解了，基本上就不会忘了。

做笔记通常速度有限，只能记个关键词和框架，不能把老师的原话一句不漏地记下来，少了很多有助于理解的信息。因此笔记只有在已经大致理解的基础上进行巩固强化、查漏补缺上比较有用，作为从零开始学习的参考帮助不大，不如上课录音回家去听。

学霸普遍上课前做好了预习，对课程内容基本上懂得差不多了，课上精力也很聚焦，老师的思路完全能跟上，上课等于复习，所以通常不做笔记，偶尔听到没想到的部分可能会标记下作为重点。

尤其是高中的数学和物理化学等理科，学透了之后，每门课程的本质就是就是不到十个公式和定理，一页纸就能写完，真的没什么可背的，重要的还是**对概念的理解和规律的运用，写一百页也写不完**。

为什么学霸课上都能理解，能跟上老师思路，普通同学就不能确保呢？关键在于学霸提前做好了预习。

不是课前翻翻书那种预习，而是按照**全靠自学搞定一切的目的自习，顺带把课后题配套习题册都刷掉的那种。**

上课对于他们来说不是学完全陌生的新知识，而是对已学过内容进行查漏补缺和巩固强化。

我个人不光不做笔记，错题本也没有。不是不做错题，而是觉得没意义。

做笔记、做错题本的最终目的都是为了让自己理解知识、会解决问题，只要达到这个目的，形式上按照自己效率高的来即可。

虽然我不做笔记，但是**我会自己从零开始推导各种公式定理、画各种非常规的元素周期表，把课本知识和现实生活、游戏内容进行YY和应用，瞎编个课程大**纲

错题如果不做会就一直随身带着有空就想想，想明白了找个垃圾箱扔进去。

总之，学习要入脑入心，形式上怎么有效怎么来，很多学霸要么懒要么珍惜时间，就不太搞形式主义的笔记了。

---

我认识的学霸真的很多，特别是大学文科系，不做笔记基本上划水的节奏。

但是，我也算是一枚学霸了，我是真的不爱做笔记。

不爱做笔记到什么程度呢，整个初中，我没有一个笔记本。整个高中，数理化三科，我有一个小本子，专门用来记非常规的公式，另一个小本子，专门用来记我觉得很刺激的句子。这两个本子一共就几十页，高中毕业我都没写一半。整个大学，我文理科都混，学物理没有做过笔记，念哲学，也同样没有笔记，一个字都没有写过。这十多年的求学生活中，我并不是懒，因为我一直坚持写日记，几十个日记本一路写来，直到最后开始写小说停下来时，已经25岁多了，至少写了15年的日记。这些年，以及今后的几十年，我也看了很多很多各类的书，其中集中时间和精力看过房地产类的书（更写过几百万字相关文章），以及投资股票类的很多书，也没有写过任何一本读书笔记，而且，我还有一个非常令人不解的习惯，就是特别讨厌在书上做记号，谁借我的书，画一行我都会骂人家。我借图书馆的书，只要有划线我就特别不舒服。我所有的书，从来都不划重点，而且不喜欢书签和折页，总之，我的书，看完和新的一样。

以上，确实有点与众不同。

请让我来解释一下为什么，有什么好处，以及问题在哪儿。

高中之前不做笔记，主要原因是老师讲的东西，我基本上都知道了，或者根本就不感兴趣。大多数上课时间，我都是在睡觉，间或清醒时，就是看其他书。我一天的作息时间，基本上是下午开始踢球，一直干到晚上，晚自习写会作业，或者看书。到宿舍开始聊天，一般在聊到半夜，实在没人聊，我自己看会书，看着大家睡着了，和猪一样香，我一个人看清醒，是一件很享受的事。然后上课补觉，一般上午的课全睡过去，午饭后回宿舍，打牌下棋为主，半智力半体力活动。下午的课，看书或睡觉，看心情。这样持续了几年，以至于大家都说是装睡的。

其实，我也知道不对，但是，只要我一认真听课，就必然会睡着。这和我晚上聊天无关。我一认真听无聊的东西，睡得特别快，特别快！举一个例子，很多年后，我策划中央二台的一个节目，带着摄像机和编导去采放某央企，集团最高层会议啊，我一边盯着录像，一边听着领导发言，然后就睡着了，居然发出了全场都听得见的轻轻鼾声，会议因此中断了片刻。同样的事还发生在很多不应该的地方，比如家长会，坐在孩子座位上听班主任老师讲话，一共才半个多小时，我能睡过去。

这样的人，是无法做课堂笔记的。所以，我慢慢养成了不做笔记的习惯。精要就是，只听我感兴趣的东西，一旦不感兴趣，或者听不进去，立刻关闭大脑。

好了，不吹牛了。有些学霸人家是真的不做笔记，不睡觉也不做那种。

其中的好处，我归纳了一下，和大家分享。

首先是三个前提要了解。

一，你必须提前了解要听的东西。比如家长会，如果我孩子成绩差，我肯定睡不着。

二，你必须提前了解要听的人物。比如集团高层会，如果是我所在的集团，我肯定睡不着。但尽管高官牛逼到说出来吓死人，但是，和央视无关就和我无关，有央视有关也和我无关（我是外聘的）。我是来捧臭脚的，只关心拿多少银子。如果提前告诉我，甲方不高兴了，合同做不下去，没银子，我肯定也睡不着。实际上，该集团负责此事的下级公司老总后来看过我的片子，赞誉有加，乐呵呵说：“董事长下来还问，那个睡觉的家伙是谁？我说是请来宣传的高人，他们不信，这回有话说了。”

三，你必须提前知道不做笔记的后果。比如这也不好，那也不灵。但如果考试都拿第一，或者直接就考满分的人，做不做笔记，又有什么区别呢？

好处至少有三个。

一，听课效率高，听课如战场，你单兵做战，轻装突击，而做笔记的人都是组团的，而且这里面还有后援团，带着机关和家属，反应能快？战斗力能强？

二，记忆力好，越来越好。不做笔记的人，一般记忆力都比较好，比如我曾经为了秀记忆力，记圆周率玩，记100位不算什么，而是比，谁记得快，不服，又比谁记得久，我能够记一学期，随时随地被他们考。但听课不做笔记，其实就是一直在训练你的记忆力。长期坚持下来，非同凡响啊。

三，全局观，大局观，以及体系化。每个听到的知识点，如果是单兵，很容易就阵亡了（遗忘），所以，必须编队，分组，建立三三制的体系。**一边听，一边想，一边编，一边扩展，大概是这样的过程。**这其中，很容易把科目的相关知识打通了，甚至能够打通各科目。所以，我认为，不做笔记的人，一定很容易了解“结构主义艹"在讲些什么。

这三个好处，都直接反应在考试成绩中。你会发现，平时，你的时间很多，只要你愿意学习，一定比其他同学要多看很多书，课外拓展的广度和深度，是做笔记的人无法跟上的，这会导致你的**全局观更好，记忆力更佳，以及反应力更快**，是一个良性循环。其次，阅读习惯也会发生巨大改变，泛读速度超快，很容易就学会一目十行，跳段强行，断章取义，偷梁换柱等阅读手法。这些都是其他人不可想像的。再者，会发现考前复习是一件新鲜的事，简单重读一遍课本，就新鲜（原来要么没看，要么忘记了），所以，复习并不枯燥，备考反而兴奋，成绩不差。

当然，真正的妙处我别有体会。

其一，你始终保持心理优势，而且有时很神秘，其他人不知道你是如何学习的。怎么想都想不通，就只能认定你很有天分了。天分这东西，如果自己信，会害死人的，如果别人信而自己清楚是怎么回事，就很好玩了。你沉浸在别人觉得你是天才的环境中，自动的排除掉不必要的竞争和麻烦，学习和生活都变得简单，比如根本不需要操心人际关系。同时，你会得到很多学习上的助力，比如别人问你题，给你更好更多的书，老师和学校给你更多的机会等。总之，笔记是一个道具，一个假面，拿上手，带上它，是累赘。放弃它，赤裸裸的简单生活，其实可能是一种装B，绝大多人都会佩服你，仰视你，何乐不为？当然，我当年不做笔记并不是装B，那时不仅没这个词，也好像不存在这种行为。但我长期上课睡觉这做装法，全市的学生都知道，时不常都有人到学校来打听我，看看我长什么样，这种心理暗示，会让我找不到台阶下来的。

其二，当你养成这种习惯时，就会滋生很多特殊能力，比如短时记忆很强，我记别人名字或电话，从来不需要对方说第二遍。在座机时代时，我记别人电话，转身离开走到自己办公桌，翻出本子，找出笔，边念边记下来时，旁边的人都惊呆了。这样初次见面时，会让人有特别强的印象，应该是好印象吧，他们会感觉这个人很厉害，很精明。

比如总结力很强，随便**听领导讲话，报告会，讲座，自动产生分层逻辑，会后转述，一二三，第一部分下面再分123，第一条下面有两个重点。有些是报告者本身的，有些是**我解释的，会让其他人产生一种幻觉，觉得你比领导水平都高。结果就是，领导自动地会拿着讲稿来让你修改。我原来常吹牛，哲学系毕业，没吃秘书这碗饭，亏了。

但我吃另一碗面，比如写人物专访，那就相当厉害了。我采访时很牛逼，一不录音，二不做笔记，三不提问，大人物坐我面前，会被我带着讲几个小时，全程进入他的回忆录。再不会讲的都控制不住。而且，他们往往什么都敢讲，越讲越嗨。完后写出稿来，绝对高大上，层次分明，逻辑很强，观点较新，创意满满。细想下来，这不是念哲学的结果，是上课从不做笔记的结果。

我一录音一做笔记，对方就不敢讲了，鲜活的素材就没有了。再如观察力很强，你一心听课，听报告，眼神都不曾离开讲者，当然信息量要比低头写笔记的人多，长时间如此，**观察力就上来了，很容易就听出玄外之音，看到更多的表情和动作语言。**所以，一般的骗子，肯定骗不到我这样的人。而且，和人做生意，谈判，或者简单和陌生人交流，我都喜欢盯着人家看，眼睛从来不偏斜，这样，容易识别坏人，也容易得到好感。当然还有很多其它能力，不一一举例了。

# 其三

其三，重点来了。很多讲课者，报告者的水平都很一般。所以，听讲分心者（不做笔记肯定分心，也有时间和精力分心），就会自然挑人家的毛病。大意就是，这个很简单呀，我懂；这个不需要讲了，已经讲太多次了；这还要强调，把我当白痴；这个真的是白痴，用不着这么废话，好不好。更多的时候是这样：这个好像他讲错了吧，我看过其他人的书，不是这个提法；这个人已经不错了，水平高可惜境界低了点；这个境界高，但是基本功和细节都不如我啊；这个好像在专业方面是大拿，很让人佩服，但是形象是不是太差了，还有口音….自动就养成了杠精体质。

在知乎上，杠精是令人讨厌的。在现实生活中，杠精也是让人讨厌的。但讨厌杠精的人，是被抬杠者，并不影响杠精的学习和进步。杠好了，就成了独立思考者，杠升级了，就成了批判者。杠成苏格拉底+或鲁迅是很有难度的，但杠得老师都有点怕你，难度系数就下来了。话说，为了杠老师，你没点水平，不考个班级第一，年级第一的，敢随便张口？

当这种能力进化成调侃之后，就更厉害了。

记得大学时在中文系宿舍看《围城》电视剧，一屋子人挤着，对着一个20寸黑白电视机，开讲。每一个人出场，每一段台词，每一个桥段，都有人杠。特别能杠的人LJJ同学，能全程编着段子走，大家都特别开心，学到很多知识啊。当然，这里的前提是，每一个人都熟读《围城》了。从此，我看再无聊的电视，都能找到兴奋点，可以全程吐槽着看下来。

补充一句，LJJ同学后来和我一起教了一段时间书，他的语文课，能把当年北京市排名前五重点高中的学生讲得晕头转向，全程参与度很高，绝对的人才啊。这个聪明的家伙，还特别能写，曾经抱着几大箱的“作品”给我看，全是他自高中时写给一个人的情书。我还记得他抚摸着情书的样子，他一边低头摸着，一边自言自语地说：“你说我容易吗？结个婚我太不容易了啊。”

这样可爱的家伙，大概是不爱做笔记的人，他应该没有那个时间和精力啊。



不做笔记的坏处就是，有些东西你记不住，就再也找不回来了。比如开头那个原理，比如我究竟赔过多少钱，比如我的一生中，究竟错过多少机会⋯

又比如我第一封情书究竟什么时候写的，具体写了些什么？再强大的记忆力，这么多年过去，也只能记得是写给谁的了。

所以，对于某个具体的学霸，明明做笔记可以考得更好，生活得更有味道，为什么他就是不爱做笔记呢？

- 除了他真的很懒以外（其实大多数时装的），他就是一直在强调自己的天分，并不愿意放弃，好像一写笔记，他就失去了魔法，变成一个麻瓜了。
- 这样的人，有一天意识自己真的很平庸时，会很伤心很伤心很伤心的。
- 伤心到想死！

真正的大神，是爱做笔记的，不仅爱做笔记，而且很会做笔记，而且任何时候都很认真的做事，关键的是，他们不觉得自己有天分！他们反而更努力。

你不喜欢做笔记，只是因为你在低阶玩啊，可能最大的目标只是为了高考上个TOP2。

**等你真正进阶的时候，你才知道读书笔记的价值罢。**

而我这样不会做笔记的人，再怎么整，也是野路子，民科民哲民文，三民主义的股民。
