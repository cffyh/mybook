# 生产力、生产关系

生产力 → 技术的本质 → 具体技术领域的规律

生产关系 → 经济基础、上层建筑、社会 → 人际关系

商品→ 产品 → 用户价值 + 商业价值

《资本论》对资本主义经济的研究从篇章结构的总体安排上看是采用了了逻辑的方法,《资本论》正文的第一段话就是

- 从逻辑方法上说明为什么要从商品研究开始,因为商品是资本主义的经济细胞,而商品货币关系则是资本主义的一切经济关系的基础,"研究商品关系确立了劳动价值理论,这是剩余价值的理论基石”。
- 再者,从《资本论》研究方法的角度来说,马克思运用的是从抽象到具体的逻辑方法来揭示资本主义经济运动规律的,是"从抽象到具体的方法将他引领到商品概念"。

马克思的"从抽象到具体"的研究方法就是"把所有这些材料置于严肃的批判之下,以发现能够说明现实运行方式的简单而有力的概念"。
