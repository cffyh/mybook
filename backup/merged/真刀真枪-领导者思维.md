# 真刀真枪-领导者思维

[五层管理者](./五层管理者.md)

什么是真刀真枪？实战，实操，实践，商场搏杀，职场历练。

真刀真枪 = 应战，不逃避。任何一点失误都可能付出血的代价。

- 逃避状态：
    - 既不去和现实作斗争，又不将理论严格化作输出；
    - 麻痹、补偿、夹杂着学习和创造。
- 思维和行动中的真与假，实与虚
    - 大脑高强度运转起来，才能真正改变大脑连接，才是真正的思考。
    - 勇敢：open，接地气 --> 深入具体细节 => 接受现实的考验，才是坚实有力的行动。
- 考验、考评、评价：外部的要求和评价是人和社会交互的具体体现，对个人来说，就是各种压力。直面压力，应战 --> 真刀真枪。

# 为什么真刀真枪，实事求是，大家都知道的道理，但是少有人做到？

- 成功的体验是稀缺的，逃避是人面对无法掌控的事情时候的自然反应。
- 成功的体验 + 成功之道的领略，强者精神走向良性循环。反之，就会恶性循环。
- 真刀真枪 => 了解现实，扬长避短，最佳状态。
    - 现实，长短，偏好，由过去的积累决定，现在又会成为未来的过去，显然这是一个马尔科夫过程。

# 相对稳定的社会结构+竞争机制。所以对个人来说，阶级地位，博弈是社会的核心。

在狩猎时代，人除了直接的生存危机外，是有充分的自由的。

在工业时代，压抑自我的状态是最频繁的。

人作为社会大生产里的一颗螺丝钉，就得压抑自己，阶级压迫是阶级社会的必然。

正视 社会的本质和发展现状。人类通过社会化的分工合作，技术进步，获得更强的改造自然的力量，从而让自己能够适应自然。如何分工合作，生产资料如何分配？

- 更极端一点，将社会化的人生看成是一场场的斗争、战斗。
- 现实主义者，以达到目的（获胜）为导向，将现实中的事物明确的区分为有利、不利、无关的因素。
- 竞争的发生不以主观意志为转移，由利益的冲突决定。
- 生产关系不改变生产力本身，但能影响生产力的发挥。

# 斗争的必要性

- 只要能做到持续高强度用脑，心流状态，无须一场场的战斗，就可以有所成就。但没有宽松的环境，就做不到持续。
- 每个人被社会分工裹挟。不占据有利地位，就会被淘汰。世界需要更强的多样性、容错性。
- 对于探索未知、拓展人类认知边界的科学家，艺术家，一般不太理睬现实的社会。但如果他们对于社会的残酷本质有认知和正视，就能提高自己的存活概率。
- 社会给与宽松的生存环境，就能够让天才性格类型的人发挥出他们的才能。

# 集中精力和资源办好最关键的一件事，你就赢了。

集中精力和资源办好最关键的一件事，你就赢了。

一场场的战斗或斗争，每一时刻，只有一个重点，但时间拉长，重点总是不断的在变换。

而这些战斗和斗争，背后得有战略来统筹，所以人生是调动注意力的艺术。

- 人一辈子能亲身参与的战斗寥寥无己，能对世界做出的改变极有限，而且还是转瞬即逝的。
- 天才期望留下更永恒的影响，所以天生不喜不睬现实世界。马克思的理论著作，也不是仅仅满足于当时斗争需求的。
