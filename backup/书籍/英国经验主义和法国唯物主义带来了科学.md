# 英国经验主义和法国唯物主义带来了科学

唯理派认为，“经验”是天赋的、内禀的感性活动。
这是由于，那些研究唯理派的经院哲学家们，可能已经有了足够多的经验，才说这是天赋的，不言自明的。于是，他们对知识的态度，就总是演绎、整理、加工，然后形成复杂命题，甚至思考许多形而上的命题。

而经验派则历来认为，“那个物质是什么”才是重要的。

我们必须要摆脱感觉、直观、表象的经验，去探求那种内在的经验活动，也即我们所说的判断、推理、意识等等一切精神活动。哪怕是宗教活动，也离不开外在的经验。人类所有的神话的神明形象，都是对自身的拟象。看似在说神的故事，并且具有许多神秘的、夸张的、混沌的概念糅杂，但诉说的就是那个民族的故事。

而经过现代人解构后，我们才发现，原来其实各地民族的“人性”都是同一的。只是，那些环境和特定历史时期所引起的偶然表象，总是迷惑了人们。
所以，经验派也认为，唯理派带来了许多“假象”。

好比“洞穴假象”，也就是人与人之间的不同，是基于社会各个环节造成的不同。
用马哲的话来说，也就是所谓的人性是随着人与人之间社会关系的变化而变化的。所以，并没有什么天赋人性，有的只是人脑对客观世界物质运动的主观能动的反映活动。

也包括语言。**唯理派对语言的定义是“逻各斯”，也即通过语言正确地反映现实**。这是对的。但唯理派否认，从外界获得经验，而是想要通过演绎（也就是他们所说的精神运动）来获得真理性知识。古希腊也正因如此，而走向衰落。

我们不可否认，唯理派的一些理论，还是不错的。也即马克思所说的“意识能动的方面反而被唯理派们抽象地发展了”。
康德的范畴论也为当时走向僵化的经验论与唯理论之间起了调和作用。

不过，黑格尔又把这种趋势踢回到了古希腊时代，幸好马克思又把它带回来了，不然德古还真的彻底完蛋。

可是，我们必须要承认一个事实，如果没有英国经验主义，那么也就没有科学。
那就更不可能有马克思了。

因为人家实际上，早年经过了唯理派的研究后，越来越发现，人的精神主观的能动性是要基于物质和环境赋予了感觉经验材料，你才能有了感性经验活动、知性经验活动和理性经验活动。
换句话说，感觉是链接人的外在性与内在性的桥梁。 没有外部的感觉经验、行为经验、事实经验等外部经验，很难说你那点内部经验能起什么作用。

如果不是培根对三段论的批判，人们甚至无法意识到，人们获得知识不是通过推断，而是通过从事实中归纳经验，然后才能进行各种内在的经验活动，从而将其一步步转化了物质的力量。

从物质到感觉-外部经验-意识活动-内部经验-实践经验，人最终实现的反而是对自然物质的操控，而不是某些唯理派所说的“拜物教”。

所以总的来说，物质是什么？
**物质是客观存在。
存在，在康德和黑格尔那里，就已经被定义为“无规定性、无范畴”的自在物。这与马哲里的“不依赖于意识而存在的客观实在是等价表述。**

- 客观存在-表象与主观意志，叔本华认为人类世界由表象和主观意志组成。 
- 客观作用于人→ 表象；
- 天才剥离意志的限制，洞察表象背后的本质。艺术家洞察&表达表象背后的本质。
