# 个人努力是基础，上级是杠杆

当领导才知道，真正决定你前途的，从来都不是你自己的能力，而是你上级的能力。

这事听着特别反常识，但现实就是这样运转的，一个人的职业天花板，在选对上级的那一刻，其实就已经被决定了。

你在基层的时候，拼的是执行力，谁干活快，谁干活好，谁就能脱颖而出，你每天琢磨的是怎么把手头的工作做到极致，怎么在同事中显得更优秀。

可一旦你开始带团队，哪怕只是个管三五个人的小组长，你就会发现，你做得再好，你的团队再拼命，如果你的上级是个庸才，你们整个部门的功劳都会被掩盖。

比如你辛辛苦苦带着团队攻克一个难题，项目大获成功，到了向上汇报的时候，你的上级连你们项目的核心亮点都讲不清楚，三言两语就被更大的领导问得哑口无言，那么你和你的团队所做的一切努力，基本等于零。

大领导不会觉得你上级不行，他只会觉得你们整个部门都不行，你上级汇报得差，等于你干得差，功劳根本穿透不了你上级那层天花板。

反过来，假如你摊上一个能力特别强的上级，情况就完全不同了，他非常清楚怎么向上管理，怎么在关键会议上呈现你们部门的价值，怎么把你们做的七十分包装成一百二十分。

可能你们团队只是完成了一个常规工作，但你上级懂得从公司的战略高度去解读，能把一件小事和公司的大目标联系起来，三两句话就能让大老板眼前一亮，觉得你们部门有思想，有价值，总能干出亮点。

这种情况下，功劳会像下雨一样，均匀地洒在你们部门的每一个人头上，大老板会觉得你上级带队有方，而你作为他得力的下属，自然也会被高看一眼，晋升的机会就来了。

在职场这个系统里，信息是层层向上传递的，功劳也是，你的直属上级就是那个信息和功劳的转换器，他要是性能不行，你输入再多高质量的信息，输出的全是乱码。

很多时候，你觉得工作干得憋屈，不是你不够努力，也不是你的团队不给力，而是你的上级没有能力把你们的价值，有效地传递给更高层级的决策者。

你在下面吭哧吭哧地挖井，以为能挖出水来，可你上级连个水管都没本事接到公司的水利系统里去，你挖得再深，也不过是口枯井。

当了管理者才会深刻体会到，组织的本质就是层级，你所有的工作成果，都必须通过你的上级这个端口，才能被组织看见和承认。

他要是会来事，能搞定他的老板，能把资源要下来，能把功劳报上去，你在他手下就能过得很滋润，哪怕你本身能力只有八十分，他也能给你争取到一百分的回报。

他要是个老实人，或者能力平平，只会埋头干活，不懂向上沟通，那你就算浑身是铁，也打不了几根钉子，你的功劳会被他天然屏蔽掉，你的职业发展也会被他牢牢卡住。

因此，当你有机会选择的时候，判断一个部门值不值得去，判断一个岗位有没有前途，最核心的标准，不是看业务本身，而是看你未来的那个顶头上司，他怎么样。

他在公司里说话有没有分量，他跟大老板的关系怎么样，他在各种会议上是能侃侃而谈，还是唯唯诺诺，这些细节，都直接决定了你未来的工作环境和发展空间。

跟对一个厉害的上级，他吃肉，你能跟着喝汤，甚至他升职了，还会顺手把你提携到他的旧位置上，这条路远比你自己单打独斗要顺畅得多。

这就是职场里最真实，也最无奈的一面，你的个人奋斗固然重要，但它的效果，会被你上级的水平，进行几何倍数的放大或缩小。
