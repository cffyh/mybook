# 逻辑在学习中的体现

在学习中就要注重用逻辑从最基本的一两个概念和原理一步一步推出课本里茫茫多的公式的基本路径，同时也是分析解决各种千奇百怪的题目的主要方法。

高中开始学习的概念和原理是很抽象的，需要理解抽象概念描述的具体事实，清晰地界定抽象概念的边界，掌握抽象概念的多种表现形式，就需要通过解决具体实际的题目来练习。比如“向量”是“有方向”的概念，平面向量是用平面内的有向线段表示的，怎么用有向线段同时表示“方向”和“长度”，以及向量的运算是怎么操作“方向”和“长度”的，需要通过足够的练习来理解。对向量进行坐标化之后，坐标又是怎么同时表示“方向”和“长度”，坐标运算与向量运算的对应关系、坐标运算是怎么表示“方向”和“长度”的，也需要通过足够的练习来理解。
