# “学习数学物理”与“学习化学”的主要区别

“学习数学物理”与“学习化学”的主要区别
学习数学物理的主要逻辑是演绎，也就是所谓的推导，从最根本的定义或原理，一步一步推出一系列结论和公式，这个过程很像一级一级爬梯子，每一步都是后一步的基础，前后有很强的连贯性。
学习化学的主要逻辑是归纳，也就是所谓的找规律，从大量零散的具体事实中找规律，把这些大量事实串起来，并用规律解释和预测新的事实，这个过程有些像整理房间，把一大堆东西按类别数理摆放清楚，有新东西也能知道要放哪里。
学习数学和物理的关键是逻辑性要严密，每一步的分析推导计算都要彻底搞明白，所以学习数学物理的关键是扎实，每一步都要彻底搞明白，不能留有任何盲点，否则很可能导致后面的知识根本看不懂。并且物理的数学化程度很高，很多重要的物理概念跟数学概念是相通甚至一致的，比如位移与向量，速度与变化率等。
学习化学的关键是类比、联想、迁移能力要强，要能将一般性规律迁移应用到不同场景中，能从一些看上去杂乱的事实中提取出规律。学习化学最好要反反复复多学几遍，一方面利用共性规律把大量个性化的零散事实串起来，另一方面利用大量基本事实来理解共性规律，两者相互支持印证。
实际上以上是“学习数学物理”与“学习化学”的主要区别，而不是“数学物理”与“化学”的区别。因为研究物理本身跟研究化学一样需要很强的归纳能力，只是物理理论的抽象程度和系统化程度非常高，所以学习阶段所要掌握的物理知识已经梳理的足够系统和精炼了，可以像学习数学那样学习，但如果是按照“探究”或研究的方法，像古代科学家那样通过实验学习总结物理规律，那么感觉上就跟化学非常相似了，只是直接学习系统理论的效率和性价比更高成本更低。
另外，虽然化学理论的系统化和精炼程度没有数学物理那么高，但实际上也已经相当高了，比如高中化学的几个重要块面，都有非常精炼简洁“一通百通”的基础理论或模型，比如解释元素性质的原子结构（电子排布）规律，解释分子结构的价键理论、价层电子对互斥模型、杂化轨道理论，解释化学反应的能量变化、速率、平衡的有效碰撞理论和过渡态理论，以及充分利用前面这些理论的非常具体形象的有机化学。
