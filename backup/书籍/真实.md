# 真实

尤瓦尔·赫拉利在谈及“真实”时，有以下一些观点：
1.	真相昂贵而稀有：他认为真相是一种非常昂贵且稀有的信息。真实地描述事物需要投入大量的时间、金钱和精力去研究、分析，相比之下，虚构则成本低廉。
2.	真相复杂而痛苦：现实是复杂的，所以真相往往也很复杂，而虚构可以随意简化。同时，真相有时会令人痛苦，而虚构可以按需美化，大多数人更倾向于简单美好的虚构故事。
3.	信息不等于真相：信息的基本功能大多不是揭示真相，而是建立联结，把人群联结起来形成网络。联结大量人的最简单方式往往是通过虚构、联想和群体错觉，而非真相。

真实 vs 真相
核心差异：
“真实” 是 “单点 / 局部的客观符合”，像 “拼图的碎片”；
“真相” 是 “整体 / 本质的客观呈现”，像 “拼完整的拼图”。 所有 “拼图碎片（真实）” 都为 “完整拼图（真相）” 服务，但单一碎片永远不是完整拼图。
逻辑关系：
 承认 “真实” 是基础 → 整合多个 “真实片段” → 挖掘 “深层逻辑” → 最终还原 “真相”。
 简言之：真相是 “完整且本质的真实”，真实是 “真相的组成单元”。
