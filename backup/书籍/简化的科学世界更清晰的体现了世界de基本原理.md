# 简化的科学世界更清晰的体现了世界de基本原理

简化的科学世界

化学的核心逻辑是结构决定性质，性质决定行为。
化学的研究内容是物质的微观成分、结构、变化规律，包括三个关键问题：

1.物质是由什么组成的？简略的答案是原子和分子。

2.原子和分子是怎么组成物质的？简略的答案是化学键和分子间作用力。

3.物质怎么从一种物质变成另一种物质的？简略的答案是原子之间的组合方式发生变化。

高中化学的学习需要记忆掌握非常多的化学元素的原子结构、化学性质、形成的分子、分子的结构、分子的化学反应、化学反应中的能量变化、反应速率、化学平衡、有机化学、化学实验的原理、仪器搭建、实验现象、现象分析等大量内容。这些大量内容不是零散混乱的，而是有很强的逻辑性和规律性，其中一个很重要的逻辑就是结构决定性质，性质决定行为。

#### 物理学家的视角

关注现实意义：物理学家会给所有概念赋予现实世界的意义，像力、质量、惯性等，依靠对现实的直觉和常识去理解现象，比如凭借经验预判物理现象如何发展 。
关注特殊案例：更感兴趣特殊情况，而非通用情形，在已知所讨论对象的实际所指后，利用常识和直觉去分析问题 。

#### 生物

高中生物课本的分册、章节本身就很有层次，从小到大依次是：组成生命的分子、细胞的结构、细胞的行为、生物体的调节（组织器官系统的功能）、遗传学规律（这个比较数学和逻辑）、种群群落生态系统。

- 生物的层次性最大的特点是：**每个层次的现象和规律可以直接解释比它高一层次的现象和规律**，所以实际上只要认真学习最微观的组成生命的分子（实际上就是高中化学的有机化学知识），那么细胞器的化学组成、结构和功能就很容易理解，细胞的行为就也很容易理解，下一个层次生物体的调节（组织器官系统的功能）也很容易理解。
- 遗传学规律主要包括两个方面：一个方面是宏观的遗传学规律，就是孟德尔种豌豆和摩根养果蝇，主要需要一定的数学功底（概率和排列组合）以及逻辑分析能力，本身数学学习概率和排列组合就很需要逻辑分析能力。另一方面是微观的分子层面的DNA在遗传规律中的行为，学好生物大分子的知识后很容易理解。
- 最后的种群群落生态系统是宏观的，以语言描述和概念阐述为主，跟前面的连续性不太强，需要记背很多知识。
- 另外，还有3条贯穿整个生物的主线：物质转化（塑造生物体需要物质）、能量利用（驱动生命活动需要吸收能量、消耗能量）、信息传递（内在生命活动、生物体之间的关系都涉及到信息的交互和对信息做出反应），所有的分子、细胞、生物体、种群群落生态系统这些层面，都涉及到这3条主线，只是高中阶段有的部分表现得很明显，有些部分表现得不太明显。
- 还有生命科学基础（实验），生物实验大都用的是物理和化学知识，跟生物本身的知识关系比较小。
