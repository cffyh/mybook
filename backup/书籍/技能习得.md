# 技能习得

#### 划小圈

和[分形数学](./分形数学.md)中的原理类似，一花一世界，局部可以窥探到整体的规律。

什么是划小圈？

- 学习的基本原则就是要钻研微观的细节事物，来理解宏观上的问题和现象。

有一个例子就是英国象棋大师，他一般下棋的时候都着重在侧方位，而把中盘的控制拱手交给对方，他其实是知道如何控制中盘而又表现的和中盘毫无关系，最后却是他赢，他把圈划得如此之小，就连象棋大师也看不出来。这个小圈的意思是**他把他自己控制在一个比较小的位置上，但是他却通过这么小的一个区域，控制了别人更多的东**西。这就是重视细微的吸收和完善，这样比学到的招式数量多，要重要的多。

作者说他能成为顶尖选手并没有什么秘诀，而是对可能是基本技能的东西有更深的理解，每天都要学得更深一点，而不是更广一点，因为学到更深可以让我们潜力中那些看不到感受不到而又极具创造力的部分挖掘出来。

他的整个学习过程的核心在一定程度上是受到了罗伯特.波席格的**摩托车维护艺术这本**书的启发。主人公文德罗，要帮助一个女生来解放她的思维，这个女生需要写她所**居住的小镇的故事。他让她从教室外面那个剧院的前台左上方的砖开始写**，结果这个学生变得文思泉涌，停都停不下来。

#### 让时间慢下来

比如练习象棋的时候，从一个只有王和兵的空白棋盘开始学习这两个棋子该怎么走，然后不停的练习这两个棋子走法，一直到自己感觉到走得很舒服为止，接着再慢慢地单独学其他的棋子的走法，然后他就很快熟悉了象棋中各个棋子的走法和价值的所在，再不用有意识的去想这些棋子，而是可以在看到棋子的外形的同时，还能想到他们的潜能。
