# 管理“善”和“恶”

在我还没有当领导的时候，我以为管理是一门关于 “善” 的艺术。

我以为当领导的核心工作，是去发现和激发人性的闪光点，是去找到那些眼里有光、内心有火的同路人，然后为他们创造一个能尽情燃烧的环境，我以为管理者最主要要做的，是信任，是赋能，是把一群优秀的人，变成一个伟大的团队。

我读了很多关于谷歌、关于奈飞的书，我相信，只要给足空间，给足尊重，人就会自我驱动，追求卓越。

后来，我真的坐上了那个位置。

然后我才明白，在我真正有机会去激发善之前，必须先完成一项更重要也更基础的工作：**管理恶**。

我所说的恶，不是指什么穷凶极恶的品质，它指的是人性中那些最普遍、最根深蒂固的与组织目标天然相悖的特质：

懒惰，贪婪，自私，嫉妒以及对责任的回避。

我当领导后学到的第一课，也是最颠覆我认知的一课，就是：

任何一个稳定组织的根基，都不是建立在对人性的乐观假设之上，而是建立在对人性弱点的深刻洞察和制度性对冲之上。

管理学上有一个著名的理论，叫 XY 理论，道格拉斯・麦格雷戈把管理者对人性的假设，分成了两种。

- Y 理论，假设人是积极向上的，热爱工作，愿意承担责任，能够自我驱动。这就是我当所谓的领导之前，所信奉的那个理想主义的世界。我以为当领导的工作，就是去找到这些 Y 型人，然后和他们一起创造奇迹。
- 而 X 理论，则假设人是天性懒惰的，厌恶工作，逃避责任，必须靠胡萝卜加大棒，才能被驱使着前进。

我过去一直以为，X 理论是工业时代落后的不人道的遗毒。

- 后来我才明白，X 理论才是这个世界上绝大多数人的默认出厂设置，而 Y 理论，是一种需要极高成本和极特殊环境才能被点燃的、珍稀的奢侈品。

我的首要工作，不是去搭建一个 Y 理论的天堂，而是先建立一个能有效管理 X 理论的地基，然后，在这个地基之上，小心翼翼地为那些极少数的 Y 型人，开辟出一小块可以发光的土壤。

这个道理，其实我们的古人早就想明白了。

法家的集大成者韩非子，早就说过，明主治吏不治民。

一个高明的君主，他管理的重点不是去教化天下百姓，而是去管束他手下的那群官吏，他靠什么管？

不是靠儒家的仁义道德，而是靠冰冷的法、术、势。

- 法，是公开透明的规章制度。
- 术，是驾驭下属、防止其蒙蔽自己的手段。
- 势，是君主手中不容挑战的绝对权威。

这套系统的底层逻辑，就是不信任任何人。它假设，所有的官吏，都有以权谋私的天然冲动，所以必须用一套天罗地网般的制度，去限制他，监督他，考核他。

这套逻辑听起来很暗黑，但它恰恰是构建一个大型稳定组织的唯一可靠路径，因为它不依赖于任何一个人的道德自觉。

当我把这些想通之后，自己过去很多看似温情的管理行为，在我眼里，都有了全新的、也是更真实的解释。

比如，为什么一定要定期搞团建、请大家吃饭？

过去我以为，这是为了增进感情，营造家一样的氛围。

现在我明白，这是成本最低的、用于润滑 X 理论摩擦的手段。一顿饭，几句笑话，确实能暂时抵消掉一部分人因为日复一日的枯燥工作而产生的倦怠和不满，它是一种低成本的非正式的维稳工具。但它绝不能替代那套冰冷的、刚性的考核制度。

比如，为什么一定要强调流程，哪怕那个流程看起来极其繁琐还不近人情？

过去我以为这当然是为了规范，为了公平。

现在我知道了，流程，首先是用来保护管理者自己的。一个清晰的、可追溯的流程，最重要的功能，是在事情搞砸了之后，能够毫无争议地定位到责任人。而我，作为流程的制定者和监督者，只要能证明我已经履行了我的监督职责，那么，至少那个执行环节的错误，就与我无关。

它是一道防火墙，确保任何一个环节的火灾都不会烧到我身上。

所以，我当领导之后，明白的第一个也是最根本的道理就是：

**管理，首先是一场关于风险控制的悲观主义实践。**

我的首要任务，不是带领大家去创造多大的辉煌，而是确保，这艘船在我手上，不会因为人性的幽暗而触礁沉没。

- 我必须先用 X 理论的逻辑，扎好篱笆，堵上所有可能出事的漏洞，然后，我才能奢侈地，用 Y 理论的逻辑，去期待船上能有几位天才的水手，和我一起看到更远的星辰。

这是第一部分的真相，它关于如何构建那个关住猛兽的笼子。

而第二部分的真相，则更为复杂。

当我用 X 理论的逻辑，把那个关住猛兽的笼子，也就是制度和流程搭建起来之后，我才真正开始了作为一个领导，最核心也最耗费心力的工作：

识别、挑选、以及驾驭那些我必须放进笼子里的形态各异的猛兽。

在这里，我当领导后明白的第二个道理，浮现了出来：

**下属的价值，从来就不是一个单一维度的东西，它是一个由能力、忠诚和威胁这三个变量，共同构成的复杂动态函数。**

当我还是员工时，我天真地以为能力是唯一的通行证。

后来我才发现，在一个权力结构中，能力只是最基础甚至不那么重要的一个考量维度。

我手下有这样几类人。

第一类人，能力极强，业绩突出，但桀骜不驯。

他看不起身边所有的同事，也包括我，他遵守规则，但只是因为规则对他有利，一旦规则妨碍到他，他会毫不犹豫地试图绕开，他像一头独狼，战斗力惊人，但也随时可能脱离队伍，甚至反噬。

第二类人，能力平庸，四平八稳，但对我言听计从，忠心耿耿。

我安排的任何工作，他都会不打折扣地执行，我说的每一句话，他都会认真领会，他是我在这个部门里最可靠的耳目和双手。

第三类人，能力和忠诚度，都处于中间状态，他们是团队里的 “大多数”，是沉默的、摇摆的基石。

现在，问题来了，如果有一个重要的晋升机会，我会给谁？

在当领导之前，我的答案，一定是第一个人，因为我相信能者上、庸者下。

而现在，我的答案是，要看情况。

如果，我当前正面临着巨大的外部压力，比如业绩断崖式下跌或者需要攻克一个极其艰难的技术难关，我需要一个人来替我打一场硬仗，力挽狂澜，那么，我会把这个机会，给第一个人。

但我会给他配一个政委，那个政委，就是第二个人。

我会用第一个人的能力，去攻城拔寨，然后用第二个人的忠诚去监视和确保，第一个人的所有行动都在我的掌控范围之内。

而如果，我当前所处的环境相对平稳，内部的人际关系更为复杂和重要，那么，我大概率会把这个机会，给第二个人。

因为，提拔第一个人，对我来说风险太高，他的服从性会变弱，他的能力固然强但可能会转化为功高震主的资本，成为我未来潜在的挑战者。

而提拔第二个人，虽然无法给部门带来惊人的业绩增量，但却能极大地巩固我自身的权威和安全，他会因为这份知遇之恩，而更加死心塌地地为我所用。

领导提拔一个人，首要的考量从来就不是这个人能为公司创造多大价值，而是这个人能为自己的权力带来多大的增益和多小的风险。

这也就是我当领导后，明白的第三个道理：

**所谓的 “对你好”，是一种需要被计算的投资，而不是一种泛滥的情感。**

我为什么要对你好？为什么要跟你谈心，给你画饼，甚至在你犯错时，替你扛雷？

因为，我在你的身上看到了某种可以为我所用的价值。这种价值，可能是你无与伦比的才华，可能是你无可替代的忠诚，也可能是你背后所连接的、我所需要的某种资源。

而我对你的好，就是我投入的成本。

我投入这些成本，是为了在未来的某个时刻，获得回报。这个回报，可能是你为我创造的惊人业绩，可能是你在关键时刻对我毫不动摇的支持，也可能是你为我带来的某个关键信息。

这是一场交易。

而一旦，我发现我对你的投入与我预期的回报，出现了巨大的偏差。比如，你的能力开始停滞不前；比如你的忠诚出现了动摇，亦或是你开始变得无法被我掌控。

那么，我就会毫不犹豫地止损。

我会慢慢地收回那些曾经给你的特权；会把你从核心的项目里，调离出来；会减少和你一对一的交流。

这个过程是缓慢的，是不动声色的但却是不可逆转的。

所以，你也千万不要把领导对你的好，当成是理所当然的。

你要时刻保持警醒，反复地问自己：

他今天之所以对我好，是因为我身上的哪种价值？这种价值是可替代的吗？我如何才能持续地、甚至超预期地，为他提供这种价值？

看懂了这一层，你才能摆脱那种 “领导到底喜不喜欢我” 的学生式困惑，开始用一种更清醒也更专业的视角，去经营你和你领导之间这种最重要、也最脆弱的合作关系。

最后，我明白了一个关于自己的真相。

当我还是员工时，我以为领导力就是不断地向上爬、获得更高的职位。

但当我当了领导才明白，真正的权力不是来自于你头衔的高度，而是来自于你根基的深度。

公司可以任命我为领导，但只有我的下属才能真正地 “接纳” 我为领导，这份接纳，来自于我是否为他们争取过利益，是否在关键时刻替他们扛过雷，是否在他们迷茫时给过专业的指点。

我的权威最终并不是来自于我的上级，而是来自于我的下属，或者应该称作我的伙伴。

我不再是一个最优秀的球员，我是一个教练，我的价值不再体现在我个人能进多少球，而体现在我能培养出多少个比我更优秀的球员。

当我开始为下属的成功而由衷地感到高兴时，我才算真正地从一个管理者开始走向一个领导者。

这或许才是在这条充满算计和博弈的路上，唯一能让我们感到慰藉的那一点点关于 “善” 的微光。
