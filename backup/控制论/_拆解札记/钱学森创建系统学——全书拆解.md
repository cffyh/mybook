# 札记 · 《钱学森创建系统学》全书拆解（基于原文 581 页）

> 钱学森晚年思想总集，以系统学讨论班讲话与书信为体裁，围绕一个核心工程：如何用"从定性到定量综合集成法"处理社会、人体、人脑、地理这类无法用还原论切割的开放复杂巨系统。

---

## 0. 这是一本什么书

这本书不是教材，是档案。全书由钱学森1986—2000年间在系统学讨论班上的讲话、政协发言稿、以及写给数十位学者的往来书信汇编而成，581页，未经后人整理润色，保留了思想形成过程中的来回试错、命名变化与自我更正。许国志院士在序中称这批材料是钱学森"为建立中国系统学而进行的全部思想努力的原始记录"。

全书的核心工程是一件事：建立一套能处理"开放的复杂巨系统"的方法论。钱学森认为，人类社会、人体、人脑、地理系统这类对象，既不能用物理学的还原论切块处理，也不能满足于整体论的概念阐发，必须有一种把专家定性经验与计算机定量运算反复迭代融合的新方法——他称之为"从定性到定量综合集成法"，英文对应 Meta-synthesis，而非 Meta-analysis。

本书的写作风格是口语化、问题驱动、充满自我否定与重新出发。钱学森多次坦承"还有很多问题没搞清楚"，并描述自己从1978年到1986年经历七个认识阶段，每次都因某篇文章或某位同志的提醒而"眼界大开"。这种示范——承认局限、等待触发、范式跳跃——本身就是认识论意义上的示范。

本书的价值不在于它已经建成了系统学，而在于它完整记录了一位科学家如何在意识到"无工具可用"的情况下，从头发明工具的全部思维过程。

## 1. 全书骨架

全书约可分为三个层次，依次递进：

**第一层：本体论定位**（全书前三分之一，1986—1990年讲话为主）
→ 什么是开放的复杂巨系统？（判断标准：还原论失效处即是）
→ 系统的四级分类：简单系统 → 大系统 → 简单巨系统 → 复杂巨系统 → 特殊复杂巨系统（含人的社会系统）
→ 社会是"特殊复杂巨系统"，因其成员是有意识、有行为选择能力的人

**第二层：方法论建构**（全书中段，1990—1993年讲话与书信）
→ 从定性到定量综合集成法（Metasynthesis）的三要素：专家群体定性判断 + 统计数据约束 + 大型计算机建模迭代
→ 综合集成研讨厅体系（HWSMsE）：人机结合、以人为主的集体决策平台
→ 大成智慧工程：将古今中外人类知识集成，融合量智（逻辑/抽象思维）与性智（形象/直感思维）
→ 四种思维类型的整合：抽象思维 + 形象思维 + 社会（集体）思维 + 灵感（顿悟）思维

**第三层：应用扩展**（全书后三分之一，书信集1993—2000年）
→ 方法论向各领域的推广：社会经济、人体/中医、地理生态、脑科学、军事、互联网
→ 对Santa Fe Institute复杂性学派的定位：有价值的"一家之言"，但缺乏综合集成，结论不能作为唯一依据
→ 系统学建立策略：从繁到简——先确立开放复杂巨系统的总理论，再将简单系统等作为特殊条件下的分支特例
→ 产业革命论（五次→七次）与社会主义建设的系统工程框架

## 2. 核心概念与框架

### A. 开放的复杂巨系统——问题的本体论定位

钱学森将所有系统按两个维度分类：子系统的数量（大/小）与种类（简单/复杂）。简单巨系统：子系统数量极大但种类少（如气体分子），可用统计力学。复杂巨系统：子系统数量极大且种类极多（数百至数千类），现有理论工具全部失效。特殊复杂巨系统：在复杂巨系统的基础上，成员是有意识、有行为选择的人，这是社会系统。

这个定位的认识论意义在于：**复杂性是方法论的判定，不是感受上的。** 凡不能用还原论方法处理、或不宜用还原论方法处理的，才是真正的复杂性问题。混沌、湍流看起来混乱，但仍属有工具可循的范畴；社会系统才是真正的复杂巨系统。

### B. 从定性到定量综合集成法——核心方法论

操作流程六步循环：听取专家定性判断 → 综合建模 → 输入统计数据运算 → 专家评审 → 修改模型 → 再运算，反复至收敛为止。三要素缺一不可：专家经验意见（每位专家是一个局部真实的"灰色系统"）、客观统计数据、系统工程大模型（几百至几千个参数）。

方法的本质是认识论上的辩证统一：不是用定量替代定性，也不是并列两者，而是把定性认识作为起点，通过计算检验不断修正，最终升华为定量认识。这条路与毛泽东《实践论》同构：从感性认识到理性认识的跃升，是一切科学的共同运动方式。

### C. 涌现与层次——系统的动力学图景

核心洞见：**低层次的混沌造成高层次的有序，而这"有序"从更高层次看又是下一层次的"混沌"，如此无穷递进。** 系统学的任务就在于研究这个跨层跃迁——从子系统的性能与相互作用，推导出整个系统的结构与功能。公式：外界影响 → 系统结构 → 系统功能。

直接推论：整体不等于部分之和，拆解会失去关键联系。一旦把复杂巨系统拆开，"紧要的东西都跑了"——最重要的性质恰恰藏在子系统之间的关系里，而非任何单一子系统中。

### D. 量智与性智——认识论的双轨结构

量智以科学技术为主，从局部到整体，研究量变，依赖抽象逻辑思维；性智从整体感受入手，靠形象思维和直感领会质的整体。两者不是"理性 vs 感性"的二元对立，而是**两条认识路线的综合集成**：量智给精度，性智给方向感；量智防止空话，性智防止机械唯物论。

### E. 系统学的建立策略——自上而下，从繁到简

不应从简单系统→大系统→巨系统逐级外推，而应先建立开放复杂巨系统的总理论，再将简单系统等作为特殊条件下的分支特例分化出来。先树立最一般的总理论，再讲各种条件下的简化特例——这是一种构建知识体系的高阶策略，可迁移到任何领域的概念建构。

## 3. 与书稿的咬合接口

- **接口1 → 补进《工程技术》《创新的三个层次》**：钱学森对"方法论创新比具体方法更根本"的反复强调——圣菲研究所有具体工具但没有方法论突破，所以无法解决复杂巨系统问题——可补充"方法论本身作为一种创新层次"的维度。

- **接口2 → 补进沟通类文章（"为什么不能直来直去"）**："混沌是有序的前提"这个动力学洞见，与沟通中"来回摸索才能升维"的主题有直接的结构性支撑。低层次混沌（分歧、模糊、摩擦）造成高层次有序（共同框架的涌现），过早收敛反而破坏系统的自我调节能力。

- **接口3 → 补进《表达是抽取，不是复印》**："复杂整体的本质藏在子系统之间的关系里，一旦分解'紧要的东西都跑了'"——好的表达本身就是综合集成：把零散感性认识"缝"起来，用结构整合，而非用逻辑替换。

- **接口4 → 补进《学习论》**：认识分阶段跃迁，不是线性累积。"前科学"（经验知识）→ 经定性整理 → 再经定量验证 → 升格为科学，这条路径可以成为《学习论》的结构性骨架之一。"典型有价值但不能死守典型"可直接入文。

- **接口5 → 启发新文章：《分层次看问题》或《工具的边界》**：不同层次的系统需要完全不同的方法，混用层次是科学上最常见的错误。拿到问题首先判断它属于哪个层次，才能选对工具。与书稿核心主线"以有限对抗无限"直接呼应。

- **接口6 → 补进《概念与技能》**：经验是入口，是原材料，但它需要被系统性加工，而非直接引用。对"经验比理论更有价值"的立场是重要的细化和限定。

- **接口7 → 补进《从目标开始》**："外界影响 → 系统结构 → 系统功能"与"从目标开始"逻辑方向对应：目标（功能）确定后，反向推导需要什么结构——这是以终为始的系统论表达。

## 4. 留存金句（原文）

1. "凡现在不能用还原论方法处理的，或不宜用还原论方法处理的问题，而要用或宜用新的科学方法处理的问题，都是复杂性问题，复杂巨系统就是这类问题。"

2. "低层次的混沌造成高层次的有序，我看这就是混沌和有序的辩证关系。"

3. "一个有经验的大夫和一个刚毕业的医生的重要区别是：刚毕业的医生主要是凭书本上的知识，于是容易犯简单化的毛病。而有经验的医生认为同样是一种病，但人跟人不一样，各人有他的身体条件和生活习惯，这都有关系。"

4. "从定性到定量，从感性认识上升到理性认识，这是辩证统一的认识论，是最难的。"

5. "客观世界是决定性的，但由于人认识客观世界的局限性，会有暂时要引入非决定性的必要。这是前进中的驿站，无可厚非，只是决不能满足于非决定性而不求进一步地澄清。"

6. "我们是要把古今中外千亿人的头脑组织成为一个伟大的思维体系，复杂超巨型系统。可否称之为'大成智慧工程'？"

7. "对开放的复杂巨系统而言，'典型'有重要意义，应该重视，是专家意见；但又不能死抱着不放，否则，那就一定会犯错误！"

8. "矛盾是一个发展运动，会转化的。我们的许多失误都在于未跟上实际，思想僵化，不知道矛盾已经转化，出现新矛盾了。"

## 5. 该带走的骨

这本书最核心的洞见凝缩为一句话：**认识复杂系统，必须在定性与定量之间反复螺旋，不能单边停留。** 还原论（只分解，只定量）和整体论（只整体，只定性）各执一端，都无法处理真正的复杂性；真正有效的认识是把专家的感性经验与计算机的定量建模反复碰撞、迭代收敛。

第二个该带走的骨是：**层次不对，工具就错。** 拿到任何问题，首先要判断它在哪个层次，才能选对工具。这个习惯——在用工具之前先定位问题的层次——比任何具体工具都更根本。

第三个洞见：**整体的性质藏在关系里，不藏在任何单一部分里。** 思维是大脑几百万神经元的整体行为，不在任何单个神经元里；沟通的结果是双方互动的涌现，不在任何单方的表达里；知识的力量在于框架能整合零散碎片，而非碎片本身的积累量。一旦分解，最重要的东西就跑了。
